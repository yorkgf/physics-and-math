# 任务总览

> 使用 Obsidian Tasks 和 Reminder 插件管理任务

## 📊 任务统计

```tasks
not done
group by due
sort by due reverse
limit 10
```

## 📅 今日任务

```tasks
not done
due today
sort by urgency
```

## 🔔 即将到期

```tasks
not done
due before in 3 days
sort by due
```
