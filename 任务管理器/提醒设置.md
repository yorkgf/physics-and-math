# 🔔 Reminder 提醒设置

配合 Obsidian Reminder 插件使用。

## 快捷语法

| 语法 | 说明 | 示例 |
|------|------|------|
| `![[reminder]] 2026-03-20` | 日期提醒 | ![reminder](2026-03-20) |
| `![[reminder]] 2026-03-20 14:30` | 时间提醒 | ![reminder](2026-03-20 14:30) |

## 在任务中使用

```markdown
- [ ] 参加会议 ⏫ 📅 2026-03-20 ![reminder](2026-03-20 09:30)
- [ ] 提交报告 📅 2026-03-21 ![reminder](2026-03-21 17:00)
```

## 插件推荐设置

在 Obsidian 设置 → Reminder：

- ✅ 显示通知
- ✅ 播放声音（可选）
- ✅ 启动时检查过期提醒
- 提前提醒时间：5 分钟

## Tasks + Reminder 配合工作流

1. **捕获**：快速写入 [[收件箱]]
2. **整理**：每天整理到对应项目，添加日期和提醒
3. **查看**：在 [[任务总览]] 查看所有任务
4. **提醒**：到达时间后 Reminder 弹出通知
5. **完成**：勾选完成，Tasks 自动归档

## 使用建议

- 重要任务一定添加提醒时间
- 每天早上查看今日任务
- 每天晚上整理收件箱
- 使用 Projects/ 文件夹按项目组织任务
