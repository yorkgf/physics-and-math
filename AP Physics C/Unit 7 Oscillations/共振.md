# 共振

**共振**（Resonance）是受迫振动的特殊现象，当驱动频率等于系统固有频率时振幅最大。

## 固有频率

无阻尼系统的固有频率：

$$\omega_0 = \sqrt{\frac{k}{m}}$$

（弹簧振子系统）

## 共振条件

当驱动力的频率 $\omega_{\text{drive}} = \omega_0$ 时，发生**共振**，振幅达到最大值。

## 共振曲线

- 共振峰的高度和宽度与阻尼系数有关
- 阻尼越小，共振峰越尖锐
- 阻尼越大，共振峰越平缓

## 品质因子

$$Q = \frac{\omega_0}{\Delta\omega}$$

其中 $\Delta\omega$ 是半功率带宽。$Q$ 越大，共振越尖锐。

## 例题

**例题**：某振动系统固有频率 $f_0 = 10$ Hz，阻尼很小，品质因子 $Q = 50$，求共振频率和半功率带宽。

**解**：
共振频率 $\approx f_0 = 10$ Hz

半功率带宽：$\Delta f = \frac{f_0}{Q} = \frac{10}{50} = 0.2$ Hz
