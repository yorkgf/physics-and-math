# Friction Forces

Friction is a force that opposes relative motion between surfaces in contact. In AP Physics C, you need to understand both static and kinetic friction, their microscopic origins, and how to apply them in complex systems.

## Types of Friction

### Static Friction ($f_s$)
- Acts on objects that are **not moving** relative to each other
- Variable magnitude: $0 \leq f_s \leq f_{s,max}$
- Maximum static friction: $f_{s,max} = \mu_s N$
- Direction: Opposes **impending** motion

### Kinetic Friction ($f_k$)
- Acts on objects that are **sliding** relative to each other
- Constant magnitude: $f_k = \mu_k N$
- Direction: Opposes **actual** motion
- Typically: $\mu_k < \mu_s$

Where:
- $\mu_s$ = coefficient of static friction
- $\mu_k$ = coefficient of kinetic friction
- $N$ = normal force

## Key Properties

1. **Friction is independent of contact area** (for most practical cases)
2. **Static friction is a range, not a single value**
3. **Once motion starts, kinetic friction takes over** (usually lower than maximum static friction)
4. **Friction coefficients are material-dependent**

## Applications in Problem Solving

### Equilibrium Problems
When an object is at rest or moving with constant velocity:
$$\sum F = 0$$

The friction force adjusts to balance other forces, up to its maximum value.

### Accelerating Systems
When net force exceeds maximum static friction:
$$f_k = \mu_k N$$

Apply Newton's second law:
$$\sum F = ma$$

### Inclined Planes
For an object on an incline with angle $\theta$:
- Normal force: $N = mg\cos\theta$
- Maximum static friction: $f_{s,max} = \mu_s mg\cos\theta$
- Component of gravity down incline: $mg\sin\theta$

**Object begins to slide when:**
$$mg\sin\theta > \mu_s mg\cos\theta$$
$$\tan\theta > \mu_s$$

## Common Mistakes to Avoid

1. Assuming friction always equals $\mu N$ (only true for maximum static or kinetic)
2. Forgetting that static friction direction opposes **impending** motion
3. Using kinetic friction coefficient when object is static
4. Ignoring friction in energy problems (friction does negative work)
