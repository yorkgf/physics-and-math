# 转动惯量

**转动惯量**（Moment of Inertia）是 AP Physics C 的核心考点，需要掌握积分计算和平行轴定理。

## 定义

$$I = \int r^2 \, dm$$

其中 $r$ 是质元到转轴的垂直距离。

## 常见刚体的转动惯量

| 形状 | 转轴 | 转动惯量 |
|------|------|----------|
| 细杆（长 $L$）| 过端点垂直 | $I = \frac{1}{3}mL^2$ |
| 细杆（长 $L$）| 过中点垂直 | $I = \frac{1}{12}mL^2$ |
| 圆盘/圆柱（半径 $R$）| 对称轴 | $I = \frac{1}{2}mR^2$ |
| 实心球（半径 $R$）| 直径 | $I = \frac{2}{5}mR^2$ |
| 薄壳球（半径 $R$）| 直径 | $I = \frac{2}{3}mR^2$ |
| 薄壁圆筒（半径 $R$）| 对称轴 | $I = mR^2$ |

## 平行轴定理

$$I = I_{\text{cm}} + md^2$$

其中 $d$ 是新转轴到质心轴的距离。

## 例题

**例题**：求均匀细杆（长 $L$，质量 $m$）绕距其一端 $L/3$ 处垂直轴的转动惯量。

**解**：
已知 $I_{\text{cm}} = \frac{1}{12}mL^2$（绕中点）
$$I = I_{\text{cm}} + m\left(\frac{L}{3} - \frac{L}{2}\right)^2 = \frac{1}{12}mL^2 + m\left(\frac{L}{6}\right)^2$$
$$= \frac{1}{12}mL^2 + \frac{1}{36}mL^2 = \frac{3}{36}mL^2 + \frac{1}{36}mL^2 = \frac{1}{9}mL^2$$
