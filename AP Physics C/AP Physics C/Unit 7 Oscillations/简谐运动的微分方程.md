# 简谐运动的微分方程

AP Physics C 需要掌握从**牛顿第二定律推导简谐运动微分方程**，这是与 AP Physics 1 的关键区别。

## 微分方程推导

以弹簧振子为例：

1. 弹簧力：$F = -kx$
2. 牛顿第二定律：$ma = -kx$
3. 加速度 $a = \frac{d^2x}{dt^2}$

得到标准微分方程：

$$\frac{d^2x}{dt^2} = -\frac{k}{m}x$$

或写作：

$$\frac{d^2x}{dt^2} + \omega^2 x = 0$$

其中 $\omega = \sqrt{k/m}$ 是角频率。

## 解的形式

通解：

$$x(t) = A\cos(\omega t + \phi)$$

或等价形式 $x(t) = A\sin(\omega t + \phi')$。

## 速度和加速度

$$v(t) = \frac{dx}{dt} = -A\omega \sin(\omega t + \phi)$$

$$a(t) = \frac{d^2x}{dt^2} = -A\omega^2 \cos(\omega t + \phi) = -\omega^2 x$$

## 例题

**例题**：质量 $m = 0.5$ kg 的物体连接劲度系数 $k = 200$ N/m 的弹簧，求振动周期。

**解**：
$$\omega = \sqrt{\frac{k}{m}} = \sqrt{\frac{200}{0.5}} = \sqrt{400} = 20 \text{ rad/s}$$
$$T = \frac{2\pi}{\omega} = \frac{2\pi}{20} = 0.314 \text{ s}$$
