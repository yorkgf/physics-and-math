# Unit 7: Oscillations 深入概念

## 保留概念

- [[简谐运动的微分方程]] - 从牛顿定律推导、方程解的形式
- [[物理摆]] - 刚体摆动、转动惯量应用
- [[共振]] - 固有频率、受迫振动

## 推导关系

```
弹簧振子：F = -kx → 微分方程 d²x/dt² + ω²x = 0
                    ↓
              解：x = A cos(ωt + φ)
                    ↓
              简谐运动 → [[共振]]

物理摆：τ = -mgd sinθ → 微分方程 → 周期 T = 2π√(I/mgd)
                ↓
           [[转动惯量]]的应用
```

> 注：基础振动概念（单摆周期、简谐运动定义等）属于 AP Physics 1 范围，已省略。
