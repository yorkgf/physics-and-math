# 角动量

**角动量**（Angular Momentum）是 AP Physics C 的核心概念，需要掌握质点和刚体的角动量公式。

## 质点的角动量

$$\vec{L} = \vec{r} \times \vec{p} = \vec{r} \times m\vec{v}$$

其中 $\vec{r}$ 是质点到参考点的位置矢量。

## 刚体绕定轴转动

$$L = I\omega$$

其中 $I$ 是转动惯量，$\omega$ 是角速度。

## 角动量定理

合外力矩等于角动量的时间变化率：

$$\vec{\tau}_{\text{net}} = \frac{d\vec{L}}{dt}$$

## 例题

**例题**：质量 $m = 2$ kg 的质点以速度 $v = 3$ m/s 做半径 $r = 0.5$ m 的匀速圆周运动，求角动量大小。

**解**：
$$L = rmv = 0.5 \times 2 \times 3 = 3 \text{ kg·m}^2/\text{s}$$

方向垂直于圆周平面（可用右手定则判断）。
