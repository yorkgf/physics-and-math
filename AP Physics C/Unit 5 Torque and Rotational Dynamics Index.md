# Unit 5: Torque and Rotational Dynamics - In-Depth Concepts

## Key Concepts

- [[Moment of Inertia]] - Calculation by integration, parallel axis theorem
- [[Torque]] - Vector cross product calculation
- [[Angular Momentum]] - Particle and rigid body forms
- [[Conservation of Angular Momentum]] - Applications with changing moment of inertia

## Derivation Map

```
[[Torque]] → Rotational Newton's 2nd Law → τ = Iα
           ↓
[[Angular Momentum]] L = Iω
           ↓
[[Conservation of Angular Momentum]] I₁ω₁ = I₂ω₂

[[Moment of Inertia]] → Integration → Parallel Axis Theorem
```
