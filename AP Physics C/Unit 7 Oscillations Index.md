- [[Resonance]] - Natural frequency, forced oscillations

## Derivation Map

```
Mass-spring: F = -kx → Differential equation d²x/dt² + ω²x = 0
                            ↓
                  Solution: x = A cos(ωt + φ)
                            ↓
                  Simple Harmonic Motion → [[Resonance]]

Physical Pendulum: τ = -mgd sinθ → Differential equation → Period T = 2π√(I/mgd)
                          ↓
                 Application of [[Moment of Inertia]]
```
