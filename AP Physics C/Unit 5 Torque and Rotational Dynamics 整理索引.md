# Unit 5: Torque and Rotational Dynamics 深入概念

## 保留概念

- [[转动惯量]] - 积分计算、平行轴定理
- [[力矩]] - 矢量和计算
- [[角动量]] - 质点和刚体形式
- [[角动量守恒]] - 转动惯量变化应用

## 推导关系

```
[[力矩]] → 转动第二定律 → τ = Iα
       ↓
[[角动量]] L = Iω
       ↓
[[角动量守恒]] I₁ω₁ = I₂ω₂

[[转动惯量]] → 积分计算 → 平行轴定理
```

> 注：基础转动概念（角位移、角速度等）属于 AP Physics 1 范围，已省略。
