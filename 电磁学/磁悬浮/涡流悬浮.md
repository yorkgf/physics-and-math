# 涡流悬浮（Eddy Current Suspension）

涡流悬浮利用**导体中感应涡流产生的磁场**与外部磁场相互作用，产生悬浮力。本质上与[[电动悬浮（EDS）]]同源，但使用**永磁体**替代超导磁体。

---

## 核心原理

### 涡流的产生

当永磁体（或电磁铁）在导体表面附近移动时：

```
    永磁体（N 极向下）
         ↓ 运动方向 v →
    ════════════════════  导体板（铝/铜）
         ↑      ↑
         i      i      ← 感应涡流（环形）
         ↑      ↑
      B'     B'        ← 涡流产生的反向磁场
         ↑      ↑
         └──────┘
         悬浮力 F ↑
```

### 悬浮力公式

对于均匀导体板，涡流悬浮力近似为：

$$F_{\text{悬浮}} = \frac{\sigma \cdot v \cdot B^2 \cdot A}{2\delta}$$

其中：
- $\sigma$：导体电导率（铜 > 铝 > 不锈钢）
- $v$：相对运动速度
- $B$：永磁体表面磁场强度
- $A$：磁体面积
- $\delta$：导体厚度

**关键特性**：$F \propto \sigma \cdot v \cdot B^2$

---

## 涡流悬浮的物理分类

### 分类一：排斥式涡流悬浮

永磁体在导体板上方静止或缓慢移动，导体板中有静态磁场变化。

```
    永磁体（固定）
         ↓  强磁场 B
    ─────────────────  导体板
         ↑  感应涡流
         ↑  反向磁场
         ↑  排斥悬浮力
```

**特点**：
- 需要永磁体有强磁场梯度（$\partial B/\partial z$ 越大，悬浮力越大）
- 悬浮力与速度无关（静态排斥）
- 但实际上如果没有运动 $\rightarrow$ 没有感应 $\rightarrow$ 没有涡流 $\rightarrow$ 没有悬浮力

### 分类二：感应式涡流悬浮（主流）

永磁体在导体板上方**高速移动**，运动产生感应涡流。

```
    永磁体（高速运动 v）
         ↓ 运动
    ════════════════════  导体板
         ↑  运动 → 感应涡流
         ↑  涡流 → 反向磁场
         ↑  悬浮力 F
```

**特点**：
- $F \propto v$，静止时悬浮力为零
- 速度越高，悬浮力越大
- 必须有辅助支撑用于起浮

## 与电动悬浮（EDS）的比较

| 特性 | 涡流悬浮 | 电动悬浮（EDS）|
|------|---------|--------------|
| 磁场来源 | 永磁体 | 超导磁体 |
| 磁场强度 | 0.3–1.5 T | 3–5 T |
| 悬浮力 | $F \propto \sigma \cdot v \cdot B^2$ | $F \propto v \cdot dB/dz$ |
| 悬浮高度 | 1–10 mm | 10–100 mm |
| 起浮速度 | ~50–200 km/h | ~100 km/h |
| 功耗 | 极低（无电源）| 极低（超导无电阻）|
| 技术成熟度 | 中等 | 高（日本已商业化）|

**核心差异**：EDS 用超导磁体产生强得多且梯度更大的磁场，涡流悬浮用永磁体，成本低但性能受限。

## 导体材料对比

| 材料 | 电导率 $\sigma$ | 密度 | 悬浮力效率 | 工程应用 |
|------|----------------|------|-----------|---------|
| 铜 | 5.9×10⁷ S/m | 8.96 g/cm³ | 最高 | 实验室演示 |
| 铝 | 3.5×10⁷ S/m | 2.70 g/cm³ | 较高，轻量 | 涡流悬浮导轨 |
| 银 | 6.3×10⁷ S/m | 10.5 g/cm³ | 最高（但太贵）| 极少数特殊场景 |
| 不锈钢 | ~1.4×10⁶ S/m | 7.8 g/cm³ | 低 | 不适合涡流悬浮 |

**结论**：铝是最常用的涡流悬浮导轨材料（高电导率 + 轻量化 + 成本合理）。

## 悬浮力分布（Halbach 阵列增强）

与 EDS 类似，涡流悬浮也使用 **Halbach 阵列** 来增强悬浮力：

```
标准永磁体排列：          Halbach 阵列：
 N S N S N S             N S→→→→ N
 S N S N S N             ↑   增强侧   ↓
（磁场分散）             ←←←←←←←←←←←
                        （磁场集中向下）
```

Halbach 阵列的优点：
- 向下磁场增强 1.4–1.7 倍
- 悬浮力提升约 40–70%
- 减少材料用量（相同效果下永磁体更少）

## 工程应用

### 磁悬浮轴承

涡流悬浮广泛用于**无接触轴承**：

```
    转子
  ┌───────┐
  │       │  ← 轴向运动
  └───────┘
     ↑  ↑  ← 径向悬浮（涡流）
     │  │
  ┌───────┐
  │  永磁体阵列 │  ← 固定在轴承座上
  └───────┘
     ↑
   导体套筒（铝/铜）← 转子上，随转子旋转
```

**工作原理**：
- 导体套筒在永磁体阵列产生的梯度磁场中旋转
- 旋转运动在导体中产生涡流
- 涡流产生反向磁场 → 推力使转子悬浮
- **转子转速越高，悬浮力越大**（$F \propto v$）

**应用场景**：
- 真空泵转子（无接触，无摩擦，无润滑剂）
- 高速飞轮储能系统
- 航空航天涡轮机械

### 磁悬浮演示实验

最简单的涡流悬浮演示：

```
步骤 1：
  把一块钕磁铁（N52）绑在玩具车上
  ↓
步骤 2：
  将铝板或铜板放在车前方
  ↓
步骤 3：
  快速推动小车撞向铝板
  ↓
步骤 4：
  小车在铝板上方悬浮滑行一段距离（速度降低 → 悬浮力减小 → 落地）
```

这个演示说明：涡流悬浮**无法静止悬浮**，但高速运动时可以产生可观的悬浮力。

### 高速管道交通（概念设计）

```
    永磁体阵列（车厢）
         ↓ 运动方向
    ════════════════════════════  铝/复合材料管道
         ↑  管道壁中的涡流
         ↑  悬浮力 F
```

- 管道内抽真空（消除空气阻力）
- 涡流悬浮提供无接触支撑
- 电磁推进提供推力
- 理论速度可达 1000+ km/h

## 涡流悬浮的能量流分析

```
永磁体磁场 B₀
    ↓ 运动穿过导体
导体中的感应涡流 i
    ↓
涡流发热（焦耳热损耗）
    P_heat = i²·R
    ↓
涡流产生反向磁场 B'
    ↓
B' 与 B₀ 相互作用
    ↓
悬浮力 F（推力分量）
    ↓
列车动能减少
```

**重要结论**：涡流悬浮的能量损耗来自**导体中的焦耳热**——这与空气阻力本质上是一样的，都是把动能转化为热能。

## 悬浮力的速度依赖性

```
悬浮力 F
   │           ╱ ← 平方增长区（低速度）
   │        ╱
   │     ╱
   │  ╱
   │╱
   └────────────────────→ 速度 v
   O          v_max
```

与 EDS 类似，涡流悬浮也存在**最佳悬浮速度区间**：
- 速度太低：悬浮力不足
- 速度太高：电磁波辐射效应开始出现，悬浮力趋于饱和
- **最佳区间**：由导体的电导率和几何形状决定
