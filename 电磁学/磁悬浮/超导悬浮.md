# 超导悬浮

超导悬浮利用**超导体的量子力学效应**（迈斯纳效应 + 磁通钉扎）实现稳定悬浮，是目前能量效率最高的磁悬浮方案。

---

## 两种超导悬浮类型

### 第一类超导悬浮：迈斯纳悬浮

**迈斯纳效应（Meissner Effect，1933年）**：

> 当温度降至临界温度 $T_c$ 以下，超导体内部磁场 $B = 0$，完全排斥外部磁场。

```
    外部磁场 B_ext
         ↓
┌─────────────────────┐
│  超导体内部          │  ← B = 0（完全抗磁）
│  排斥所有磁场        │
│                     │
│  ← ← ← ← ← ← ← ← ←←  ← 被排斥的磁场线
└─────────────────────┘
         ↑
    悬浮力 F（排斥力）
```

**悬浮特性**：
- 超导体被外部磁场"推走"，悬浮高度取决于磁场强度
- 悬浮是**瞬时的**（降温即悬浮，无需运动）
- 但超导体必须是**完整的块体**，不能有裂缝

**典型应用**：超导磁悬浮演示（块状 YBCO 高温超导体 + 液氮冷却 + 永磁体）

### 第二类超导悬浮：磁通钉扎悬浮（工程主流）

第二类超导体（高温超导体 YBCO 等）允许磁场**部分穿透**内部，形成量子化的磁通涡旋：

```
    外部磁场 B_ext
         ↓
┌─────────────────────┐
│  ┊  ┊  ┊  ┊  ┊  ┊  ┊  ← 磁通涡旋（被钉扎）
│  ○  ●  ○  ●  ○  ●  │  ← 磁通量子
│  ┊  ┊  ┊  ┊  ┊  ┊  ┊  ← 晶格缺陷钉扎点
└─────────────────────┘
         ↑
悬浮力 F + 锁定力
```

## 磁通钉扎（Flux Pinning）机制

### 什么是磁通钉扎

第二类超导体内部存在大量晶格缺陷（孪晶界、脱溶相、位错等），这些缺陷形成"钉扎中心"。

当外加磁场穿透超导体后：
1. 磁场以量子化的"磁通管"形式存在
2. 每个磁通管的能量在钉扎中心处**更低**
3. 磁通管被"钉"在缺陷位置，不能自由移动
4. 试图移动超导体 → 磁通管被拉扯 → 产生**钉扎力**

### 钉扎力方程

$$F_{\text{钉扎}} \propto n_{\text{钉扎中心}} \cdot B \cdot J_c$$

其中：
- $n_{\text{钉扎中心}}$：单位体积内钉扎中心数量
- $B$：穿透的磁感应强度
- $J_c$：**临界电流密度**（超导体能承载的最大无电阻电流）

**临界电流密度**是超导悬浮性能的关键：
- $J_c$ 越大 → 钉扎力越强 → 悬浮稳定性越好
- 高温超导体 YBCO 的 $J_c$ 可达 $10^6$–$10^8$ A/cm²

---

## 迈斯纳效应与磁通钉扎的对比

| 特性 | 迈斯纳效应（第一类）| 磁通钉扎（第二类）|
|------|------------------|-----------------|
| 超导体类型 | 纯金属（Pb, Nb, Hg）| 合金/化合物（YBCO, NbTi）|
| 临界温度 $T_c$ | < 10 K（低温）| > 77 K（高温，液氮）|
| 悬浮效果 | 完全排斥，强排斥力 | 部分穿透，有锁定力 |
| 静止稳定性 | 良好 | 极佳（锁定效应）|
| 工程难度 | 高（需极低温）| 中等（液氮可满足）|
| 悬浮高度 | 可达数厘米 | 数毫米至数厘米 |

## 高温超导体 YBCO 的悬浮机制

### 材料特性

| 参数 | YBCO（YBa₂Cu₃O₇₋δ）|
|------|-------------------|
| 临界温度 $T_c$ | 92 K（约 −181°C）|
| 临界磁场 $H_{c2}$ | ~100 T（极高）|
| 工作温度 | 77 K（液氮沸点）|
| 晶格结构 | 钙钛矿层状结构 |

### 悬浮工作过程

```
步骤 1：冷却
  超导体置于磁体上方
  注入液氮 → 超导体降至 77 K 以下
  ↓

步骤 2：迈斯纳效应
  超导体排斥外加磁场 → 建立初始悬浮
  ↓

步骤 3：场冷（Field Cooling）
  在有磁场条件下冷却 → 磁通被"冻结"在超导体内
  ↓

步骤 4：悬浮建立
  超导体 + 悬浮体整体悬浮
  悬浮高度由磁通钉扎力决定
  ↓

步骤 5：锁定
  试图移动超导体 → 钉扎力抵抗移动
  悬浮高度基本锁定（有一定范围的活动性）
```

### "悬浮但不完全悬浮"的特性

磁通钉扎悬浮有一个有趣的特性：**悬浮体既可以悬浮，又可以被轻微按压而不会完全脱离**。

- 按下去 → 钉扎力抵抗 → 松手后弹回原位
- 提起一点 → 钉扎力抵抗 → 松手后落回原位
- 这种**弹性锁定**是 EDS 和 EMS 都不具备的特性

## 日本 L0 超导磁悬浮列车

### 系统架构

```
车体顶部：
  ┌──────────────────────────────────────┐
  │  低温超导磁体（Cryocab）             │
  │  - 超导线：NbTi（4.2 K，液氦）       │
  │  - 磁场强度：~5 T                    │
  │  - 超导闭合回路（持续电流，无功耗）  │
  └──────────────────────────────────────┘

车体底部：
  ┌──────────────────────────────────────┐
  │  悬浮与导向磁体（CSC：悬浮线圈）    │
  │  - 8 字形线圈（推进）                │
  │  - 横向线圈（导向）                   │
  └──────────────────────────────────────┘

导轨：
  ┌──────────────────────────────────────┐
  │  铝导轨 + 悬浮线圈（被动）            │
  │  - 无需供电                           │
  │  - 列车超导磁体运动时感应电流         │
  └──────────────────────────────────────┘
```

### 悬浮过程详解

| 速度 | 悬浮状态 | 物理机制 |
|------|---------|---------|
| 0 km/h | 无悬浮（车轮支撑）| 迈斯纳效应无法直接悬浮静止物体 |
| 0–100 km/h | 辅助车轮支撑 | 车轮提供初始动力 |
| ~100 km/h | EDS 悬浮开始 | 运动感生涡流 → 悬浮力建立 |
| > 100 km/h | EDS 悬浮稳定 | 悬浮力随速度增加而增强 |
| 高速（500+ km/h）| EDS + 磁通钉扎协同 | 悬浮力 >> 车重，极度稳定 |

### 关键参数

| 参数 | L0 系列 |
|------|---------|
| 最高运营速度 | 603 km/h（2023年）|
| 悬浮高度 | 10 cm |
| 超导线温度 | 4.2 K（液氦，−269°C）|
| 磁体重量 | ~5 吨/车厢 |
| 悬浮功耗 | ≈ 0（超导无电阻）|
| 冷却功耗 | ~10–20%（用于制冷）|
| 能效 | 所有方案中最高 |

## 冷却系统

### 冷却方式对比

| 冷却方式 | 温度 | 超导体 | 应用 |
|---------|------|--------|------|
| 液氦冷却 | 4.2 K | NbTi | 日本 L0（低温超导）|
| 液氮冷却 | 77 K | YBCO, Bi-2223 | 高温超导悬浮演示 |
| 制冷机冷却 | 10–77 K | REBCO 带材 | 研究阶段 |

### 冷却成本

液氦（4.2 K）的成本远高于液氮（77 K），这是高温超导悬浮研究的重要动力：
- **液氮**：~0.5 元/升
- **液氦**：~200 元/升（全球供应紧张）

日本 L0 的主要技术挑战之一是**液氦的稳定供应和密封循环**。
