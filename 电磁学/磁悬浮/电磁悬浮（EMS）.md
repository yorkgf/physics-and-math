# 电磁悬浮（EMS, Electromagnetic Suspension）

电磁悬浮是**有源主动悬浮**的典型代表，通过**实时电磁反馈控制**将不稳定的磁悬浮变为可工程实现的受控悬浮。

---

## 核心原理

### 系统架构

```
      位置传感器（测距）
           ↓ 间隙 d
        ┌──────────┐
        │ 控制器   │ ← PID 反馈
        └────┬─────┘
             ↓ 控制电流 I
      ┌─────────────────┐
      │   电磁铁线圈    │ ← 吸引力 F
      │   F = k·I²/d²  │
      └────────┬────────┘
               │ 间隙 d
               ↓
        ┌──────────────┐
        │   导轨       │
        │  （铝/钢被动）│
        └──────────────┘
             ↑
          列车车体
```

### 悬浮力公式

电磁悬浮的悬浮力由磁吸力公式给出：
$$F_{\text{悬浮}} = \frac{\mu_0 N^2 A I^2}{4 d^2}$$

其中：
- $N$：线圈匝数
- $A$：磁极面积
- $I$：通电电流
- $d$：气隙长度（悬浮高度）

**关键特性**：悬浮力 $F \propto I^2 / d^2$，即电流越大悬浮力越大，间隙越小悬浮力越大。

> 这就解释了为什么必须用反馈控制：间隙 $d$ 变小 → 悬浮力急剧增大 → 如果不降低电流，悬浮体会被"吸上去"撞到导轨。

### 平衡条件

当悬浮力等于重力时：
$$\frac{\mu_0 N^2 A I^2}{4 d^2} = mg$$

这个平衡是**不稳定的**——任何微扰都会破坏平衡：
- 悬浮体下降 → $d$ 减小 → $F$ 增大 → 继续下降（直至碰撞）
- 悬浮体上升 → $d$ 增大 → $F$ 减小 → 继续上升（直至悬浮力为零）

**结论**：没有反馈控制，电磁悬浮无法稳定工作。

## 反馈控制系统

### PID 控制原理

控制器实时调节电流 $I$，使悬浮高度保持在目标值 $d_0$：

$$I(t) = k_p \, e(t) + k_i \int_0^t e(\tau) d\tau + k_d \frac{de}{dt}$$

其中偏差 $e(t) = d(t) - d_0$。

| 控制项 | 作用 |
|--------|------|
| 比例项 $k_p e$ | 立即响应当前偏差（力图当前误差）|
| 积分项 $k_i \int e$ | 消除稳态误差（纠正历史累积偏差）|
| 微分项 $k_d \dot{e}$ | 预测未来偏差（抑制振荡，提前制动）|

### 控制系统方框图

```
目标高度 d₀
    ↓  (−)
    │    ┌────────────────────────────────────┐
    │    │            PID 控制器              │
    │    │  k_p, k_i, k_d 共同调节电流        │
    │    └────────────┬───────────────────────┘
    │                 ↓ 控制电流 I
    │          ┌─────────────┐
    │          │ 电磁铁线圈   │ → 悬浮力 F
    │          └──────┬──────┘
    │                 ↓ 悬浮力 F
    │                 ↓
    │             [车体 + 导轨]
    │                 ↓
    │          位置传感器（测量实际高度）
    │                 ↓
    └────────────────┐
                      ↑
            实际高度 d(t)（反馈）
```

## 关键技术参数

### 德国 Transrapid 系统参数

| 参数 | 值 |
|------|-----|
| 悬浮高度 | 10–15 mm |
| 轨道类型 | 钢梁 + 铝导轨（兼作推进轨）|
| 悬浮功耗 | ~5–10% 总功耗 |
| 最高速度 | 501 km/h（2003年）|
| 悬浮系统重量 | ~5–7 吨/车厢 |
| 间隙传感器精度 | 0.1 mm 级 |
| 失控风险 | 断电后需备用电源或机械支架 |

### 控制系统挑战

1. **非线性**：悬浮力 $\propto I^2/d^2$ 是强非线性关系，PID 参数需要在线调节
2. **时滞**：传感器→控制器→电磁铁整个回路存在 ~1–5 ms 延迟
3. **振动抑制**：列车高速行驶时，气流和轨道不平顺会造成剧烈振动
4. **容错**：任何一路电磁铁失效都需要备用磁铁接管（冗余设计）

## 安全性设计

### 断电保护机制

电磁悬浮最大的安全隐患：**断电即失浮**。工程上采用多重冗余：

1. **备用电源**：悬浮系统配备独立蓄电池，断电瞬间切换
2. **滑橇设计**：列车底部安装紧急滑橇，断电时接触导轨滑行而非坠落
3. **分段供电**：轨道分多段供电，单段断电不影响其他段悬浮
4. **自举设计**：列车有机械车轮辅助启动，悬浮建立后才离开地面

### 冗余磁铁布置

```
列车底部横截面：

[M] [M] [M] [M] [M] [M]  ← 多组独立电磁铁
 ↑   ↑   ↑   ↑   ↑   ↑
每个 M 由独立控制器控制
任意 1-2 个失效，其他接管
```

## 技术优势与劣势

### 优势

- **技术成熟**：原理简单，已商业化运营（上海磁浮线）
- **任意速度下均可悬浮**：静止到高速全程悬浮
- **悬浮高度可精确控制**：通过电流精确调节
- **制造成本相对较低**（相比超导悬浮）

### 劣势

- **必须持续供电**：断电即失浮，安全性依赖备用电源
- **气隙极小**：10–15 mm 意味着对轨道平整度要求极高
- **功耗不可忽视**：需要持续电能维持悬浮
- **高速时气动阻力大**：空气阻力是限制速度的主要因素
