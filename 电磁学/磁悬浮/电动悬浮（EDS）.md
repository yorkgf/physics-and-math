# 电动悬浮（EDS, Electrodynamic Suspension）

电动悬浮利用**电磁感应原理**（楞次定律）产生悬浮力。与电磁悬浮不同，电动悬浮**不需要主动反馈控制**，悬浮力随速度增加而自动增强。

---

## 核心原理

### 楞次定律（Lenz's Law）

感应电流的磁场总是**抵抗引起感应电流的磁通量变化**：

$$\mathscr{E} = -\frac{d\Phi}{dt} \quad \text{（法拉第电磁感应定律）}$$

在电动悬浮中，超导磁体在金属导轨上方移动，磁通量变化产生感应电流，感应电流产生**反向磁场**，从而产生悬浮力。

### 悬浮力产生机制

```
    超导磁体（向下 N 极）
      ↓ 运动方向 v →
    ═══════════════════  金属导轨（铝/铜）
           ↑
      感应涡流 i
           ↑
      反向磁场 B'（由 i 产生）
           ↑
      悬浮力 F ∝ v · (dB/dz)
```

**关键公式**：

$$F_{\text{悬浮}} = C \cdot \frac{dB}{dz} \cdot v \cdot \sin(\omega t)$$

其中：
- $C$：与导轨电导率和几何形状相关的常数
- $dB/dz$：超导磁体磁场的空间梯度（垂直方向）
- $v$：相对运动速度

**最重要的特性**：$F_{\text{悬浮}} \propto v$

---

## 悬浮力-速度特性曲线

```
悬浮力 F
    │                  ← 平方增长区（低速，趋肤深度变化）
    │             ╱
    │          ╱  ← 线性增长区（中高速）
    │       ╱
    │    ╱
    │ ╱
    └──────────────────────→ 速度 v
    O   v_min（最小悬浮速度）
```

### 三个速度区间

| 速度区间 | 悬浮力特性 | 物理原因 |
|---------|-----------|---------|
| $v < v_{\text{min}}$ | 悬浮力不足，无法悬浮 | 感应涡流太弱 |
| $v_{\text{min}} < v < v_{\text{max}}$ | $F \propto v$，悬浮力逐渐增强 | 涡流增强，趋肤深度接近稳定 |
| $v > v_{\text{max}}$ | $F \propto 1/v$，悬浮力趋于饱和 | 电磁波辐射开始主导 |

**典型 $v_{\text{min}}$**：约 100 km/h（电动悬浮列车的起浮速度）

## 趋肤效应与感应深度

### 趋肤深度（Skin Depth）

高频或高速变化磁场在导体中的渗透深度：
$$\delta = \sqrt{\frac{2}{\mu_0 \sigma \omega}}$$

对于铝导轨，典型值：
- $\delta \approx 1$ mm（100 Hz 量级）

### 对电动悬浮的影响

当超导磁体的磁场以速度 $v$ 扫过金属导轨时，等效频率：
$$\omega \approx \frac{v}{\ell}$$

其中 $\ell$ 是超导磁体的特征长度（约 1 m）。

- $v = 100$ m/s（约 360 km/h）→ $\omega \approx 100$ rad/s → $\delta \approx 5$ mm
- 趋肤深度内的感应涡流提供悬浮力
- 速度越高，等效频率越高，趋肤深度越浅，但涡流强度增大

## 起浮问题与辅助系统

### 起浮困难

电动悬浮的致命缺陷：**静止时 $v = 0$，悬浮力为零**。

解决方案：需要**辅助悬浮系统**配合使用：

```
速度区间          悬浮方式
─────────────────────────────
 0–~100 km/h    机械车轮 + 辅助电磁悬浮
~100 km/h以上   电动悬浮自动接管
~100 km/h以上   车轮收起（全悬浮运行）
```

### 日本 L0 系列车的实际方案

| 阶段 | 速度 | 悬浮方式 |
|------|------|---------|
| 起动 | 0–~100 km/h | 橡胶辅助车轮 |
| 过渡 | ~100 km/h | 电动悬浮逐渐建立 |
| 高速 | 500+ km/h | 电动悬浮 + 超导磁体 |

## 技术参数对比（EDS vs EMS）

| 参数 | 电动悬浮 EDS | 电磁悬浮 EMS |
|------|------------|------------|
| 悬浮高度 | 30–100 mm | 10–15 mm |
| 悬浮力来源 | 感应涡流（无源）| 电磁铁电流（有源）|
| 静止悬浮 | ❌ 不能 | ✅ 可以 |
| 起浮速度 | ~100 km/h | 0 km/h |
| 悬浮功耗 | 极低 | 5–10% 总功耗 |
| 高速稳定性 | 极佳（速度越高越稳）| 依赖控制精度 |
| 轨道要求 | 中等（不需要超精密）| 极高（气隙太小）|
| 技术成熟度 | 高（日本已商业化）| 高（德国已商业化）|

## 磁路设计

### Halbach 阵列

现代 EDS 系统广泛使用 **Halbach 磁体阵列**，将磁场能量集中在一侧（面向导轨侧），减少背向磁场，提高悬浮效率。

```
理想 Halbach 阵列磁场分布：

  →→→→→→→→→→→→→→→→→  ← 增强侧（面向导轨）
  ←←←←←←←←←←←←←←←←←←  ← 减弱侧（背向）
```

**优势**：
- 导轨侧磁场强度提升约 1.4–1.7 倍
- 悬浮力效率提高
- 减少对周围环境的磁场干扰

## 代表应用

### 日本 SCMAGLEV（低温超导磁悬浮）

| 参数 | 值 |
|------|-----|
| 最高试验速度 | 603 km/h（2023年，有乘客）|
| 悬浮高度 | 10–30 mm |
| 超导温度 | −269°C（液氦，4.2 K）|
| 磁体类型 | NbTi 超导线（超导）|
| 导轨类型 | 铝 + 不锈钢叠层 |
| 悬浮功耗 | 几乎为零（电阻为零）|
| 起浮方式 | 辅助车轮 + EDS 过渡 |

### 工作机制（完整流程）

```
1. 静止：超导磁体建立强磁场（~5T），辅助车轮支撑
2. 加速至 100 km/h：车轮提供支撑，超导线圈持续通电
3. 100 km/h 以上：电动悬浮力 > 车重，EDS 自动接管
4. 高速运行：EDS 悬浮力随速度提升而增强（极其稳定）
5. 减速落地：EDS 力减弱，辅助车轮重新接触轨道
```

## 能量流分析

```
超导磁体磁场 B₀
    ↓（运动扫过导轨）
金属导轨中感应涡流 i
    ↓
涡流产生反向磁场 B'
    ↓（B' 与 B₀ 相互作用）
悬浮力 F
    ↓
能量来源：列车的动能
    （悬浮力本质上是"空气阻力"的类似物，消耗动能）
```

电动悬浮的悬浮力本质上是**制动力的垂直分量**——悬浮力越大，列车的动能损失越多。这与电磁悬浮（悬浮功耗来自电能）有本质区别。
