# 规范变换（Gauge Transformation）

## 问题的来源：势函数不唯一

物理量是 $\mathbf{E}$ 和 $\mathbf{B}$，而势函数 $\varphi$（电势）和 $\mathbf{A}$（磁矢势）是辅助工具。

由 $\mathbf{B} = \nabla\times\mathbf{A}$，对任意标量函数 $\chi(\mathbf{r}, t)$，变换：

$$\mathbf{A} \;\to\; \mathbf{A}' = \mathbf{A} + \nabla\chi$$

不改变 $\mathbf{B}$，因为 $\nabla\times(\nabla\chi)\equiv 0$（梯度无旋）。

但同时，为保证电场 $\mathbf{E} = -\nabla\varphi - \partial\mathbf{A}/\partial t$ 也不变，$\varphi$ 必须同步变换：

$$\varphi \;\to\; \varphi' = \varphi - \frac{\partial\chi}{\partial t}$$

这一对变换合称**规范变换**：

$$\boxed{\mathbf{A}' = \mathbf{A} + \nabla\chi, \qquad \varphi' = \varphi - \frac{\partial\chi}{\partial t}}$$

$\chi$ 称为**规范函数**，可以任意选取——不同的选取方式称为不同的**规范（Gauge）**。

> **规范不变性**：物理可观测量 $\mathbf{E}$、$\mathbf{B}$ 在规范变换下不变。

---

## 为什么要选规范

势函数有无穷多种选法，选规范就是**约束这个自由度**，使方程变得最简便。不同问题选不同规范：

---

## 库仑规范（Coulomb Gauge）

$$\boxed{\nabla \cdot \mathbf{A} = 0}$$

### 推导结果

代入安培定律 $\nabla\times\mathbf{B} = \mu_0\mathbf{J}$，利用[[矢量分析/矢量恒等式总结|旋度的旋度恒等式]]：

$$\nabla\times(\nabla\times\mathbf{A}) = \nabla(\underbrace{\nabla\cdot\mathbf{A}}_{=\,0}) - \nabla^2\mathbf{A} = -\nabla^2\mathbf{A}$$

得到矢量[[电磁学/泊松方程与拉普拉斯方程|泊松方程]]：

$$\nabla^2\mathbf{A} = -\mu_0\mathbf{J}$$

电势满足：

$$\nabla^2\varphi = -\frac{\rho}{\varepsilon_0}$$

### 特点

| 优点 | 缺点 |
|------|------|
| 方程结构最简单（标准泊松方程） | 不满足相对论协变性 |
| $\varphi$ 有直接物理含义（库仑势） | $\mathbf{A}$ 的解不是局域的（$\varphi$ 变化"瞬间"传播） |
| 静磁、量子力学常用 | 不适合处理辐射问题 |

### 适用场景

- **静磁学**（本笔记的主要背景）
- **量子力学**中的电磁耦合（$\hat{H} = \frac{1}{2m}(\hat{p}-e\mathbf{A})^2 + e\varphi$）
- **[[电磁学/Neumann公式]]的推导**（矢量泊松方程的积分解 → 磁矢势积分形式）

## 洛伦兹规范（Lorenz Gauge）

$$\boxed{\nabla \cdot \mathbf{A} + \mu_0\varepsilon_0\frac{\partial\varphi}{\partial t} = 0}$$

### 推导结果

代入麦克斯韦方程，$\varphi$ 和 $\mathbf{A}$ 各自满足**波动方程**：

$$\nabla^2\varphi - \mu_0\varepsilon_0\frac{\partial^2\varphi}{\partial t^2} = -\frac{\rho}{\varepsilon_0}$$

$$\nabla^2\mathbf{A} - \mu_0\varepsilon_0\frac{\partial^2\mathbf{A}}{\partial t^2} = -\mu_0\mathbf{J}$$

用四维符号写得更紧凑：$\Box^2 A^\mu = -\mu_0 J^\mu$（其中 $\Box^2 = \nabla^2 - \frac{1}{c^2}\partial_t^2$ 是达朗贝尔算子）。

### 特点

| 优点 | 缺点 |
|------|------|
| 满足相对论协变性（$\varphi$ 和 $\mathbf{A}$ 地位对称） | 方程比库仑规范复杂一些 |
| 电磁波、辐射问题的首选 | $\varphi$ 无直接静电意义 |
| $\varphi$ 和 $\mathbf{A}$ 以光速传播，因果性自然满足 | |

---

## 两种规范对比

| | 库仑规范 | 洛伦兹规范 |
|--|---------|----------|
| 条件 | $\nabla\cdot\mathbf{A}=0$ | $\nabla\cdot\mathbf{A}+\mu_0\varepsilon_0\dot\varphi=0$ |
| $\varphi$ 方程 | $\nabla^2\varphi=-\rho/\varepsilon_0$（泊松） | $\Box^2\varphi=-\rho/\varepsilon_0$（波动） |
| $\mathbf{A}$ 方程 | $\nabla^2\mathbf{A}=-\mu_0\mathbf{J}$（泊松） | $\Box^2\mathbf{A}=-\mu_0\mathbf{J}$（波动） |
| 协变性 | ✗ | ✓ |
| 常用场景 | 静磁、量子力学 | 电动力学、辐射 |

---

## 深层意义

规范不变性不只是数学便利——它是自然界的基本对称性：

$$\text{U(1) 规范对称性} \;\Longleftrightarrow\; \text{光子无质量} \;\Longleftrightarrow\; \text{电荷守恒}$$

这一思想在粒子物理中推广为 SU(2)、SU(3) 规范理论，成为标准模型的基础。
