# 泊松方程与拉普拉斯方程

这两个方程描述**势场与其源之间的关系**，是静电学和静磁学的核心方程，也广泛出现在引力、流体、热传导等领域。

---

## 从哪里来

### 静电学推导

麦克斯韦方程给出（静电，无时变）：

$$\nabla \cdot \mathbf{E} = \frac{\rho}{\varepsilon_0}, \qquad \nabla \times \mathbf{E} = 0$$

由无旋条件引入标量电势 $\varphi$（$\mathbf{E} = -\nabla\varphi$），代入散度方程：

$$\nabla \cdot (-\nabla\varphi) = \frac{\rho}{\varepsilon_0}$$

$$\boxed{\nabla^2\varphi = -\frac{\rho}{\varepsilon_0}} \qquad \text{（泊松方程）}$$

无电荷区域（$\rho = 0$）退化为：

$$\boxed{\nabla^2\varphi = 0} \qquad \text{（拉普拉斯方程）}$$

### 静磁学中的类比

在[[电磁学/规范变换|库仑规范]] $\nabla\cdot\mathbf{A}=0$ 下，由安培定律推出矢量泊松方程：

$$\boxed{\nabla^2\mathbf{A} = -\mu_0\mathbf{J}} \qquad \text{（矢量泊松方程）}$$

三个分量各自满足与静电泊松方程相同结构的标量方程。

---

## 两个方程的关系

$$\underbrace{\nabla^2\varphi = -\frac{\rho}{\varepsilon_0}}_{\text{泊松方程（有源）}} \xrightarrow{\;\rho=0\;} \underbrace{\nabla^2\varphi = 0}_{\text{拉普拉斯方程（无源）}}$$

| | 泊松方程 | 拉普拉斯方程 |
|--|---------|------------|
| 适用区域 | 有源区（有电荷/电流） | 无源区（真空、导体外） |
| 解的特点 | 特解 + 齐次通解 | 仅齐次解 |
| 典型场景 | 已知电荷分布求电势 | 已知边界条件求空间电势 |

---

## 泊松方程的积分解

泊松方程的解可以写成**格林函数卷积**的形式。

核心：$\nabla^2\left(\dfrac{1}{r}\right) = -4\pi\delta^3(\mathbf{r})$（见[[矢量分析/矢量恒等式总结]]）

对于 $\nabla^2 G = -\delta^3(\mathbf{r}-\mathbf{r}')$，格林函数为 $G = \dfrac{1}{4\pi|\mathbf{r}-\mathbf{r}'|}$，由此得积分解：

$$\varphi(\mathbf{r}) = \frac{1}{4\pi\varepsilon_0}\int \frac{\rho(\mathbf{r}')}{\lvert\mathbf{r}-\mathbf{r}'\rvert}\,dV'$$

这正是我们熟知的**库仑积分**——从泊松方程的角度看，它是方程 $\nabla^2\varphi = -\rho/\varepsilon_0$ 的通解。

类比地，矢量泊松方程的解就是[[电磁学/磁矢势]]的积分表达式：

$$\mathbf{A}(\mathbf{r}) = \frac{\mu_0}{4\pi}\int \frac{\mathbf{J}(\mathbf{r}')}{\lvert\mathbf{r}-\mathbf{r}'\rvert}\,dV'$$

## 拉普拉斯方程的性质

拉普拉斯方程的解（**调和函数**）有几个特殊性质：

**① 无极值定理**
> 调和函数在区域内部不能取得最大值或最小值，极值只能在边界上取到。

**② 均值定理**
> 调和函数在任意一点的值，等于以该点为球心的任意球面上的平均值：
> $$\varphi(\mathbf{r}_0) = \frac{1}{4\pi R^2}\oiint_{|\mathbf{r}-\mathbf{r}_0|=R} \varphi\,dS$$

**③ 唯一性定理**
> 给定边界条件（第一类：$\varphi|_{\partial V}$，或第二类：$\partial\varphi/\partial n|_{\partial V}$），拉普拉斯方程的解**唯一确定**。

→ 唯一性定理是**镜像法**的理论基础：只要构造的镜像解满足相同边界条件，它就等于真实解。

## 在电磁学中的出现位置

```
有电荷区域        无电荷区域
     │                │
∇²φ = −ρ/ε₀      ∇²φ = 0
泊松方程          拉普拉斯方程
     │                │
  库仑积分         球谐函数展开
  （格林函数解）    （镜像法、分离变量法）
```

- 导体表面附近：边界是等势面，内部满足 $\nabla^2\varphi=0$
- 真空中磁矢势：$\mathbf{J}=0$ 处满足 $\nabla^2\mathbf{A}=0$（矢量拉普拉斯方程）
- 电磁波方程：加上时间项后，泊松方程推广为波动方程
