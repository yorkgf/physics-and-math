# Four Prices of Money (货币的四个价格)

## 概述

**Four Prices of Money** 是 Perry Mehrling 提出、Zoltan Pozsar 广泛应用的货币分析框架。该框架认为货币体系可以从四个"价格"维度来理解。

---

## 货币的四个价格

### 1. Par (平价)

**定义**：不同形式货币之间一对一的兑换价格

```
平价关系：
┌─────────────────────────────────────────────────────┐
│  准备金 ←──→ 现钞      (银行与央行的平价)            │
│  银行存款 ←──→ 银行存款  (银行间平价)                │
│  银行存款 ←──→ 现钞      (银行与公众的平价)          │
└─────────────────────────────────────────────────────┘
```

**关键洞察**：
- 平价确保交易无缝进行
- **2008 年平价被打破**——如果美联储没有介入，正常交易将无法进行
- 2022 年俄罗斯外汇储备被冻结也是平价危机

---

### 2. Interest (利率)

**定义**：未来货币的价格（今天的货币 vs 明天的货币）

```
利率层级：
┌────────────────────────────────────┐
│  Fed Funds Rate    (银行间无担保)   │
├────────────────────────────────────┤
│  Repo Rate         (有担保借贷)     │
├────────────────────────────────────┤
│  Libor/OIS         (国际市场)       │
├────────────────────────────────────┤
│  各类信用利差      (风险溢价)       │
└────────────────────────────────────┘
```

**关键洞察**：
- 利率是你放弃流动性、出借货币一段时间的价格
- 取决于对手方风险，利率各不相同
- Pozsar 关注短期融资市场的"管道"问题

---

### 3. Foreign Exchange (汇率)

**定义**：外国货币的价格

```
汇率体系：
┌─────────────────────────────────────────┐
│   固定汇率 (Peg)                        │
│   - 沙特里亚尔钉住美元                   │
│   - 港币联系汇率                         │
├─────────────────────────────────────────┤
│   浮动汇率                              │
│   - 欧元/美元                           │
│   - 日元/美元                           │
├─────────────────────────────────────────┤
│   管理浮动 (Dirty Float)                │
│   - 人民币汇率                          │
└─────────────────────────────────────────┘
```

**关键洞察**：
- 汇率危机（如 1997 年亚洲金融危机）是一种货币危机
- 货币武器化（冻结储备）会改变汇率风险评估

---

### 4. Price Level (物价水平)

**定义**：货币的实际购买力——商品和服务的价格

```
物价水平决定因素：
┌────────────────────────────────────────┐
│  名义因素：货币供应量、财政政策         │
├────────────────────────────────────────┤
│  实际因素：商品供给、供应链、战争        │
├────────────────────────────────────────┤
│  通胀 = 太多货币追逐太少商品            │
└────────────────────────────────────────┘
```

**关键洞察**：
- Pozsar 强调："你可以印钱，但不能印石油取暖或印小麦吃"
- 当前通胀是**结构性**的（去全球化、战争、供应链重构）
- 不是简单的需求管理问题

---

## 危机与四个价格

Pozsar 指出，每次重大危机都涉及其中一个价格：

| 危机 | 年份 | 受冲击的价格 | 表现形式 |
|------|------|-------------|---------|
| 亚洲金融危机 | 1997 | Foreign Exchange | 汇率崩溃、货币贬值 |
| 次贷危机 | 2008 | Par | 回购市场冻结、平价破裂 |
| 欧洲主权危机 | 2010-2012 | Interest | 主权债券利差飙升 |
| 新冠疫情 | 2020 | Interest + Price Level | 流动性危机 + 后续通胀 |
| 俄乌冲突 | 2022 | Price Level + FX | 大宗商品冲击 + 储备货币重构 |

**结论**：每次重大危机都是货币危机的某种形式。

---

## 框架应用

### 分析当前环境

```
2022-2024 年的四个价格状态：

Par:        稳定（但信任下降，黄金需求上升）
Interest:   快速上升（央行紧缩）
FX:         剧烈波动（美元走强，新兴市场承压）
Price Level:结构性上升（去全球化、战争）
```

### 政策含义

- **货币政策**主要影响 Interest
- **汇率政策**影响 FX
- **财政政策/供给政策**影响 Price Level
- **金融稳定政策**维护 Par

---

## 与 Bretton Woods III 的联系

Pozsar 使用四个价格框架分析 Bretton Woods III：

- **Par**：内部货币 vs 外部货币的平价重构
- **Interest**：更高的期限溢价 (Term Premium)
- **FX**：多极储备货币世界
- **Price Level**：商品通胀结构性上升
