# FX hedging costs (FX对冲成本)

## 定义

外国投资者购买美元资产需要对冲汇率风险，通过FX swap借入美元、贷出本币，产生的成本就是FX对冲成本。

计算公式：
```
FX对冲成本 = 3个月美元OIS + Libor-OIS spread + 交叉货币基差(相对于美元Libor)
```

---

## 关键命题（Pozsar）

后危机时代，外国私人投资者成为美国国债边际买家，他们都需要对冲汇率风险 → **FX对冲成本决定了美国国债对他们有没有吸引力**。

当：
```
3个月FX对冲成本 > 10年期美国国债收益率
```
→ **对冲后收益率为负**，外国投资者不会买，这就是**相对于对冲成本的曲线inversion**。

这才是真正有经济意义的inversion，传统的2s/10s、3s/10s衡量已经过时。

## 2019年情况

- 日元投资者对冲成本约 **3.0%**
- 10年期美债收益率约 **2.7%**
- 对冲后收益率 **-0.15%** → 不买

- 欧元投资者对冲成本约 **2.7%**
- 10年期美债收益率约 **2.7%**
- 对冲后收益率 **-0.05%** → 几乎不买

因此，外国投资者罢工，一级dealer必须接盘，repo市场压力增大，美联储缩表提前结束。
