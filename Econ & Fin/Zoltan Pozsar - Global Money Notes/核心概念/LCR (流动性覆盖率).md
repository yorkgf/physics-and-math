# LCR (流动性覆盖率)

## 定义

**流动性覆盖率（Liquidity Coverage Ratio, LCR）** 是巴塞尔III最重要的改革之一，要求银行持有足够的高流动性资产（HQLA）来应对30天的资金流出压力。

公式：
```
LCR = 优质流动性资产储备 / 未来30天的资金净流出量 ≥ 100%
```

---

## 对货币市场的结构性影响

Pozsar认为，LCR是后危机时代改变一切的监管改革：

1. **创造了全球性的准备金要求制度**
   - 以前只有美国境内活期存款需要准备金
   - LCR要求**所有30天内到期短期负债**都需要持有HQLA，包括离岸欧洲美元
   - 这意味着全球都需要美元准备金/HQLA

2. **非经营性存款的100% HQLA要求**
   - 机构非经营性存款 → 100% HQLA requirement
   - 零售存款 → 只要 5-20% HQLA requirement
   - 所以银行**主动赶走**机构非经营性存款

## 关键推论（Pozsar）

**根本没有超额准备金这回事**。

所有QE创造的准备金现在都是LCR要求必需的HQLA，一分钱都不多余。美联储资产负债表不可能大幅缩表。
