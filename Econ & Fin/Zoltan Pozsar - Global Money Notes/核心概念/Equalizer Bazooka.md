# Equalizer Bazooka

## 定义

"Equalizer Bazooka" 是Zoltan Pozsar提出的最优设计，用于standing repo facility：

- 定价：o/n RRP + 25 bps → 正好在美联储目标区间顶部，作为ceiling
- 开放对象：**只对G-SIBs开放**
- 目标：equalize collateral and reserves → 让抵押品和准备金完全等价，解除银行日内流动性约束

> [[Global Money Notes #25 - Design Options for an o/n Repo Facility]]

---

## 为什么叫"bazooka"

借鉴了Mario Draghi "whatever it takes" bazooka的说法：

- 只要宣布存在这个工具，**市场就会相信它**
- 银行会释放预防性持有的超额准备金 → 实际上不一定需要使用
- "It wouldn't have to be used for it to have an impact" → 存在就是力量，不用也能解决问题

就像Draghi的OMT，宣布了之后问题就解决一半了，不一定需要实际使用。

---

## 为什么最优

- 只给G-SIBs开放：解决了道德风险、公平性问题
- 定价在顶部：正好作为ceiling，管住隔夜利率不会突破目标区间
- 释放约$5000亿闲置准备金 → 改善全球美元流动性，压缩跨货币基差
