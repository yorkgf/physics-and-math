# 影子银行体系 (Shadow Banking System)

## 定义

**影子银行**是指在传统商业银行体系之外，从事"借短放长"货币交易功能的金融机构和市场活动。

>Pozsar定义：影子银行是**并行银行体系 (Parallel Banking System)**，通过货币市场工具（如回购、商业票据）融资，持有长期信贷资产。

---

## 核心特征

| 特征 | 说明 |
|------|------|
| **期限转换** | 以短期负债融资持有长期资产 |
| **信用转换** | 通过证券化将低流动性资产转化为高流动性负债 |
| **流动性转换** | 创造类似货币的短期债权工具 |
| **监管套利** | 不受传统银行监管约束（资本充足率、准备金等） |

---

## 影子银行 vs 传统银行

```
传统银行：
存款人 → 商业银行 → 贷款
         ↓
      央行流动性支持 + 存款保险

影子银行：
投资者 → 货币市场基金/券商/SPV → 长期债券/ABS/MBS
         ↓
      无央行支持 + 无存款保险 → 依赖回购市场流动性
```

---

## 影子银行体系构成

1. **发起机构** (Originators)：发放贷款的银行或非银机构
2. **证券化工具** (SPV/SPE)：将贷款打包成证券
3. **交易商** (Dealers)：券商，承担做市和库存职能
4. **货币市场基金** (MMFs)：提供短期融资
5. **回购市场** (Repo Market)：影子银行的"存款"机制

---

## 2008年危机中的角色

危机前的影子银行体系是**私人货币交易**的核心：

- 券商通过回购为债券库存融资
- 货币市场基金购买回购协议（视为"安全"资产）
- 当次贷违约→回购抵押品质量受质疑→回购挤兑→去杠杆螺旋

> 这就是**私人货币交易体系的内生性崩溃**。

---

## 危机后的演变

巴塞尔III（SLR、LCR）使私人货币交易成本过高：

- 私人部门退出 → **影子银行萎缩**
- 美联储通过QE接手 → **公共货币交易崛起**

影子银行体系并未消失，而是被**公共部门货币交易体系**所补充和替代。
