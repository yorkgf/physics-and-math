# Hierarchy of Money (货币层级)

## 概述

**货币层级 (Hierarchy of Money)** 是理解现代货币体系的核心框架，由 Perry Mehrling 和 Zoltan Pozsar 发展完善。它描述了不同形式货币之间的等级关系，以及这种层级如何在压力和危机中显现。

> "我们通常从工具、价格和中介的角度思考货币及其层级（黄金、准备金、存款；有担保、无担保、外汇互换；央行、银行、交易商），而不是从货币和国家的角度。"
> — Zoltan Pozsar

```
┌─────────────────────────────────────────────────────────┐
│                   传统货币层级                           │
│              （从最安全到风险最高）                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  第1层：黄金 (Gold)                                     │
│         ↓                                               │
│  第2层：央行准备金 (Central Bank Reserves)              │
│         ↓                                               │
│  第3层：银行存款 (Bank Deposits)                        │
│         ↓                                               │
│  第4层：回购协议 (Repos)                                │
│         ↓                                               │
│  第5层：商业票据 (Commercial Paper)                     │
│         ↓                                               │
│  第6层：其他短期债务                                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## Pozsar 的扩展：加入国家维度

2022 年后，Pozsar 提出应将**国家 (State)** 纳入货币层级：

```
┌─────────────────────────────────────────────────────────┐
│              扩展的货币层级（含国家）                     │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  第1层：黄金/外部货币 (Outside Money)                   │
│         ↓                                               │
│  第2层：央行准备金                                      │
│         ↓                                               │
│  ★第2.5层：国家/主权 (State/Sovereignty) ★              │
│         ↓                                               │
│  第3层：银行存款                                        │
│         ↓                                               │
│  第4层：回购协议                                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 为什么加入国家？

```
俄乌冲突的教训：

内部货币 (Inside Money)              外部货币 (Outside Money)
├─ 外汇储备                          ├─ 黄金（自主保管）
├─ 美国国债                          ├─ 实物大宗商品
├─ 存放在他国的资产                  └─ 难以被制裁
└─ 可被冻结、被制裁

→ 国家的治理能力直接影响货币的实际价值
```

## 货币层级与流动性层级

```
并行的两个层级：

        货币层级                           流动性层级
    (Quality of Money)               (Quality of Liquidity)
          ↓                                   ↓
      黄金/储备                         货币后盾
          ↓                                   ↓
      银行存款                          最后贷款人
          ↓                                   ↓
      回购协议                          交易商流动性
          ↓                                   ↓
      商业票据                          私人流动性期权
```

## 层级在压力下的表现

### 正常时期

```
平价 (Par) 维持：

1美元准备金 = 1美元存款 = 1美元回购 = 1美元现钞
     ↓            ↓           ↓           ↓
   央行         银行        交易商       公众
```

### 危机时期

```
平价破裂，层级显现：

1美元准备金
     ↓
0.99美元存款 （银行信用风险折价）
     ↓
0.95美元回购 （抵押品质量折价）
     ↓
0.90美元商业票据 （流动性折价）

→ 资金向上层"逃向质量" (Flight to Quality)
```

## 2008 年危机：平价破裂的案例

```
危机前的私人货币创造：

现金池 (Cash Pools)
       ↓
  回购协议 (Repos)
       ↓
  交易商资产负债表
       ↓
  杠杆债券组合

问题：
- 回购协议被视为"货币"
- 但没有存款保险和央行支持
- 次贷违约 → 抵押品质量受质疑 → 回购挤兑
```

### 美联储的干预

```
平价修复：

危机前：                         危机后：
回购 = 货币                      准备金 = 货币
（私人创造）                     （公共创造）
     ↓                               ↓
交易商资产负债表              美联储资产负债表
     ↓                               ↓
无官方支持                    有央行支持

→ 货币层级从私人向公共转移
```


## 投资含义

### 1. 理解"无风险"的相对性

```
"无风险"资产层级：

理论无风险 ──→ 本币国债
                ↓
实际操作 ──→ 需要考虑国家风险
                ↓
极端情况 ──→ 即使是美国国债也可能被政治化
```

### 2. 多极货币世界

- 美元主导地位面临挑战
- 人民币、欧元作为替代
- 黄金作为中性储备资产

### 3. 资产配置调整

```
在货币层级重构的环境中：

增加：黄金、大宗商品（外部货币）
减少：对单一法币的过度暴露
关注：国家的治理能力
```
