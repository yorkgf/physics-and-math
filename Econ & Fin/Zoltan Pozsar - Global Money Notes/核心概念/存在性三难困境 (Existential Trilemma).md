# 存在性三难困境 (Existential Trilemma)

## 定义

**美联储现在面临一个新的不可能三角，原本这是固定汇率新兴市场的问题，现在美联储也遇到了：**

你不可能同时拥有：
1. **对银行资产负债表的约束** ← 巴塞尔III SLR/LCR要求
2. **离岸美元与在岸美元的平价汇率** ← 跨货币基差接近零（平价）
3. **面向国内目标的货币政策** ← 加息周期正常化

三者必须放弃一个。

---

## 为什么会出现这个三难

- 货币基金改革掐断了欧洲美元市场最大的融资动脉（prime基金）
- 边际美元需求现在必须由美国银行通过FX掉期满足
- 美国银行受到SLR约束，资产负债表空间有限
- 每多一笔FX掉期交易，边际成本上升 → 跨货币基差越来越负
- 离岸美元比在岸美元更贵 → 平价破了

## 可能的解决方式

| 放弃哪个 | 结果 |
|---------|------|
| 放弃对银行资产负债表的约束 | 放松SLR，解放资产负债表 → 套利回来，基差回到平价 → 可以继续加息 |
| 放弃平价 | 让基差越来越负（到-150bps）→ 美元升值 → 进口通缩 → 加息受阻 |
| 放弃国内货币政策 | 停止加息周期 → 无法正常化 |
| **QEE（量化欧洲美元宽松）** | 美联储成为**Dealer of Last Resort**，通过美元互换线提供美元 → 放弃对美联储资产负债表规模的控制，但保住平价和加息 |
