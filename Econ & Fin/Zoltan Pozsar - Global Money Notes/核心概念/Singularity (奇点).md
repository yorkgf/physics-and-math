# Singularity (奇点)

## 概述

**奇点 (Singularity)** 是 Zoltan Pozsar 在 2020 年新冠疫情期间提出的概念，借用了天体物理学的术语来描述货币市场的一种极端状态：当利率接近零下限时，融资利差的"无限密度"。

> "在天体物理学中，奇点指的是黑洞中心的无限密度，以及宇宙大爆炸前的最早状态。在货币市场中，我们将奇点定义为零下限处的融资利差'无限'密度。"
> — Zoltan Pozsar, Global Money Notes #30

## 货币市场的奇点状态

### 定义

```
奇点状态的特征：

1. 利率接近零下限 (Zero Lower Bound)
2. 各类融资利差极度压缩
3. Libor-OIS 利差收窄至 10-20 bps
4. 期限溢价消失
```

### 2020 年疫情期间的奇点

```
┌─────────────────────────────────────────────────────────┐
│           2020年4-6月的奇点状态                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  美联储行动：                                           │
│  • QE 以每周 $350亿 速度增加准备金                      │
│  • 财政部现金余额下降释放流动性                         │
│  • 外汇互换额度提供美元流动性                           │
│  • 常备回购便利支持交易商                               │
│                                                         │
│  结果：                                                 │
│  • 流动性泛滥                                           │
│  • 利差压缩至极低水平                                   │
│  • 奇点临近                                             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 奇点的引力效应

### 利差压缩

```
奇点的"引力"：

Libor-OIS 利差
     │
10bp ├────────────────────────  ← 奇点状态
     │                      ╱
20bp │                   ╱
     │                ╱
50bp │             ╱
     │          ╱
100bp│       ╱     ← 正常状态
     │    ╱
     └──┴──────────────────────→ 时间
       危机前  危机  奇点
```

### 对货币政策的影响

```
奇点状态下的政策选择：

选项1：继续 QE
       ↓
   奇点加深，利差进一步压缩
       ↓
   金融机构盈利能力下降

选项2：收益率曲线控制 (YCC)
       ↓
   直接限制特定期限收益率
       ↓
   但可能扭曲市场信号

选项3：负利率
       ↓
   突破零下限
       ↓
   金融体系重构
```

## 奇点的退出

### 从奇点到正常化

```
退出奇点的条件：

1. 减少流动性供应
   • 停止 QE
   • 缩减资产负债表
   • 提高利率

2. 增加安全资产供给
   • 财政赤字扩大
   • 国债发行增加

3. 金融机构修复
   • 修复盈利能力
   • 重建资本缓冲
```

### 风险

```
退出奇点的风险：

快速退出：                    缓慢退出：
    ↓                            ↓
市场动荡                      低利率陷阱
    ↓                            ↓
流动性危机                    僵尸企业增加
    ↓                            ↓
可能需要                      生产力下降
重新宽松
```

## 奇点与期限溢价

### 期限溢价的消失与回归

```
奇点状态：                    正常状态：

期限溢价 ≈ 0                  期限溢价 > 0
    ↓                            ↓
长期债券昂贵                  长期债券需要补偿
    ↓                            ↓
收益率曲线平坦                收益率曲线向上倾斜
    ↓                            ↓
投资者被迫承担                投资者获得
更多风险                      风险补偿

Pozsar 预测：期限溢价将结构性上升
```

## 投资含义

### 1. 债券投资

```
奇点环境：                    后奇点环境：

长期债券收益低                期限溢价回归
    ↓                            ↓
缩短久期                      关注收益率曲线变化
寻找替代收益                  通胀保护债券
```

### 2. 资产配置

```
传统 60/40 在奇点环境中失效：

原因：
• 债券收益率过低
• 股票风险溢价被压缩
• 股债相关性可能转正

调整：
• 增加实物资产（大宗商品、房地产）
• 增加另类投资
• 关注绝对收益策略
```

### 3. 货币环境

```
奇点退出后的货币走势：

美元：短期强势（避险需求）
      长期结构性压力（债务水平）

新兴市场：分化加大
         外债负担是关键

黄金：对冲货币贬值的工具
```
