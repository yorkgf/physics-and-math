# 战争金融 (War Finance)

## 概述

**战争金融 (War Finance)** 是 Zoltan Pozsar 在 2022 年俄乌冲突后提出的核心概念，描述了在战争和地缘政治冲突背景下，货币体系、财政政策和央行政策的根本性转变。

> "这是战争金融……美联储似乎致力于确保抵押品供应（美国国债发行）不会以压倒性优势超过准备金供应。"
> — Zoltan Pozsar, 2020 (新冠疫情期间)
> 
> "战争颠覆了货币的主导地位，并充当新货币体系诞生的助产士。"
> — Zoltan Pozsar, 2022

## 历史背景

### 历史上的战争金融

```
┌─────────────────────────────────────────────────────────┐
│                  战争与货币体系变革                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   一战 (1914-1918)      金本位暂停 → 通胀               │
│   二战 (1939-1945)      Bretton Woods I 诞生           │
│   冷战 (1947-1991)      美元体系巩固                   │
│   反恐战争 (2001-)      量化宽松 (QE) 先例             │
│   俄乌冲突 (2022-)      Bretton Woods III?             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 关键洞见

战争时期，常规的货币政策和财政约束被打破：
- **利率不敏感**：政府借款不再受利率水平约束
- **财政主导**：央行被迫配合财政需求
- **货币武器化**：外汇储备可被冻结

## 俄乌冲突的金融维度

### 1. 储备冻结的 precedents

```
被冻结的央行储备：
┌──────────────────────────────────────────┐
│  委内瑞拉 (2019)    $3.8亿 + $9亿黄金     │
│  阿富汗 (2022)      $70亿                 │
│  俄罗斯 (2022)      $3,880亿              │
├──────────────────────────────────────────┤
│  信号：西方储备不再"无风险"               │
└──────────────────────────────────────────┘
```

### 2. 内部货币 vs 外部货币

```
┌─────────────────────────────────────────────┐
│      内部货币 (Inside Money)                 │
│  - 外汇储备                                   │
│  - 以他国央行/银行负债形式持有                 │
│  - 可被冻结、被制裁                           │
├─────────────────────────────────────────────┤
│      外部货币 (Outside Money)                │
│  - 黄金（自主保管）                           │
│  - 实物大宗商品                               │
│  - 难以被制裁                                 │
└─────────────────────────────────────────────┘
```

俄罗斯将其黄金储备全部存放在国内——这使得西方冻结其储备的能力受限。

### 3. 债务违约的悖论

```
西方对俄罗斯的债权 vs 债务：

俄罗斯欠西方：    ~$400亿
西方欠俄罗斯：    ~$3,880亿（已被冻结）

如果俄罗斯因被制裁而无法偿债，
谁才是真正的违约方？
```

## 战争金融的特征

### 1. 商品即货币

```
"我们的商品，你们的问题"
       —— 产油国对西方的回应

石油、天然气、粮食、金属 = 新的"硬通货"
```

### 2. 财政主导 (Fiscal Dominance)

```
正常时期：            战争时期：
央行独立 ←────→    央行配合财政
控制通胀             配合战争融资
利率敏感             利率不敏感
```

### 3. 西方的四大任务 (The To-Do List)

Pozsar 提出的西方应对策略：

| 任务 | 含义 | 投资含义 |
|------|------|---------|
| **Re-arm** | 重新武装 | 国防支出激增 |
| **Re-shore** | 产业链回流 | 制造业投资 |
| **Re-stock** | 战略储备 | 大宗商品需求 |
| **Re-wire** | 能源转型 | 绿色投资 |

这些任务的共同特征：
- **必须执行**：无论利率多高
- **资本密集**：大量借贷需求
- **商品密集**：通胀压力

## 战争金融 vs 正常金融

| 维度 | 正常金融 | 战争金融 |
|------|---------|---------|
| **利率敏感性** | 高 | 低/无 |
| **通胀目标** | 2% | 次要 |
| **央行独立性** | 高 | 低 |
| **货币锚定** | 信用/内部货币 | 商品/外部货币 |
| **国际合作** | 全球化 | 阵营化 |
| **风险偏好** | 追求收益 | 安全第一 |

## 投资含义

### 1. 资产配置变化

```
传统 60/40 组合：          Pozsar 建议：
60% 股票                   20% 现金
40% 债券                   40% 股票
                           20% 债券
                           20% 大宗商品
```

### 2. 新的"安全资产"

```
传统安全资产：              战争金融时代安全资产：
- 美国国债                 - 黄金
- 投资级债券               - 大宗商品
- 现金                     - 股票（实物资产暴露）
                           - 短久期通胀保护债券
```

### 3. 期限溢价上升

- 长期债券的"无风险"属性受到质疑
- 债券需要提供更高的期限溢价才能吸引投资者
- "价值投资在股票已死，因为债券没有价值"
