# 全球美元体系

## 核心命题（Pozsar）

后危机时代，全球美元体系发生了结构性变化：

1. **巴塞尔III**约束了银行资产负债表
2. **货币基金改革**掐断了欧洲美元市场的主要融资动脉
3. 全球美元短缺成为结构性问题，不是周期性问题
4. 离岸美元现在比在岸美元更贵，跨货币基差持续为负
5. 美联储必须成为**Dealer of Last Resort**，通过美元互换线提供量化欧洲美元宽松（QEE）

## 核心矛盾

- ECB和BoJ持续大规模QE → 本地投资者要买美元资产赚收益
- 这些投资者需要对冲汇率风险 → 需要FX swap借美元
- 私人银行资产负债表空间不够 → 无法满足所有需求
- 美联储必须提供公共资产负债表空间 → 否则美元短缺越来越严重
