### 战争金融 (War Finance)

**定义**：在战争和重大危机时期，货币政策和财政政策的正常约束被打破的状态。

**特征**：
- 利率不敏感：政府借款不再受利率约束
- 财政主导：央行被迫配合财政需求
- 货币武器化：外汇储备可被冻结

```
战争金融的四大任务：

1. Re-arm      → 重新武装
2. Re-shore    → 产业链回流
3. Re-stock    → 战略储备
4. Re-wire     → 能源转型

共同特征：资本密集、商品密集、利率不敏感
```

→ [[战争金融 (War Finance)]]

### 奇点 (Singularity)

**定义**：零利率下限处融资利差的"无限"密度状态。

```
奇点特征：

Libor-OIS 利差 → 10-20 bps
期限溢价 → 接近零
收益率曲线 → 极度平坦

Pozsar："你可以印钱，但不能印石油或小麦"
→ 从名义问题到实际问题的转变
```

→ [[Singularity (奇点)]]

## 四、Bretton Woods III 框架

### 货币体系的演变

```
┌─────────────────────────────────────────────────────────────┐
│                  国际货币体系演变                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   BW I          BW II               BW III?                 │
│   (1944-71)     (1971-2022)         (2022-?)                │
│                                                             │
│   黄金          美元（内部货币）     大宗商品（外部货币）     │
│     ↓              ↓                      ↓                 │
│   美元          美国国债             黄金、石油、粮食         │
│                                                             │
│   实物锚         信用锚               实物锚回归              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 核心论点

1. **从内部货币到外部货币**
   - 内部货币（外汇、国债）可被冻结
   - 外部货币（黄金、商品）自主保管

2. **商品储备替代外汇储备**
   - "我们的商品，你们的问题"
   - 产油国重新思考货币锚定

3. **多极货币世界**
   - 人民币通过互换结算贸易
   - 不需要持有大量人民币储备

→ [[Bretton Woods III]]

## 八、关键洞见总结

### Pozsar 的核心贡献

1. **揭示了影子银行的货币创造功能**
   - 回购协议是影子银行的"存款"
   - 但没有存款保险和央行支持

2. **解释了 2008 危机后的结构性变化**
   - 从私人 carry 到公共 carry
   - 美联储成为边际货币创造者

3. **预测了 2019 回购危机**
   - 准备金稀缺导致货币市场动荡
   - 监管改革的意外后果

4. **提出 Bretton Woods III 框架**
   - 地缘货币体系转变
   - 大宗商品作为新锚定

5. **强调"管道"的重要性**
   - 金融体系的技术细节不是背景
   - 而是决定市场运作的核心
