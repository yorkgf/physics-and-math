# 1.2 金融市场分类体系

## 一、按期限分类（最基本分类）

```
┌─────────────────────────────────────────────────┐
│                 金融市场分类                      │
│                                                 │
│  货币市场 ──── 短期（<1年）──── 短期融资           │
│    │                              (银行间拆借/商票) │
│  资本市场 ──── 长期（≥1年）──── 资本形成           │
│    │                              (股票/债券)       │
│  保险市场 ──── 风险对冲 ───────── 风险管理         │
│    │                                          │
│  外汇市场 ──── 汇率交易 ───────── 国际资本流动     │
│    │                                          │
│  衍生品市场 ── 风险/收益 ───────── 风险定价       │
└─────────────────────────────────────────────────┘
```

## 二、各子市场的核心特征对比

| 市场 | 期限 | 核心功能 | 主要工具 | 典型利率/价格 |
|------|------|---------|---------|-------------|
| **货币市场** | 隔夜~1年 | 流动性管理 | 国库券/商业票据/同业拆借 | Fed Funds / LIBOR / SOFR |
| **债券市场** | 1年~30年+ | 长期融资/利率定价 | 国债/企业债/市政债 | 国债收益率曲线 |
| **股票市场** | 永久 | 权益融资/公司治理 | 普通股/优先股 | P/E倍数/指数 |
| **外汇市场** | 即期/远期 | 汇率定价/国际结算 | 即期/远期/掉期 | 汇率 |
| **大宗商品市场** | 现货/期货 | 商品定价/通胀对冲 | 原油/黄金/农产品 | CRB指数 |
| **衍生品市场** | 各种期限 | 风险对冲/投机 | 期权/期货/互换 | 隐含波动率 |

## 三、按交易工具分类

### 债务市场（Debt Market）
- **核心特征**：发行方承诺还本付息，持有者是债权人
- **风险**：信用风险（发行方违约）、利率风险（价格对利率反向变动）
- **例子**：国债、企业债、抵押贷款、MBS

### 权益市场（Equity Market）
- **核心特征**：持有者是公司所有者，分享公司剩余收益
- **风险**：市场风险、系统性风险
- **例子**：普通股、优先股

### 衍生品市场（Derivatives Market）
- **核心特征**：价值从基础资产派生，不持有底层资产
- **功能**：对冲风险 / 投机获利 / 价格发现
- **例子**：期权、期货、互换、信用衍生品（CDS）

## 四、按市场结构分类

```
一级市场（发行市场）
  └── 首次发行（IPO）、增发、直接发行
  └── 特点：没有固定场所，投行承销

二级市场（流通市场）
  └── 交易所市场（纽交所、纳斯达克）
  └── 场外市场（OTC，如债券柜台市场）
  └── 特点：提供流动性，价格发现
```

## 五、按参与主体分类

| 市场 | 主要参与者 | 目的 |
|------|----------|------|
| 银行间市场 | 央行、银行 | 流动性调剂 |
| 政府债券市场 | 央行、银行、基金、外国投资者 | 避险、储备管理 |
| 企业融资市场 | 企业、投行、基金 | 融资 |
| 零售市场 | 居民、散户投资者 | 储蓄、投资 |
| 批发市场 | 机构投资者、对冲基金 | 套利、杠杆交易 |

## 六、金融市场之间的层级关系

```
                        ┌──────────────┐
                        │   央行/货币政策 │
                        │   (基准利率)   │
                        └──────┬───────┘
                               │ 决定
         ┌─────────────────────┼─────────────────────┐
         ↓                     ↓                     ↓
  ┌────────────┐        ┌────────────┐        ┌────────────┐
  │  货币市场   │        │  债券市场   │        │  外汇市场   │
  │ (基准利率) │        │(利率曲线)  │        │ (汇率)    │
  └─────┬──────┘        └─────┬──────┘        └─────┬──────┘
        │                     │                     │
        │ 套利/预期            │                     │
        ↓                     ↓                     ↓
  ┌──────────────────────────────────────────────────┐
  │                  股票市场                         │
  │           (对未来盈利的预期折现)                   │
  └──────────────────────────────────────────────────┘
                               │
                               ↓
  ┌──────────────────────────────────────────────────┐
  │               大宗商品市场                         │
  │            (实体需求/通胀预期)                      │
  └──────────────────────────────────────────────────┘
                               │
                               ↓
  ┌──────────────────────────────────────────────────┐
  │                   实体经济                        │
  │              (GDP/就业/通胀)                      │
  └──────────────────────────────────────────────────┘
```

## 七、关键关系：利率是所有市场的锚

```
央行政策利率
    ↓ 决定
货币市场利率（短期利率）
    ↓ 通过利率期限结构
债券市场利率（长期利率）
    ↓ 影响
股票市场估值（P/E = 1/r）
    ↓ 反映
企业投资意愿
    ↓ 影响
经济增长与通胀
    ↓ 反馈
央行货币政策
```
