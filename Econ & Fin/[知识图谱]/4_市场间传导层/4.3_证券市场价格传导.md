# 4.3 证券市场价格传导

## 一、证券市场的层级与联动

证券市场价格传导的本质是**风险重新定价**的过程：

```
                    证券市场联动层级
                          │
      ┌──────────────────┼──────────────────┐
      ↑                  ↑                  ↑
   无风险利率          信用利差           风险偏好
  (国债收益率)       (企业债券利差)      (股票/大宗商品)
      │                  │                  │
  货币市场利率        债券市场定价        高风险资产
  决定估值基准        反映信用环境         被重新定价
```

## 二、利率传导：国债收益率 → 所有资产

### 核心逻辑：折现率

```
国债收益率是所有资产的"定价之锚"：

股票价值 = Σ(未来盈利) / (1+r)^t
              ↑
          r = 折现率 = 无风险利率 + 风险溢价

当国债收益率↑ → 所有资产折现率↑ → 所有资产价值↓
当国债收益率↓ → 所有资产折现率↓ → 所有资产价值↑
```

### 传导链条

```
第一步：央行加息（或市场通胀预期上升）
        ↓
第二步：短期国债收益率↑（反映政策利率预期）
        ↓
第三步：中期国债收益率↑（2-10年期）
        ↓
第四步：长期国债收益率↑（10-30年期，形成收益率曲线）
        ↓
第五步：企业债券利率↑（国债收益率 + 信用利差）
        ↓
第六步：股票折现率↑（PE倍数下降）
        ↓
第七步：股价下跌（同时盈利预期也可能下调）
        ↓
第八步：财富效应 → 消费↓ → 实体经济↓
```

## 三、信用利差传导

### 什么是信用利差扩大？

```
信用利差 = 企业债收益率 - 国债收益率

正常时期（经济好）：
  国债3% + 信用利差1.5% = 企业债4.5%
  （风险可控，企业正常经营）

危机时期：
  国债3% + 信用利差8% = 企业债11%
  （市场认为违约风险大增，企业融资成本暴涨）
```

### 信用利差扩大的传导

```
信用利差急剧扩大
        │
        ├──→ 企业融资成本急剧上升
        │    （债券发行几乎停滞 → 企业被迫削减投资）
        │
        ├──→ 银行放贷标准收紧（担心企业违约）
        │    → 贷款难度↑
        │
        ├──→ 高收益债券（垃圾债）价格暴跌
        │    → 持有垃圾债的基金净值暴跌
        │    → 投资者赎回 → 基金被迫抛售 → 进一步下跌
        │
        ├──→ 违约率预期上升
        │    → 评级机构下调企业评级
        │    → 触发更多被动卖出
        │
        └──→ 违约→破产→失业→消费↓→经济↓
```

## 四、股票市场的板块轮动传导

```
经济周期的不同阶段，板块表现差异：

复苏期（经济↑ + 利率↓）：
  金融股↑↑（银行受益利差扩大）
  消费股↑↑（可选消费恢复）
  工业股↑↑（资本开支增加）

过热期（经济↑ + 通胀↑）：
  能源股↑↑↑（油价上涨）
  材料股↑↑（大宗商品涨价）
  科技股↓（估值压力）

滞胀期（经济↓ + 通胀↑）：
  黄金↑↑（避险）
  必需消费品↑（抗通胀）
  股票全面下跌

衰退期（经济↓↓ + 通胀↓）：
  国债↑↑（避险）
  现金为王
  股票/商品全面下跌
```

## 五、杠杆资金的放大效应

```
                        杠杆放大机制

正常市场（杠杆率低）：
  股价↓5% → 投资者组合↓5% → 恐慌有限

高杠杆市场：
  股价↓5% → 追加保证金通知
        ↓
  杠杆投资者被迫平仓
        ↓
  强制卖出 → 股价进一步下跌
        ↓
  触发更多追加保证金
        ↓
  更多强制平仓
        ↓
  流动性踩踏

2008年案例：
  量化交易基金使用相同模型
        ↓
  股价跌破关键技术位
        ↓
  触发程序化止损
        ↓
  雷曼股价单日暴跌50%
  （基本面价值 vs 流动性清算价的巨大差异）
```

## 六、市场间传染（Market Contagion）

### 什么是市场传染？

当一个市场发生危机，**原本不相关的市场**也开始暴跌的现象。

```
传统金融理论：
  各市场高度分散化 → 个别市场危机不会传染

实际发生（尤其是2008年）：
  各市场相关性在危机时趋近于1
  → 分散化完全失效
  → 传染无处不在
```

### 传染的三大机制

#### 机制1：流动性传染
```
A市场危机
  → A市场投资者需要流动性
  → 抛售流动性好的其他资产
  → 原本不相关的B市场被拖累下跌

2008年：
  MBS/CDO市场危机
    → 持有CDO的投资者被追缴保证金
    → 抛售黄金/国债等"无关"资产
    → 黄金在2008年10月也出现暴跌
```

#### 机制2：风险偏好传染
```
A市场危机 → 投资者风险偏好↓（变保守）
  → 资金从高风险资产流向低风险资产
  → 所有高风险资产同步下跌

传染范围：
  股票↓ → 大宗商品↓ → 新兴市场股票↓ → 高收益债券↓
  → 房地产信托(REITs)↓ → 货币↓

唯一下跌的是：美元、美国国债、黄金（避险资产）
```

#### 机制3：资产负债表传染
```
A机构因A市场亏损导致资本不足
  → 被动去杠杆化（监管要求）
  → 抛售所有资产
  → B市场、C市场、D市场全面承压

B机构看到A机构问题
  → 担心自己的交易对手也受影响
  → 不愿与A机构进行任何交易
  → A机构难以融资/平仓
  → 更多抛售

银行间市场冻结 [[5.5_流动性危机]]
```

## 七、2008年危机的市场传染时间轴

```
2007年初：次级贷违约率上升（MBS价格开始下跌）
    ↓ 传导
2007年7月：CDO大规模减值（贝尔斯登基金清盘）
    ↓ 传导
2007年9月：信用利差急剧扩大（高收益债券利差翻倍）
    ↓ 传导
2008年3月：贝尔斯登危机（股价单日暴跌50%）
    ↓ 传导
2008年9月：雷曼倒闭（全球市场同步崩盘）
    ↓ 传导
2008年9月16日：AIG危机（CDS赔付担忧）
    ↓ 传导
2008年9月17日：货币市场基金跌破净值（"打破1美元"）
    ↓ 传导
2008年9-10月：所有资产被抛售（VIX飙至80）
    ↓ 传导
2008年10月：LIBOR-OIS利差创历史新高（银行间冻结）
    ↓ 传导
2008年11月：企业债券市场冻结（融资停滞）
    ↓ 传导
2009年3月：标普500跌至666点（底部确认）
```

## 八、防止市场传染的机制

| 机制 | 作用 | 2008年问题 |
|------|------|-----------|
| 熔断机制（Circuit Breaker） | 单日跌幅过大时暂停交易 | 2008年10月仍暴跌9%+ |
| 央行流动性注入 | 向市场提供美元流动性 | 美元互换额度缓解但不够及时 |
| 存款保险/国债担保 | 防止银行挤兑 | 初期不够有力，后续TARP才到位 |
| 系统重要性机构监管 | 防止"大而不能倒" | 监管赶不上创新 |
