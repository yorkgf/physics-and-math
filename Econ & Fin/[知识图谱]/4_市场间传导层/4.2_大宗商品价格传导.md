# 4.2 大宗商品价格传导

## 一、大宗商品的价格决定机制

大宗商品价格由**三重属性**共同决定：

```
                    商品价格三角
                        │
    ┌──────────────────┼──────────────────┐
    ↑                  ↑                  ↑
  实物供需         金融属性           避险属性
(产量/库存/       (期货/ETF/         (黄金/白银/
 天气/地缘)        投机资金)           恐慌需求)
    │                  │                  │
    └──────────────────┼──────────────────┘
                       ↓
               市场价格(现货/期货)
```

## 二、大宗商品价格传导的五大路径

### 路径1：通胀预期传导（最重要的传导）

```
商品价格上涨
        │
        ├──→ 原材料成本↑ → PPI（生产者价格）↑
        │
        ├──→ 消费品价格↑ → CPI（消费者价格）↑
        │
        ├──→ 通胀预期↑ → 名义利率预期↑
        │
        ├──→ 国债被抛售 → 债券收益率↑（价格↓）
        │   （通胀债券TIPS收益率上升）
        │
        └──→ 股票估值承压（P/E = 1/r, r上升则PE下降）
                │
                ↓
            股市下跌
```

### 路径2：企业盈利传导

```
商品价格上涨（以石油为例）
        │
        ├──→ 航空公司成本↑ → 利润↓ → 股价↓
        │
        ├──→ 化工企业成本↑ → 利润↓ → 股价↓
        │
        ├──→ 物流运输成本↑ → 各行业成本普遍上升
        │
        ├──→ 农业成本↑（化肥/运输） → 食品价格上涨
        │
        └──→ 但能源公司（开采/冶炼）利润↑ → 股价↑
                │
                ↓
            股市出现板块分化
            （能源板块↑，消费/运输板块↓）
```

### 路径3：消费购买力传导

```
食品/能源价格上涨
        │
        ├──→ 居民消费必需品支出↑
        │    （"菜单效应"：每次加油/购物都感受到涨价）
        │
        ├──→ 可支配收入不变 → 非必需品消费↓（"油老虎"效应）
        │
        ├──→ 消费信心下降
        │
        ├──→ 零售/旅游/娱乐等非必需消费板块↓
        │
        └──→ GDP增长承压
                │
                ↓
            企业营收预期↓ → 股价↓
```

### 路径4：央行货币政策传导

```
商品价格冲击（供给侧）：
  原油价格暴涨（如1973/1979年石油危机）
        ↓
  成本推动型通胀（不是需求过热）
        ↓
  央行面临两难：
  ├─ 加息 → 抑制通胀，但伤害经济
  └─ 不加息 → 通胀失控

  历史上央行通常选择加息
        ↓
  经济衰退（1980年代初沃尔克时代）
        ↓
  大幅加息 → 经济硬着陆 → 失业率飙升

2008年案例（需求侧崩塌）：
  金融危机 → 需求崩溃
        ↓
  原油从$147跌到$32（跌78%）
        ↓
  通胀压力消失 → 各国央行降息
        ↓
  但通缩担忧上升
        ↓
  实体经济收缩更严重（需求端问题）
```

### 路径5：汇率联动传导

```
美元与商品的关系（反过来也成立）：

美元贬值
        ↓
黄金/石油/金属以美元计价↓
        ↓
非美元区购买成本↓
        ↓
全球需求↑
        ↓
商品价格上涨（自我强化）

商品价格上涨
        ↓
产油国（俄罗斯/沙特/海湾国家）收入↑
        ↓
产油国货币升值（卢布/沙特里亚尔）
        ↓
澳元/加元等商品货币升值（资源出口国）
        ↓
美元指数相对走弱
        ↓
美元进一步贬值
```

## 三、关键大宗商品的传导特征

### 原油：全球经济的"血液"

```
原油价格上涨100%时的影响链：
  │
  ├──→ 运输成本↑ → 所有商品成本普遍上升
  │
  ├──→ 化工原料↑ → 塑料/化纤/化肥涨价
  │
  ├──→ 农业成本↑ → 食品价格上涨
  │
  ├──→ 居民出行成本↑ → 可支配收入减少
  │
  ├──→ 企业利润压缩（除非转嫁）
  │
  ├──→ 通胀↑ → 央行加息预期↑ → 股市估值承压
  │
  └──→ GDP增速放缓（ IMF估计：油价每涨10%，全球GDP-0.2%）
```

### 黄金：避险属性的极端案例

```
黄金具有独特的"零收益资产"特性：
  黄金不产生利息/股息
        ↓
  低利率环境 → 持有黄金的机会成本↓ → 黄金吸引力↑
        ↓
  高利率环境 → 持有黄金的机会成本↑ → 黄金吸引力↓

黄金的避险传导：
  风险事件 → 避险需求 → 黄金↑
        ↓
  黄金↑ → 反映市场恐慌程度
        ↓
  黄金↑ + 美元↑ → 极端避险（2008年10月）
        ↓
  黄金↑ + 美元↓ → 货币信用危机（2009-2011）
```

### 铜："铜博士"的经济先行指标

```
铜的工业属性决定其领先性：
  铜 → 广泛用于建筑/电力/制造业
        ↓
  经济扩张期 → 铜需求↑ → 铜价领先GDP上涨
        ↓
  经济衰退期 → 铜需求↓ → 铜价领先GDP下跌

铜价与股市的关系：
  铜价↑ → 经济预期乐观 → 股票↑
        ↓
  铜价急涨 → 通胀担忧 → 央行收紧预期 → 股市↑或↓
  （取决于经济过热程度）

铜油比（Copper/Oil Ratio）：
  极端高值 → 经济/股市可能过热
  极端低值 → 经济/股市可能衰退
```

## 四、2008年危机中的商品市场崩盘

```
                    商品市场的三段式崩盘

Phase 1：狂热期（2007-2008.7）
  美元贬值 + 中国需求 + 金融投机
        ↓
  原油：$60 → $147（涨145%）
  黄金：$650 → $1,000（涨54%）
  铜：$6,000 → $8,900（涨48%）

Phase 2：流动性危机期（2008.7-12月）
  雷曼倒闭 → 全球恐慌 → 所有资产被抛售换美元
        ↓
  原油：$147 → $32（跌78%，最剧烈）
  黄金：$1,000 → $700（跌30%）
  铜：$8,900 → $2,800（跌69%）

Phase 3：QE时代（2009-2011）
  美联储QE → 美元泛滥 → 避险+通胀预期
        ↓
  黄金：$700 → $1,900（涨171%，创历史新高）
  原油：$32 → $100+（恢复）
  铜：$2,800 → $10,000+（再创历史新高）
```

### 商品市场崩盘的传导

```
原油价格暴跌$32
        │
        ├──→ 能源公司利润暴跌 → 股市能源板块崩盘
        │    （埃克森美孚/雪佛龙等市值大幅缩水）
        │
        ├──→ 产油国政府收入暴跌
        │    （俄罗斯/委内瑞拉/中东国家财政危机）
        │
        ├──→ 能源公司债券违约 → 高收益债券市场地震
        │
        ├──→ 石油美元（Petrodollar）回流量↓
        │    （产油国从美国国债撤资 → 美债收益率↑）
        │
        └──→ 美国页岩油公司债务违约
             （高成本生产商破产）
```

## 五、通缩与商品市场的恶性循环

```
                       债务-通缩螺旋
                          [[5.3_债务-通缩循环]]

商品价格下跌（供给过剩/需求不足）
        │
        ├──→ PPI负增长（通缩）
        │
        ├──→ 企业营收↓ → 债务负担相对↑（实际债务↑）
        │
        ├──→ 企业减少投资 → 裁员
        │
        ├──→ 失业↑ → 居民收入↓ → 消费↓ → 需求进一步↓
        │
        ├──→ 商品价格进一步下跌
        │
        └──→ 螺旋恶化

2008-2009年实际数据：
  CRB大宗商品指数：2008年6月 → 2009年2月
  跌幅：47%（仅8个月）
  全球PPI：2008年高位 → 2009年低位：-15%
  全球CPI：2008年高位 → 2009年：通缩担忧
```
