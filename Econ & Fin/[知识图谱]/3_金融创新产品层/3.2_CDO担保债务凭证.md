# 3.2 CDO（担保债务凭证）

## 一、定义与基本原理

**CDO（Collateralized Debt Obligation）** 是将**多种债务资产**打包后进行**分层**的证券化产品。与MBS不同，CDO可以以多种资产为底层抵押。

```
MBS：底层资产 = 仅住房抵押贷款
CDO：底层资产 = MBS + 企业债券 + 银行贷款 + 其他债务工具
```

## 二、CDO的类型

| 类型 | 底层资产 | 危机中角色 |
|------|---------|-----------|
| **CBO**（担保债券 Obligation） | 企业债券 | 高收益债券（垃圾债）打包 |
| **CLO**（担保贷款 Obligation） | 银行贷款 | 杠杆贷款打包 |
| **ABS**（资产支持证券） | 信用卡/汽车/学生贷款 | 消费贷款打包 |
| **CDO²**（CDO of CDO） | 其他CDO的份额 | 极度复杂的再打包 |

## 三、CDO的现金流转结构

```
┌─────────────────────────────────────────────────────┐
│                    CDO结构                          │
│                                                     │
│         ┌─────────────────────────┐                 │
│         │      底层资产池         │                 │
│         │  (MBS + 债券 + 贷款等)   │                 │
│         └──────────┬──────────────┘                 │
│                    │ 产生现金流                      │
│                    ↓                               │
│         ┌─────────────────────────┐                 │
│         │   现金流分配（分 waterfall）│              │
│         └──────────┬──────────────┘                 │
│                    ↓                               │
│  ┌──────────────────────────────────────────────┐   │
│  │  优先级 Senior Tranche  70-80%   评级AAA   │   │
│  │   → 首先偿付，最后才亏损                    │   │
│  ├──────────────────────────────────────────────┤   │
│  │  次优先级 Mezzanine Tranche 10-20%  评级A  │   │
│  │   → 损失首先由优先级吸收                    │   │
│  ├──────────────────────────────────────────────┤   │
│  │  股权层 Equity Tranche   5-10%   无评级    │   │
│  │   → 最高收益，首先承受损失，最后才获偿付    │   │
│  └──────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

## 四、分层机制：如何制造"AAA级"证券

### 核心逻辑（数学保证）：
```
资产池违约率假设：10%

优先级（承担前10%损失）：
  底层10%违约 → 优先级吸收
  优先级本金完全不受损
  → 评级AAA

次优先级（承担10-20%损失）：
  底层违约超过10%时才开始受损
  → 评级BBB~A

股权层（承担超过20%的损失）：
  底层违约超过20%时才开始受损
  → 评级最低/无评级
```

### 关键漏洞：模型依赖假设

```
评级的核心假设：
1. 违约率服从正态分布（用历史数据校准）
2. 不同底层资产违约的相关性有限
3. 房价持续上涨（降低违约率）

实际发生的问题：
1. 次贷违约率远超历史最高值（假设失效）
2. 不同MBS的违约相关性在危机中接近1（"跳伞效应"）
3. 房价开始下跌（反向加速违约）
```

## 五、CDO的规模扩张

```
                        CDO市场扩张
                            │
        ┌───────────────────┼───────────────────┐
        ↓                   ↓                   ↓
   2000年             2005年               2007年
   ~$1万亿           ~$3万亿              ~$6万亿

驱动因素：
  ① 次级房贷规模扩大 → 次级MBS供给增加
  ② 投资银行竞争激烈 → 疯狂寻找CDO抵押品
  ③ 评级机构"顺从" → 高风险产品获高评级
  ④ 全球资金追逐收益 → 投资者大量购买
```

## 六、CDO²（CDO的平方）：复杂性再升级

```
普通CDO：
  MBS/债券 → 打包 → CDO → 分为Senior/Mezz/Equity

CDO²（Synthetic CDO）：
  多个CDO的股权层 → 再次打包 → CDO²
        ↓
  CDO²的优先级 → 评级AAA
        ↓
  创造路径：
  次级房贷 → MBS → CDO股权层（10-20%违约风险）
          → CDO²优先级（"看似"极低风险）
          → 最终投资者持有

致命问题：
  看似分散风险的CDO²，实际上底层资产高度相关
  当次贷危机爆发时，CDO²的"AAA级"优先级同样血本无归
```

## 七、CDO在危机中的崩盘路径

```
第一步：违约率上升
  次级贷借款人在加息周期无力偿还
  → 违约率从~5%升至~15%+

第二步：MBS价值缩水
  止赎房屋涌入市场
  → 房价下跌
  → 抵押品价值缩水
  → MBS基础价值下降

第三步：CDO减值
  底层MBS价值缩水
  → CDO各层级同时承受压力
  → 评级机构被迫下调CDO评级

第四步：评级下调引发抛售
  下调→强制卖出（监管限制持有低评级资产）
  → 进一步压低价格
  → 银行资产负债表上CDO减值
  → 资本充足率下降

第五步：去杠杆化
  银行被迫抛售资产
  → 资产价格螺旋下跌
  → 流动性枯竭
  → 雷曼/贝尔斯登等机构危机
```

## 八、CDO市场的关键人物与利益冲突

| 角色 | 利益 | 冲突 |
|------|------|------|
| **投资银行** | 承销费（越多越好） | 希望发行规模越大越好 |
| **评级机构** | 评级费（发行方付费） | 不给高评级→失去客户 |
| **对冲基金** | 追求高回报 | 持有CDO股权层做空优先级（双重下注） |
| **银行** | 持有CDO（自以为对冲了） | 实际上不清楚自己持有的真实风险 |

> **讽刺**：部分对冲基金知道CDO是高风险的，但通过购买CDS（[[3.3_CDS信用违约互换]]）做空，同时向投资者推销这些产品。这构成了典型的利益冲突。
