# 3.1 MBS（抵押支持证券）

## 一、定义与基本原理

**MBS（Mortgage-Backed Securities）** 是将**住房抵押贷款池**打包成证券出售的投资工具。MBS持有者获得贷款池产生的本息收入。

```
传统模式（银行的资产负债表内）：
  借款人 ──月供──▶ 银行（持有贷款）
  银行自己承担信用风险和提前还款风险

MBS模式（证券化，移出资产负债表）：
  借款人 ──月供──▶ 贷款池 ──现金流──▶ MBS投资者
                           ↑
                    SPV（特殊目的机构）
                    隔离发起银行的风险
```

## 二、MBS的两种主要类型

| 类型 | 全称 | 特征 | 典型发行机构 |
|------|------|------|-------------|
| **Agency MBS** | 政府支持机构MBS | 房利美/房地美/吉利美担保 | 两房 |
| **Non-Agency MBS** | 私有MBS | 无政府担保，信用风险自负 | 投行/私有机构 |

```
住房抵押贷款池
       │
       ├── 有担保（MBS）── 政府机构担保──→ 投资者
       │                                      （两房提供担保，承担违约风险）
       │
       └── 无担保（MBS）── 无政府背书──→ 投资者
                                      （自己承担违约风险）
```

## 三、MBS的现金流结构

### 简单MBS（月供分配）：
```
所有贷款月供汇总
        │
        ├── 扣除服务费 ──▶ 服务银行（处理还款）
        │
        └── 剩余现金流 ──▶ MBS投资者（按份额分配）
```

### 过手证券（Pass-through）结构：
- 贷款池产生的现金流直接"过手"给投资者
- 投资者按持有份额获得本金+利息
- 结构简单，但投资者承担提前还款风险

## 四、MBS在2008年危机中的核心角色

### 次级贷款 + MBS = 危机放大器

```
低门槛放贷标准（NINJA贷款：No Income, No Job, No Assets）
    ↓
次级借款人获得抵押贷款
    ↓
贷款被打包成MBS
    ↓
MBS被出售给全球投资者
    ↓
银行释放资本 → 继续放贷
    ↓
房地产泡沫膨胀
    ↓
违约率上升 → MBS贬值 → 投资者亏损
    ↓
资金链断裂 → 危机爆发
```

## 五、MBS规模的量化

| 指标 | 2000年 | 2007年 | 说明 |
|------|--------|--------|------|
| 美国MBS未偿余额 | ~$5万亿 | ~$14万亿 | 主要由两房持有 |
| 次级贷款占比 | ~5% | ~20% | 次贷占总房贷比例 |
| 非机构MBS（私有） | ~$1万亿 | ~$3万亿 | 无政府担保，风险更高 |

## 六、MBS对金融体系的双刃剑效应

### 正面功能（金融创新本身没错）
```
银行发放贷款 → 打包出售 → 资本释放
        ↓
银行可以发放更多贷款
        ↓
住房金融市场流动性增加
        ↓
更多人获得住房贷款
        ↓
促进住房市场发展
```

### 负面后果（当激励机制扭曲时）
```
高风险贷款 → 高佣金 → 贷款经纪商疯狂放贷
        ↓
银行："反正可以打包卖掉" → 风控标准下降
        ↓
MBS投资者依赖评级机构
        ↓
评级机构：利益冲突 → 高风险产品给高评级
        ↓
风险在整个金融体系扩散
```

## 七、MBS的市场结构与投资者

```
MBS投资者类型：
├── 养老金基金（追求稳定现金流，持有优先级）
├── 保险公司（资产负债匹配，长期配置）
├── 共同基金（追求收益，持有中间层）
├── 对冲基金（杠杆操作，追求高回报，持有股权层）
└── 外国央行/投资者（外汇储备投资渠道）

各层级风险分配：
  优先级（~80%）→ 养老金/保险（低风险）→ 银行间市场质押品
  次优先级（~15%）→ 共同基金/对冲基金（中风险）
  股权层（~5%） → 对冲基金（高风险）→ 首先承受损失
```

## 八、MBS如何演变为CDO

MBS只是第一步，当MBS继续被打包时就变成了**CDO**：

```
次级抵押贷款池
       ↓
    MBS（抵押支持证券）
       ↓
    CDO（担保债务凭证）← 将MBS作为抵押品再次打包
       ↓
  CDO分级（Senior/Mezz/Equity）
       ↓
   部分CDO再次被打包 → CDO²
```
