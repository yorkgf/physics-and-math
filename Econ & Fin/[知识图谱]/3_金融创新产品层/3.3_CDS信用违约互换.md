# 3.3 CDS（信用违约互换）

## 一、定义与基本原理

**CDS（Credit Default Swap）** 本质上是**债务的保险**：买方支付定期费用，在信用事件（如违约、破产）发生时获得赔付。

```
CDS的基本结构：

CDS买方（寻求保护）        CDS卖方（承保风险）
    │                           │
    │ ────定期保费（如1%）──────▶│
    │                           │
    │  "我持有这个债券"           │
    │  "如果它违约，你赔我"        │
    │                           │
    ▼ 违约事件发生              │
  "赔付！"  ◀──赔偿（如本金）──│
```

## 二、C CDS的双重用途

### 用途1：对冲（正确用途）
```
投资者持有企业债券 → 担心发行方违约
    ↓
购买CDS对冲违约风险
    ↓
支付保险费（CDS溢价）
    ↓
如果债券违约 → CDS赔付弥补损失

目的：降低而非放大风险
```

### 用途2：投机（杠杆放大风险）
```
投机者认为某CDO会违约 → 不持有任何CDO
    ↓
购买CDS（做空该CDO的信用）
    ↓
如果CDO违约 → CDS赔付获利
    ↓
无需持有底层资产 → 纯粹的投机工具
```

## 三、C DS的名义本金规模

CDS最危险的特点：**名义本金远超实际经济规模**

| 时间 | CDS未偿名义本金 | 美国GDP | 比例 |
|------|--------------|--------|------|
| 2001年 | ~$1万亿 | ~$10万亿 | 10% |
| 2007年 | **$62万亿** | ~$14万亿 | **440%** |

```
62万亿美元是什么概念？
  → 超过全球GDP总量
  → 美国国债总额约$9万亿
  → CDS名义本金是美债的7倍

原因：CDS可以无限量发行
  → 不需要持有底层资产
  → 理论上可以重复购买同一资产的CDS
  → 纯粹的投机工具
```

## 四、CDS市场结构

```
CDS市场参与者：

1. 银行（商业银行/投行）
   → 使用CDS对冲自己的资产组合
   → 也作为CDS卖方收取保费

2. 保险公司（AIG就是最大的CDS卖方之一）
   → 承诺在违约时赔付
   → 收取保费，但自己资产规模有限

3. 对冲基金
   → 投机性持有CDS
   → 做空信用市场

4. 养老金/资产管理公司
   → 用CDS保护债券组合
```

## 五、AIG：CDS最大"炸弹"

### AIG如何陷入危机

```
AIG作为CDS卖方：
  向全球投资者出售CDS
  承诺：如果你持有的CDO/债券违约，我赔你
  收取：保费（看起来是稳赚不赔的生意）

AIG的假设：
  大量分散的CDS违约概率很低
  收取保费 → 投资高评级资产 → 赚取利差
  看起来是完美的商业模式

问题：
  信用违约事件大规模同时发生时
  → AIG需要同时赔付数百亿美元的CDS
  → AIG自有资本只有数百亿
  → 根本不够赔付 → 破产边缘
```

### AIG危机的量化

```
AIG在2008年9月：
  CDS敞口：~$3000亿+
  自有资本：~$800亿
  实际缺口：~$2000亿+

如果没有政府救助：
  AIG倒闭 → 触发数千亿CDS赔付
  → 持有CDS的银行突然获得赔付（意外之财）
  → 但AIG的CDS交易对手无法收到钱
  → 引发更大规模的连锁违约
```

## 六、CDS如何放大危机

### 路径1：CDS卖方面临灭顶之灾
```
违约事件发生 → CDS卖方需要赔付
        ↓
AIG/银行等CDS卖方资本不足
        ↓
CDS卖方破产或需要政府救助
        ↓
更多的违约担忧 → 更多的CDS赔付压力
        ↓
螺旋式崩溃
```

### 路径2：CDS加速CDO贬值
```
CDO价值下跌
        ↓
CDS买方要求追加担保品（Variation Margin）
        ↓
CDO持有者被迫变现其他资产补充保证金
        ↓
抛售压力 → 资产价格进一步下跌
        ↓
更多CDO减值
        ↓
更多CDS赔付要求
```

### 路径3：谁也不清楚风险有多大
```
CDS柜台交易（OTC，无中央清算）
        ↓
每个机构不知道对手方有多少CDS敞口
        ↓
银行间互不信任
        ↓
同业拆借市场冻结（[[5.5_流动性危机]]）
```

## 七、CIS的积极功能（正常状态下）

```
CDS不是天然邪恶的，它有合理的用途：

1. 信用风险转移
   持有债券的银行 → 购买CDS → 信用风险转移给保险公司
   → 银行释放资本 → 放更多贷款

2. 价格发现
   CDS溢价 = 市场对未来违约概率的定价
   → 为信用风险定价提供参考

3. 流动性提供
   CDS可以随时买入卖出
   → 持有CDS比持有实物债券更灵活

问题在于：当CDS市场过度膨胀，投机需求远超对冲需求时
CDS从风险管理工具变成了系统性风险的来源
```

## 八、CDS与[[3.2_CDO担保债务凭证]]的关系

```
CDO + CDS 的致命组合：

CDO创造路径：
  次级贷款 → MBS → CDO（分层）→ CDO优先级（评级AAA）

CDS对冲逻辑：
  投资者持有CDO优先级（看似安全）
  但担心风险 → 购买CDS作为"保险"

问题一：CDO规模很大 → 需要大量CDS覆盖
问题二：评级机构错误评级 → CDO风险被低估 → CDS保费低廉
问题三：CDS卖方（AIG等）杠杆过高 → 无力赔付

当危机发生时：
  CDO贬值 → CDS触发赔付
  → AIG等卖方资不抵债
  → 引发更大的系统性恐慌
```
