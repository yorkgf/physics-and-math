# 3.4 分层结构与风险转移

## 一、分层（Tranching）的本质

**分层**是证券化最核心的机制：通过将资产池的现金流按**优先级顺序**分配，将不同风险偏好的投资者连接到同一资产池。

```
                        分层结构原理
                            │
        ┌───────────────────┼───────────────────┐
        ↓                   ↓                   ↓
     优先级              次优先级             股权层
   Senior Tranche      Mezzanine Tranche   Equity Tranche
        │                   │                   │
    评级AAA            评级A~BBB           无评级/高风险
        │                   │                   │
    收益率低            收益率中等           收益率最高
        │                   │                   │
   首先偿付            损失时其次           首先承受损失
   最后才亏损          才亏损               最后才获偿付
        │                   │                   │
    机构投资者/         对冲基金/            对冲基金/
    银行/货币基金        共同基金             发起人自持
```

## 二、分层如何"制造"AAA级证券

### 关键数学

假设资产池总价值 = $100，违约率假设 = 5%

```
$100的资产池
    │
    ├── 优先级 ($80)：
    │     在$0-$80的损失范围内不受损
    │     实际要承受的是：$5违约 / $80优先级 = 6.25%
    │     → 评级AAA（看起来安全）
    │
    ├── 次优先级 ($15)：
    │     优先级吸收$5后，剩余$10由次优先级承担
    │     → 损失承担比例：$10 / $15 = 67%
    │     → 评级BB/B
    │
    └── 股权层 ($5)：
          完全吸收前$5损失
          → 首先承受损失，但获得超额回报（高收益）
```

### 评级机构的错误假设

```
评级机构认为：
  ① 各MBS的违约相关性低（假设~0.1）
  ② 房价持续上涨（违约率低）
  ③ 资产池足够分散

实际发生：
  ① 所有次级贷款在房地产泡沫破裂时同时违约
     → 违约相关性在危机中接近1（"跳伞效应"）
  ② 房价开始下跌（预期反转）
  ③ 地理分散不够（区域性危机同时爆发）

结果：即使优先级理论上承受6%损失，
      实际损失可能远超80%的缓冲垫
```

## 三、证券化的资金链条全景

```
                        过长的资金链
                            │
  借款人 ←─── 贷款 ───── 贷款经纪商
                          │
                    发放费（按件付费）
                          ↓
                    发起银行（贷出贷款）
                          │
                    发放费 + 短炒利差
                          ↓
                    两房（收购/担保MBS）
                          │
                    担保费 + 证券化利差
                          ↓
                    投资银行（打包CDO）
                          │
                    承销费 + 打包费 + 自营交易
                          ↓
                    评级机构（评级CDO）
                          │
                    评级费（发行方付费→利益冲突）
                          ↓
                    全球投资者（持有各层级）
                          │
                    最终回报（优先级低收益/股权层高收益）

    每个环节：
      ✓ 只赚手续费/佣金
      ✗ 没有人最终对贷款质量负责
      ✗ 原始借贷关系越来越模糊
```

## 四、道德风险的多层放大

```
第一层：贷款经纪商
  激励：发放更多贷款 → 赚更多佣金
  结果：NINJA贷款（无收入无工作无资产）
  监督：谁也不管，反正银行会买

第二层：发起银行
  激励：发放更多贷款 → 快速卖给两房
  结果：不做尽职调查，风控标准下降
  监督：两房说了不挑食

第三层：两房（Fannie/Freddie）
  激励：扩大担保规模 → 收取更多担保费
  结果：大量收购高风险次级MBS
  监督：政府隐性背书 → 政治压力大于风控

第四层：投资银行
  激励：打包更多CDO → 收取更多承销费
  结果：疯狂寻找CDO抵押品，甚至创造CDO²
  监督：自营交易和客户业务没有隔离

第五层：评级机构
  激励：给高评级 → 获取更多业务
  结果：明知风险高还给AAA
  监督：没有真正独立的第三方监督

第六层：最终投资者
  激励：追求高回报 → 购买高收益CDO
  结果：依赖评级，忽视底层风险
  监督：监管对高评级资产有资本优惠
```

## 五、风险转移 vs 风险消失

```
证券化的表面效果：
  银行资产负债表 ──移除──▶ 特殊目的机构(SPV)
  风险   ✓ 转移了
  银行资本 ✓ 释放了

但实际效果（[[5.4_系统性风险]]）：
  风险没有消失 ──只是被分散──▶ 转移到全球投资者
                          │
                    每个投资者都觉得风险可控
                    但没有人意识到所有人都在持有
                    同一类风险资产
                     →
                    当危机来临时，所有人同时受损
                    分散 → 集中的悖论
```

## 六、监管套利：资本要求规避

```
商业银行面临的资本充足率要求：
  持有贷款 → 需要8%资本覆盖

证券化操作：
  发放贷款 → 打包成MBS → 出售
  → 贷款从资产负债表移除
  → 无需持有对应资本

监管套利的结果：
  银行表面上"低风险"（资本充足）
  实际上通过持有MBS/CDO的优先级（表外）
  承担了实质上的信用风险
  但没有为这个风险配置相应资本
```

## 八、对中国金融市场的启示

```
证券化本身的逻辑没有问题，但需要：

1. 激励相容
   → 发起银行必须持有部分风险（保留条款）
   → 不能完全"贷完就卖"

2. 信息透明
   → 投资者必须能看到底层资产
   → 不能创造CDO of CDO这种极度不透明的产品

3. 监管同步
   → 监管不能落后于金融创新
   → 证券化产品需要资本要求

4. 评级独立
   → 不能由发行方付费
   → 需要真正的第三方评级
```
