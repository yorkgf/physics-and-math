# 2.1 货币市场

## 一、定义与核心特征

**货币市场**是交易**短期（通常≤1年）**、**高流动性**、**低风险**金融工具的市场。

| 特征 | 说明 |
|------|------|
| 期限 | 隔夜 ~ 1年 |
| 目的 | 流动性管理，而非长期投资 |
| 风险 | 低（接近无风险） |
| 收益率 | 最低（无风险利率的下限） |
| 参与者 | 央行、银行、机构投资者、政府 |

## 二、主要子市场与工具

### 1. 银行间同业拆借市场（Interbank Market）
- **定义**：银行之间互相借出/借入短期资金的市场
- **关键利率**：Fed Funds Rate（美国）、EURIBOR（欧洲）、SHIBOR（中国）
- **重要性**：银行间互相信任的度量仪

```
银行A（有超额储备）───借出──→ 银行B（流动性不足）
          ↓
   无担保/有担保拆借利率
```

> **危机中的异常**：2008年后LIBOR-OIS利差飙升，反映银行间极度不信任

### 2. 商业票据市场（Commercial Paper, CP）
- **定义**：企业发行的短期无担保本票，期限通常≤270天
- **特点**：不需要SEC注册，成本低于银行贷款
- **重要性**：企业短期流动资金的重要来源
- **风险**：企业信用风险

### 3. 国库券市场（Treasury Bills）
- **定义**：政府发行的短期债券，期限：4/8/13/26/52周
- **特点**：以折扣方式发行，无息票
- **用途**：政府融资 + 央行公开市场操作工具

### 4. 回购协议市场（Repo / Reverse Repo）
- **定义**：以政府债券为担保的短期融资协议
- **Repo**：卖出国债+约定未来购回（融资）
- **Reverse Repo**：买入国债+约定未来卖出（借出资金）
- **重要性**：货币市场基金、对冲基金的主要融资渠道

### 5. 大额可转让定期存单（CDs）
- **定义**：银行发行的大额定期存款凭证，可转让
- **特点**：流动性好，利率高于普通存款

## 三、货币市场的核心利率指标

```
                    货币市场利率体系
                           │
        ┌──────────────────┼──────────────────┐
        ↓                  ↓                  ↓
  央行政策利率        基准参考利率         市场实际利率
  (Policy Rate)      (Benchmark)         (Market Rates)
        │                  │                  │
  Fed Funds Rate    LIBOR / SOFR      商业票据利率
  (美国联邦基金利率)  (各货币基准)        银行间拆借利率
                          │             国库券收益率
                    货币市场利率的        回购协议利率
                     "锚"和"基准"
```

**关键利率走廊机制**：
```
        央行贷款利率（上限）
             ↑
  LIBOR/SOFR ←── 市场利率
             ↓
       央行存款利率（下限）
```

```
                    央行货币政策
                        │
                        │ 公开市场操作
                        ↓
               ┌────────────────┐
               │   货币市场    │ ←── 商业银行流动性管理
               │ (短期资金池)  │
               └───────┬───────┘
                       │ 定价基准
         ┌─────────────┼─────────────┐
         ↓             ↓             ↓
   ┌──────────┐  ┌──────────┐  ┌──────────┐
   │ 债券市场  │  │ 外汇市场  │  │ 衍生品市场 │
   │(长期利率) │  │ (汇率)   │  │ (利率期权) │
   └──────────┘  └──────────┘  └──────────┘
```

## 五、货币市场与其他市场的连接

| 传导路径 | 机制 | 典型案例 |
|---------|------|---------|
| 货币市场 → 债券市场 | 收益率曲线预期 | 短期利率上升→债券价格下跌 |
| 货币市场 → 外汇市场 | 利差→资本流动 | 高利率吸引外资→本币升值 |
| 货币市场 → 股票市场 | 融资成本 | 利率上升→估值下降 |
| 货币市场冻结 → 实体经济 | 企业无法获得流动资金 | 2008年商业票据市场停滞 |

## 六、2008年危机中的货币市场

### 发生了什么
- 2008年9月，第一储备基金（Reserve Primary Fund）因持有雷曼兄弟商业票据而跌破净值（"打破1美元面值"）
- 引发货币市场基金大规模挤兑，$3000亿+流出
- 货币市场基金通常被视为"准现金"，净值跌破引发全面恐慌

### 传导路径
```
雷曼倒闭 → 持有其CP的货币基金减值
    ↓
投资者恐慌赎回货币基金
    ↓
货币基金被迫抛售其他CP → 商业票据市场冻结
    ↓
企业无法再融资 → 流动资金枯竭
    ↓
实体经济急速恶化
```

### 央行应对
- 美联储创造新工具（CPFF商业票据融资工具）直接购买商业票据
- FDIC为银行定期存款提供无限担保
- 与14家央行建立美元互换额度，缓解全球美元荒
