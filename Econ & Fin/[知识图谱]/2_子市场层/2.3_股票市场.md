# 2.3 股票市场

## 一、定义与核心特征

**股票市场**是交易**公司股权**的市场，股票持有者是公司**剩余索取权**的拥有者。

```
股票定价基本公式（DDM — 股息贴现模型）：
  P = D / (r - g)

P = 股票价格
D = 预期股息
r = 投资者要求回报率（= 无风险利率 + 风险溢价）
g = 股息增长率

等价表达（简化版PE）：
  P/E = 1 / (r - g)
  → 利率越低，PE估值越高
```

## 二、股票市场的核心功能

| 功能 | 说明 |
|------|------|
| **融资功能** | 企业通过IPO/增发直接获得股权融资 |
| **价值发现** | 市场价格反映公司未来盈利预期 |
| **资源配置** | 资本流向高回报行业/公司 |
| **公司治理** | 股价下跌→收购威胁→约束管理层 |
| **财富效应** | 股价变动→居民财富→消费变动 |

## 三、影响股票价格的核心因素

### 1. 分子端：企业盈利（Earnings）
```
经济周期
    ↓
企业营业收入
    ↓
企业净利润
    ↓
EPS（每股收益）
    ↓
股票价格支撑
```

### 2. 分母端：折现率（Discount Rate）
```
折现率 r = 无风险利率 + 股权风险溢价

无风险利率（由债券市场决定）
    ↓ 货币市场/债券市场
  央行政策利率
  10年期国债收益率

股权风险溢价（由股票市场供需决定）
    ↓
  市场情绪
  不确定性程度
  风险偏好
```

### 3. 市场情绪与资金流动
```
风险偏好上升 → 资金流入股市 → 股价上升
    ↓
风险偏好下降 → 资金流出股市 → 股价下跌

杠杆资金（融资融券）加剧波动：
  股价↑ → 保证金充足 → 杠杆加码 → 买更多 → 股价↑↑↑
  股价↓ → 追加保证金 → 强制平仓 → 卖更多 → 股价↓↓↓
```

## 四、股票市场的分层结构

```
                    股票市场
                       │
       ┌───────────────┼───────────────┐
       ↓               ↓               ↓
  一级市场          二级市场          衍生品市场
  (发行/IPO)        (交易所)          (股指期货/期权)
       │               │               │
   企业融资        投资者交易          风险对冲/投机
```

## 五、股票市场与实体经济的连接

### 财富效应（Wealth Effect）
```
股票价格上涨
    ↓
居民部门财富增加
    ↓
消费支出增加（MPC ~ 3-5%）
    ↓
企业营收增加
    ↓
经济产出增加（GDP）
```

> 美联储估计：美国家庭每增加1美元财富，消费增加约3-5美分。

### 托宾Q效应
```
股价上升 → Q值（市场价值/重置成本）> 1
    ↓
企业更有动力发行股票融资
    ↓
投资增加
    ↓
经济扩张
```

## 七、股票市场与其他市场的关系

### 股票 vs 债券（股债轮动）
```
经济扩张 → 企业盈利↑ → 股票吸引力↑
              ↓ (债券资金流向股票)
         债券被抛售 → 债券收益率↑

经济衰退 → 债券吸引力↑ → 资金流向债券
              ↓
         债券价格上涨 → 收益率↓
         股票继续下跌
```

### 股票 vs 大宗商品
```
经济繁荣 → 商品需求↑ → 商品价格上涨
              ↓
         能源/材料板块股票受益

经济衰退 → 商品需求↓ → 商品价格下跌
              ↓
         资源类股票下跌
```

### 股票 vs 外汇
```
美元贬值（美国股市下跌的副产品）
    ↓
美国资产吸引力下降
    ↓
资本流出美国
    ↓
新兴市场股市承压（美国下跌→全球联动）
```

## 八、股票市场在2008年危机中的表现

```
2007年10月：标普500高点 1,565点
    ↓
2008年3月：贝尔斯登救助，跌至1,270点
    ↓
2008年9月15日：雷曼倒闭，跌至1,166点
    ↓
2008年10月：恐慌高峰，跌至968点
    ↓
2009年3月9日：谷底 666点
    ↓
从高点到低点跌幅：57%（历时16个月）

VIX恐慌指数：
  正常 ~15 → 2008年10月峰值 80.86（历史纪录）
```

## 九、市场联动与系统性下跌

2008年危机的关键特征：**所有风险资产同时下跌，相关性趋近1**
```
原因一：系统性流动性枯竭
  所有投资者都需要流动性
  → 抛售所有可变现资产

原因二：杠杆去化（去杠杆化）
  杠杆投资者被追缴保证金
  → 无论什么资产都强制卖出

原因三：风险偏好急剧收缩
  投资者不愿持有任何风险资产
  → 从股票到商品到新兴市场全面下跌

结果：传统的"分散化投资"完全失效
```
