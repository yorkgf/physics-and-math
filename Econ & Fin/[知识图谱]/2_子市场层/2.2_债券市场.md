# 2.2 债券市场

## 一、定义与核心特征

**债券市场**是交易**长期债务工具**（期限≥1年）的市场，是**长期利率**的定价场所。

```
债券价格与利率反向变动：
  P = C/(1+r) + C/(1+r)² + ... + (C+F)/(1+r)ⁿ
  当利率(r)↑ → 债券价格(P)↓  (利率风险)
```

## 二、债券市场的分层

```
                    债券市场
                       │
       ┌───────────────┼───────────────┐
       ↓               ↓               ↓
  政府债券市场      企业债券市场       资产支持债券
  (主权信用)        (公司信用)        (结构化产品)
       │               │               │
  国债/地方债       投资级/高收益     MBS/CDO/ABS
       │               │               │
  央行操作对象      企业融资主战场     证券化产品
```

## 三、核心利率指标：收益率曲线

### 什么是收益率曲线
将不同期限债券的收益率连成曲线，反映市场对未来利率和经济的预期。

```
正常曲线（向上倾斜）：
  收益率
    │           ────────
    │        ──
    │     ──
    │   ─
    └──────────────────→ 期限
       短      中      长

衰退预警（倒挂）：
  收益率
    │ ─
    │   ─
    │      ──
    │          ────────
    └──────────────────→ 期限
       短      中      长
    ↑ 短期收益率 > 长期收益率
    (通常预示经济衰退)
```

### 收益率曲线的经济含义

| 曲线形态 | 含义 | 信号 |
|---------|------|------|
| 正常向上倾斜 | 期限溢价正数 | 健康经济 |
| 平坦 | 市场对未来不确定 | 观望情绪 |
| 倒挂 | 短期利率>长期利率 | **衰退预警** |
| 极度陡峭 | 长端大幅高于短端 | 通胀预期上升 |

## 四、信用利差（Credit Spread）

**信用利差 = 企业债券收益率 - 同期限国债收益率**

```
                   债券收益率分解
                        │
         ┌──────────────┼──────────────┐
         ↓              ↓              ↓
   无风险利率      信用风险溢价     流动性溢价
   (国债收益)      (企业违约风险)   (变现难度)
         │              │              │
         └──────────────┴──────────────┘
                        │
                   投资者实际要求的
                    债券到期收益率
```

### 信用利差扩大的意义

| 利差变化 | 含义 |
|---------|------|
| 利差收窄 | 信用环境宽松，经济扩张 |
| 利差扩大 | 信用环境收紧，经济担忧 |
| **利差急剧扩大** | **信用危机/金融动荡** |

> 2008年危机中，高收益债券（垃圾债）与国债的利差从~3%飙升到>20%，反映信用市场彻底冻结。

## 五、债券市场的关键子市场

### 1. 政府债券市场
- **功能**：政府融资 + 金融市场定价基准
- **特点**：流动性最好，被视为无风险利率
- **重要指标**：10年期国债收益率（全球利率之锚）

### 2. 投资级企业债券
- **发行主体**：大型企业（BBB及以上评级）
- **功能**：企业长期融资的主要渠道
- **特点**：信用利差相对稳定

### 3. 高收益债券（垃圾债）
- **定义**：评级低于BBB的债券
- **特点**：高收益，高风险
- **投资者**：追求高回报的对冲基金、高收益债券基金

### 4. 资产支持证券（ABS/MBS）— 债券市场的证券化延伸
- 详见：[[3.1_MBS抵押支持证券]]

## 六、债券市场与其他市场的连接

### 债券-股票关系（股债轮动）
```
债券市场上涨（利率下降）
    ↓
股票相对吸引力下降（估值承压）
    ↓
也可能反映经济衰退预期（股票仍下跌）
    ↓
取决于：债券上涨是因为"衰退"还是"通胀下降"
```

### 债券-汇率关系
```
美国国债收益率上升
    ↓
美元资产吸引力上升
    ↓
资本流入美国
    ↓
美元升值
```

### 债券-大宗商品关系（通胀预期）
```
通胀预期上升
    ↓
债券名义利率要求提高
    ↓
债券价格下跌（收益率上升）
    同时
大宗商品价格上涨（黄金/原油）
```

## 七、债券市场在2008年危机中的表现

| 时间 | 现象 | 机制 |
|------|------|------|
| 2007年前 | 次级MBS收益率与国债相近 | 评级机构错误评级 |
| 2007年中 | CDO/MBS利差开始扩大 | 违约率上升暴露 |
| 2008年9月 | 投资级债券全面下跌 | 恐慌性抛售 |
| 2008年10月 | 股债双杀 | 流动性危机 |
| 2009年后 | 国债受追捧（避险） | 危机深化 |
