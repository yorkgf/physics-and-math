# 2.5 大宗商品市场

## 一、定义与核心特征

**大宗商品市场**是交易**基础原材料**（能源、金属、农产品）的市场，具有**实物属性**和**金融属性**双重特征。

```
                    大宗商品市场
                        │
      ┌─────────────────┼─────────────────┐
      ↓                 ↓                 ↓
   能源类            金属类            农产品类
  (原油/天然气)    (黄金/铜/铝)       (小麦/玉米/大豆)
      │                 │                 │
   经济引擎         工业晴雨表         食品通胀
```

## 二、大宗商品的分类

### 1. 能源类（Energy）
| 商品 | 特性 | 金融属性 |
|------|------|---------|
| **原油（WTI/布伦特）** | 最重要的大宗商品，全球经济血液 | 极度敏感于地缘政治/供给 |
| **天然气** | 区域性市场（各交易所割裂） | 与原油相关性高 |
| **煤炭** | 电力生产原料 | 区域性 |

### 2. 金属类（Metals）
```
贵金属：
  黄金 ─── 避险资产、通胀对冲、央行储备
  白银 ─── 工业+贵金属双重属性

工业金属：
  铜 ───── "铜博士"（经济先行指标）
  铝 ───── 建筑业/制造业
  镍/锂 ─── 新能源革命（近年新焦点）
```

### 3. 农产品（Agriculture）
```
谷物：小麦/玉米/大豆（全球粮食基础）
软商品：咖啡/糖/棉花（气候敏感）
畜类：活牛/瘦肉猪
```

## 三、大宗商品的定价机制

### 供需驱动（基本面）
```
供给侧：产量/库存/产能/天气/地缘政治
    ↓
供需缺口 → 价格调整
    ↓
盈利变化 → 投资变化 → 产能周期（2-5年滞后）
```

### 金融驱动（越来越重要）
```
宽松货币政策
    ↓
利率降低 → 持有大宗商品的机会成本↓
    ↓
资金流入大宗商品ETF/期货
    ↓
金融需求推高商品价格
    ↓
价格与实体供需脱节（金融化效应）
```

> **关键发现**：2008年危机前油价暴涨（$147/桶），部分原因是金融投机而非真实供需失衡。

## 四、大宗商品市场与其他市场的关系

### 商品-通胀关系（最直接的连接）

```
大宗商品价格上涨
    ↓
PPI（生产者价格指数）↑
    ↓
CPI（消费者价格）↑
    ↓
通胀上升
    ↓
央行被迫加息
    ↓
经济增长承压
    ↓
企业盈利预期下降
    ↓
股市下跌
```

### 商品-债券关系

```
商品价格↑ → 通胀预期↑
    ↓
债券名义利率要求↑
    ↓
债券被抛售 → 债券价格↓ → 收益率↑

实际利率 = 名义利率 - 通胀
通胀↑ → 实际利率可能不变甚至↓（如果央行不加息）
→ 对股市影响不确定
```

### 商品-外汇关系

```
美元贬值
    ↓
原油/黄金以美元计价 ↓（对非美元区买家）
    ↓
全球需求↑
    ↓
商品价格上涨
    ↓
产油国/资源国货币升值（澳元、加元）
    ↓
美元进一步承压（自我强化）
```

### 商品-股票关系

```
经济扩张期：
  商品需求↑ → 商品价格↑ → 能源/材料板块利润↑ → 股价↑

经济衰退期：
  商品需求↓ → 商品价格↓↓ → 能源/材料板块崩盘
```

## 五、大宗商品市场的金融化

### 传统模式 vs 金融化模式

```
传统模式：
  生产商 ←→ 贸易商 ←→ 消费者
  (实物交割，价格由供需决定)

金融化模式：
  大量机构投资者通过期货/ETF参与
  (金融需求主导价格，期货溢价)
```

### 2008年危机中的商品市场异动

| 阶段 | 时间 | 原油价格 | 原因 |
|------|------|---------|------|
| 泡沫期 | 2007-2008.7 | $37→$147 | 美元贬值+金融投机+供给担忧 |
| 崩盘期 | 2008.7-2008.12 | $147→$32 | 金融危机→需求崩溃→流动性枯竭 |
| 恢复期 | 2009-2010 | $32→$80 | 量化宽松+美元贬值+供给收缩 |

### 黄金的特殊性

```
危机时黄金也出现下跌（约30%）：
    → 原因：流动性危机，持有黄金的投资者被迫变现
    → 特征：真正的"避险资产"流动性反而在危机时最差

例外（当信心完全崩溃时）：
    → 黄金上涨（法币信用受损）
    → 2008年后黄金一路涨到$1,900
```

```

                    │
       ┌────────────┼────────────┐
       ↓            ↓            ↓

```

## 七、商品价格与实体经济的双向关系

### 供给冲击（Supply Shock）
```
1970年代石油危机（OPEC禁运）：
    油价暴涨4倍
        ↓
    成本推动型通胀
        ↓
    央行大幅加息
        ↓
    经济衰退
        ↓
    滞胀（Stagflation）= 高通胀+低增长
```

### 需求崩溃（Demand Collapse）
```
2008年金融危机：
    经济衰退→需求减少
        ↓
    油价从$147跌到$32（跌幅78%）
        ↓
    产油国收入暴跌
        ↓
    资源出口国经济崩溃
        ↓
    全球贸易萎缩进一步加剧
```
