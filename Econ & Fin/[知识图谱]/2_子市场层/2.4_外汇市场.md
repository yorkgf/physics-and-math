# 2.4 外汇市场

## 一、定义与核心特征

**外汇市场**是交易**不同货币**的市场，日均交易量约$7.5万亿美元，是全球最大的金融市场。

### 汇率的基本逻辑
```
汇率是两个国家经济基本面的比率

购买力平价（长期）：
  1单位本币 = 等量购买力对应的外币
  通胀差异 → 汇率调整方向

利率平价（中期）：
  r本币 - r外币 = 预期汇率变动
  利差 → 资本流动 → 汇率变化

资产市场理论（短期）：
  汇率 = 货币资产的相对价格
  由货币供给、利率、信心决定
```

## 二、汇率决定的核心机制

### 1. 利率平价（Interest Rate Parity）
```
高利率货币
    ↓
资本流入
    ↓
本币需求↑ → 即期升值
远期贬值（锁定价差）

套利均衡：
  高利率 = 即期贬值预期 + 低利率货币远期升值
```

### 2. 货币供给与汇率
```
央行扩大货币供给
    ↓
本币供给↑ → 本币贬值
(货币数量论：MV = PY)

例外：危机时美元
  即使美联储扩张货币，美元仍升值
  原因：全球美元荒，所有人需要美元偿还美元债务
```

### 3. 资本账户与经常账户
```
经常账户逆差（进口>出口）
    ↓
需要借入外币融资
    ↓
本币贬值压力

但：如果资本流入（外币投资本国资产）
    ↓
经常账户逆差被资本流入抵消
    ↓
本币不一定贬值
```

## 三、外汇市场在金融体系中的位置

```
                    外汇市场
                       │
       ┌───────────────┼───────────────┐
       ↓               ↓               ↓
   国际贸易          跨境资本          央行干预
   (贸易结算)       (投资/套利)        (汇率政策)
       │               │               │
  实体经济连接     金融市场连接       政策连接
```

## 四、外汇市场的关键参与者

| 参与者 | 行为特征 |
|--------|---------|
| **央行** | 外汇干预（调节汇率）+ 储备管理 |
| **商业银行** | 贸易结算、跨境投融资、套利 |
| **机构投资者** | 跨境资产配置（影响资本流动） |
| **对冲基金** | 投机性交易（可能引发汇率波动） |
| **出口商/进口商** | 套期保值（对冲汇率风险） |

## 五、汇率与其他市场的传导路径

### 汇率传导路径一：国际贸易

```
本币贬值
    ↓
出口商品以外币计价更便宜 → 出口↑
进口商品以本币计价更贵 → 进口↓
    ↓
贸易顺差扩大（或逆差缩小）
    ↓
本国出口企业营收↑ → 利好股票
    ↓
经济产出↑（如果经济有闲置资源）
```

### 汇率传导路径二：资本流动

```
美元贬值
    ↓
美国资产以本币计价的回报↓（对外国投资者）
    ↓
外国投资者减少持有美国资产
    ↓
美元进一步贬值（自我强化）

美元升值（危机初期）
    ↓
全球需要美元偿债/补充保证金
    ↓
美元成为"避险货币"
    ↓
美元流动性危机（"美元荒"）
```

### 汇率传导路径三：资产负债表

```
本币大幅贬值
    ↓
持有外币债务的企业/银行
资产负债表恶化（本币计价的债务↑）
    ↓
企业违约风险↑
    ↓
银行不良贷款↑
    ↓
金融危机深化
```

### 汇率传导路径四：大宗商品

```
美元是世界商品计价货币
    ↓
美元走强 → 黄金/原油/金属以美元计价
              ↓（以其他货币计价的买家变贵）
         全球商品需求↓ → 压制通胀

美元走弱 → 商品以美元计价↓ → 全球通胀上升压力
```

## 六、外汇市场在2008年危机中的表现

```
危机初期（2008年9-10月）：美元升值
    → 全球"美元荒"，各国都需要美元偿还美元债务
    → 所有资产被抛售换取美元
    → 符合"流动性危机"的典型特征

危机深化（2009年）：美元贬值
    → 量化宽松，美联储大规模印钞
    → 对美元信心下降
    → 黄金价格上涨（避险）

新兴市场汇率：
    → 资本外流，本币大幅贬值
    → 外汇储备消耗
    → 部分国家被迫向IMF求助
```

### 外汇市场传导全球危机的机制

```
美国金融危机
    ↓
全球美元流动性收紧（互换额度有限）
    ↓
新兴市场央行抛售美国国债换取美元
    ↓
美国国债收益率上升（加剧美国信贷紧缩）
    ↓
新兴市场本币贬值
    ↓
新兴市场进口商品涨价 → 输入性通胀
    ↓
新兴市场央行被迫收紧政策
    ↓
新兴市场经济进一步恶化
```

## 七、外汇市场的重要指标

| 指标 | 含义 |
|------|------|
| **DXY美元指数** | 美元兑一篮子货币（欧元、日元、英镑等）的加权汇率 |
| **利差交易（Carry Trade）** | 借入低利率货币，买入高利率货币赚取利差 |
| **隐含波动率** | 外汇期权定价反映的未来汇率不确定性 |
| **外汇储备** | 央行持有的外币资产，可用于干预汇率 |

## 八、人民币汇率与全球金融的关系

```
人民币升值压力（2000年代）
    → 中国出口商品价格相对更贵 → 出口竞争力下降
    → 热钱流入中国 → 资产泡沫压力
    → 外汇储备膨胀 → 人民币被动投放 → 通胀压力

人民币贬值压力（2015/2016）
    → 资本外流 → 外汇储备下降
    → 股市下跌联动
    → 全球市场同样承压（中国是全球增长引擎）
```
