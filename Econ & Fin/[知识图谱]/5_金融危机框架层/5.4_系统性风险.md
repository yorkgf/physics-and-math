# 5.4 系统性风险（Systemic Risk）

## 一、系统性风险的定义

**系统性风险**是指**整个金融体系而非个别机构的功能受损**，导致实体经济受到严重负面冲击的风险。

```
单一风险 vs 系统性风险：

单一风险：
  A银行亏损 → A银行倒闭
  → 不影响B银行
  → 对整体经济影响有限

系统性风险：
  A市场崩溃 → 信心丧失
  → B机构担心交易对手 → 停止放贷
  → C企业无法融资 → 裁员
  → D家庭收入↓ → 消费↓
  → E银行因实体经济恶化而亏损
  → 回到D，形成恶性循环
  → 整个金融体系瘫痪 → 实体经济崩溃
```

## 二、系统性风险的来源

### 来源一：规模（Too Big to Fail）
```
系统重要性金融机构（SIFI）：
  规模太大、相互关联太深
  → 一旦倒闭会拖垮整个系统

五大独立投行在2008年前：
  高盛、摩根士丹利、雷曼、贝尔斯登、美林
  → 持有大量衍生品
  → 彼此之间有大量交易对手关系
  → 一家倒闭 → 连锁反应

最终结果：
  五大投行：3家倒闭（雷曼/贝尔斯登清盘）
            2家转为银行控股公司（高盛/摩通）
            1家被收购（美林被美国银行收购）
```

### 来源二：关联性（Too Interconnected to Fail）

```
                            金融机构关联网络

              银行A ←──────→ 银行B
             ↙ ↑    ↘ ↙     ↙ ↑
           ↙   ↑  ↘  ↑  ↙     ↑ ↘
         ↙     ↑  ↘↑  ↙       ↑   ↘
       银行C ←─┴───  ↘  ───────→ 银行D
         │            ↘↑               │
         │              ↘             │
         │         衍生品市场          │
         │              ↑             │
         └──────────────┴─────────────┘
                       ↑ ↑
                 影子银行/对冲基金
                       ↑ ↑
                    实体经济
```

**关联性的危险性**：
```
正常时期：
  A银行与B银行互相交易 → 效率↑ 成本↓
  → 银行间市场活跃
  → 资金配置效率高

危机时期：
  A银行可能倒闭
  → B银行可能损失大量对A的敞口
  → B银行也可能面临困境
  → 迅速传染到C、D、E...

  关联性越高 → 传染越快 → 越危险
```

### 来源三：同质性（Too Homogeneous to Diversify）

```
2008年危机的同质性灾难：

大量金融机构使用相同的风险模型（Gaussian Copula）
    ↓
持有类似的资产组合（MBS/CDO）
    ↓
当MBS/CDO下跌时
    ↓
所有机构同时亏损
    ↓
所有机构同时被追缴保证金
    ↓
所有机构同时抛售
    ↓
市场在一天内同时崩溃

"我们都在同一个浴缸里" —— 某银行风险主管
```

## 三、系统性风险的传导机制

```
                系统性风险传导路径

路径1：直接敞口传染
  机构A持有MBS → MBS贬值 → A亏损
  A是B的交易对手 → B对A有敞口 → B也亏损
  → B是C的交易对手 → C也亏损
  → 连锁反应直到所有机构

路径2：流动性传染 [[4.4_流动性与信心传导]]
  一个机构抛售资产换流动性
  → 其他同类资产价格下跌
  → 持有同类资产的机构也被迫抛售
  → 流动性螺旋

路径3：信息传染 [[4.4_流动性与信心传导]]
  雷曼倒闭的消息
  → 所有人都担心"下一个是谁"
  → 银行间停止互相拆借（同业市场冻结）
  → 即使有偿付能力的机构也借不到钱

路径4：心理传染
  恐慌具有社会性
  → 投资者看到别人抛售 → 自己也开始抛售
  → 即使基本面没有变化
  → 价格下跌本身成为下跌的原因
```

## 四、系统性风险的衡量指标

| 指标 | 说明 | 预警阈值 |
|------|------|---------|
| **CoVaR** | 一个机构对整个系统的边际风险贡献 | 高于平均VaR |
| **SRISK** | 某机构在压力情景下的资本缺口 | > 某机构市值 |
| **DID**（Distress Dependency） | 机构之间违约的相关性 | > 0.3 |
| **CDS利差** | 市场隐含的违约概率 | 急剧扩大 |
| **LIBOR-OIS利差** | 银行间流动性紧张程度 | > 50bp |
| **VIX** | 市场恐慌程度 | > 40 |

## 五、"大而不能倒"与道德风险

```
"大而不能倒"(TBTF)的悖论：

市场预期：
  "大银行不能倒，否则会连累整个系统"
  → 政府会救助
        ↓
银行的行为：
  既然不会倒 → 放心大胆冒险
  → 杠杆率越来越高
  → 高风险业务越来越多
        ↓
结果：
  过度冒险 → 果然出事了
  → 政府不得不救助（因为TBTF）
  → 道德风险被进一步强化
  → 下一次危机规模更大

2008年后TARP救助：
  问题资产救助计划：$7,000亿
  → 银行拿到了钱
  → 高管薪酬照旧
  → 但没有改变TBTF的道德风险逻辑
```

## 六、2008年危机的系统性风险积累过程

```
系统性风险积累（2001-2006年）：

① 规模扩张
  五大独立投行 → 高杠杆运营（30-40倍杠杆）
  两房担保规模 → $5万亿
  金融业利润 → 占全部上市公司40%

② 关联性扩张
  CDS名义本金 → $62万亿（全球GDP）
  影子银行规模 → 接近商业银行
  跨境资金流动 → 高度连通

③ 同质性扩张
  相同风险模型 → VaR模型（都看同样风险）
  相同资产偏好 → CDO（MBS再打包）
  相同评级 → AAA级CDO占比极高

④ 监管套利
  商业银行 → 通过SPV转移风险 → 规避资本要求
  投资银行 → 不受商业银行监管
  → 整体系统资本不足

↓
系统性风险积累到临界点

↓
[[5.1_明斯基时刻]]

↓
系统性风险爆发（2007-2009年）
```

## 七、防止系统性风险的政策工具

```
2008年后金融监管改革：

多德-弗兰克法案（2010年）：
  ✓ "沃尔克规则"：禁止银行自营交易
  ✓ 系统重要性机构(SIFI)资本要求提高
  ✓ 设立金融稳定监督委员会(FSOC)
  ✓ 要求SIFI提交"生前遗嘱"(Living Will)

中央清算机制：
  OTC衍生品 → 强制中央清算
  → 降低关联性风险

压力测试：
  定期对系统重要性银行进行压力测试
  → 提前发现资本不足的问题

宏观审慎监管：
  逆周期资本缓冲（经济好时多留资本）
  → 为危机时准备缓冲
```

## 八、系统性风险 vs 非系统性风险

```
                    风险分类对比

非系统性风险（可通过分散化消除）：
  A公司亏损 → B公司不受影响
  → 持有多个公司股票 → 风险分散

系统性风险（无法通过分散化消除）：
  整个金融体系出问题 → 无论持有什么股票都亏损
  → 2008年：持有任何资产都可能亏损
  → 只有现金/国债能保值

核心含义：
  分散化可以消除非系统性风险
  但系统性风险是系统性本身的问题
  → 需要系统性解决方案（监管/央行）
```
