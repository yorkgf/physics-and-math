# 5.2 资产负债表衰退（Balance Sheet Recession）

## 一、资产负债表衰退的定义

**资产负债表衰退**由前日本央行官员、哈佛大学经济学教授辜朝明（Richard Koo）提出，指的是**私人部门（企业/居民）专注于修复资产负债表而停止借款，即使利率为零也依然如此**的经济状态。

```
正常经济：
  企业：借更多钱 → 投资扩张 → 利润↑ → 经济↑

资产负债表衰退：
  资产价格暴跌
        ↓
  企业/家庭净资产变为负数
  （资产<负债，杠杆率>100%）
        ↓
  目标从"利润最大化"转变为"债务最小化"
        ↓
  即使有闲置现金 → 全部用于还债
        ↓
  不再新增借款（甚至提前还债）
        ↓
  货币政策失效（零利率也不借钱）
        ↓
  流动性陷阱 [[5.5_流动性危机]]
```

## 二、资产负债表衰退的形成路径

```
第一步：泡沫形成（资产价格↑）
  低利率 + 宽松信贷 → 资产价格上涨
  → 企业/居民杠杆率上升
  → 借贷扩张（看起来"财务健康"）

第二步：泡沫破裂（资产价格↓↓）
  利率上升/需求下降 → 资产价格暴跌
  → 抵押品价值大幅缩水
  → 净资产变为负数（债务 > 资产）

第三步：债务最小化目标确立
  企业CEO/家庭首要目标：还债
  → 不再进行正常的商业投资
  → 停止消费（居民）
  → 提前偿还银行贷款

第四步：信用收缩（贷款需求消失）
  所有企业都在还债 → 没有人想借钱
  → 货币乘数急剧下降
  → M2（货币供应）收缩
  → 即使央行降息也无法刺激信贷

第五步：政策失灵
  货币政策（降息）：无效
  → 企业不愿借款（资产负债表未修复前不扩张）
  → 财政政策（政府支出）：唯一有效工具
  → 需要政府大规模举债来弥补私人部门缺口
```

## 三、资产负债表衰退 vs 传统经济衰退

| 维度 | 传统衰退 | 资产负债表衰退 |
|------|---------|--------------|
| 原因 | 需求不足 | **资产价格暴跌导致债务超重** |
| 家庭行为 | 减少消费 | **全部收入用于还债** |
| 企业行为 | 减少投资 | **不借款、不投资、全力还债** |
| 银行角色 | 惜贷 | **惜贷 + 企业不借钱** |
| 货币政策 | 有效（降息→借贷↑） | **失效（零利率也不借）** |
| 有效政策 | 货币+财政 | **只有财政政策（政府举债）** |
| 典型案例 | 普通经济衰退 | 1990年代日本、2008年后美国 |

## 四、2008年危机中的资产负债表衰退

```
                          资产价格暴跌

美国房价：2006年高点 → 2009年低点
  平均跌幅：约30%
  部分地区跌幅：50%+

美股：标普500
  2007年高点1,565 → 2009年低点666
  跌幅：57%

财富蒸发：
  家庭净资产：减少$11万亿
  股市市值：减少$8万亿
  房价下跌：减少$3万亿

                        ↓

      企业/居民资产负债表严重受损

                        ↓

              资产负债表衰退启动

                        ↓

修复过程（2010年后）：
  企业 → 不投资 → 囤积现金（ Apple持$2,000亿现金）
  居民 → 减少消费 → 提高储蓄率
  美国家庭储蓄率：2007年3% → 2015年7%+
  企业利润 → 不再投资扩张 → 回购股票而非建厂

                        ↓

            复苏极其缓慢（耗时8年）
  失业率恢复危机前水平：直到2016年
  GDP增速：长期低于危机前
```

## 五、资产负债表衰退的自我强化机制

```
资产价格下跌
        ↓
抵押品价值缩水
        ↓
净资产变负
        ↓
债务最小化 → 消费/投资↓
        ↓
需求减少 → 物价下跌
        ↓
实际债务负担↑（通胀降低债务价值的功能消失）
        ↓
更多企业/居民陷入资产负债表衰退
        ↓
需求进一步减少
        ↓
企业营收↓ → 裁员 → 失业↑ → 消费↓↓

这就是为什么资产负债表衰退
往往比普通衰退持续更久、更难恢复
```

## 六、日本1990年代 vs 美国2008年

```
日本（1990年代）：资产负债表衰退的原型

  1980年代末：房地产/股市泡沫
  1989年：央行加息 → 泡沫破裂
  1990年代：资产价格持续下跌
  → 企业拼命还债 → 不投资
  → 持续"失去的20年"
  → 央行零利率 → 仍然无效

  日本的错误应对：
    ✓ 央行快速降息（正确）
    ✗ 政府财政刺激不足（过于保守）
    ✗ 没有及时处理银行不良贷款
    → 衰退持续远超预期

美国（2008年后）：

  2006年：房价泡沫破裂
  2008年：雷曼倒闭
  2009年：资产负债表衰退启动

  美国的成功应对：
    ✓ 央行：量化宽松（直升机撒钱）
    ✓ 财政：TARP + 7870亿刺激
    ✓ 政府大幅举债（弥补私人部门缺口）
    ✓ 直接救助系统重要性机构

    结果：比日本恢复更快（但仍有8年）
```

## 七、资产负债表衰退中的政策组合

### 货币政策（失效阶段）

```
利率降至零 → 企业仍不借款
量化宽松 → 银行储备↑ → 但贷款↑有限
（银行只是把储备存回央行套利，没有放贷）

量化宽松真正的作用：
  → 压低长期利率（支持债券市场）
  → 恢复金融市场功能
  → 阻止债务-通缩螺旋（而非刺激信贷）
```

### 财政政策（唯一有效工具）

```
资产负债表衰退中唯一有效的政策：

政府代替私人部门举债：
  政府举债 ↑ → 弥补私人储蓄缺口
        ↓
  保持总需求不崩溃
        ↓
  企业营收改善 → 慢慢修复资产负债表
        ↓
  逐步退出衰退

风险：政府债务过高 → 主权债务危机
```

### 结构改革

```
资产负债表衰退是"病"，不是"病因"：
  → 需要修复资产负债表（还债/资产增值）
  → 需要修复实体经济结构（去杠杆化）

去杠杆化路径：
  ① 节约（还债）= 痛苦的去杠杆
  ② 通货膨胀（稀释债务）= 缓慢的去杠杆
  ③ 债务违约/重组 = 突然的去杠杆
  ④ 债务货币化（央行买债）= 最快但后遗症最多

[[5.3_债务-通缩循环]] 就是去杠杆化过程的理论框架
```
