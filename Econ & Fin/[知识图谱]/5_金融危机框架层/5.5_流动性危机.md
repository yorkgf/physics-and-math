# 5.5 流动性危机（Liquidity Crisis）

## 一、流动性危机的定义

**流动性危机**是指**金融体系中的流动性突然枯竭**，导致正常的借贷、交易和支付清算功能瘫痪的现象。

```
流动性危机的核心矛盾：

正常状态：
  "我不担心银行会倒闭，我随时可以取钱"
  银行：短期借款 + 长期贷款 → 期限错配
  → 只要所有人不是同时取钱，银行就是安全的

危机状态：
  恐慌 → 所有人同时取钱
  → 银行挤兑（1836年英国 → 2008年货币市场基金）
  → 即使健康的机构也会因流动性枯竭而倒闭
  → 雷曼倒闭：不是因为资不抵债（坏账太多）
        而是因为没有人愿意借钱给它
        → 流动性枯竭 → 被迫清算资产
        → 资不抵债 → 倒闭
```

## 二、流动性危机的两种类型

```
                    流动性危机分类

                        │
        ┌───────────────┼───────────────┐
        ↓               ↓               ↓
   流动性危机        融资流动性        两者同时
   (Market         (Funding           发生
   Liquidity)      Liquidity)         (2008)
        │               │               │
   市场买卖价差      融资渠道枯竭       双重打击
   急剧扩大         无法借到钱        最致命
```

### 类型1：市场流动性危机

```
定义：资产无法按"公平价格"变现

市场流动性正常：
  想卖100万股 → 可以快速找到买家
  → 价格接近市场价
  → 几乎没有损失

市场流动性枯竭：
  想卖100万股 → 找不到买家
  → 必须大幅压低价格才能成交
  → 损失巨大
  → 引发更多抛售

典型案例：
  2008年MBS市场：
    持有CDO的机构想要变现
    → 找不到任何买家
    → 必须打5折、3折甚至更低
    → 账面亏损→真实亏损
```

### 类型2：融资流动性危机（资金枯竭）

```
定义：无法获得现金/融资来满足偿付需求

融资流动性正常：
  企业：银行授信额度$1亿 → 随时可用
  银行：同业拆借市场 → 随时借到钱

融资流动性危机：
  企业：银行突然要求提前还款+不再续期
  银行：同业市场冻结，没人愿意借出
  → 即使有资产也无法融资
  → 无法偿债 → 违约 → 破产

典型案例：
  雷曼兄弟：
    持有大量MBS/CDO
    无法融资（没人愿意借钱）
    → 被迫申请破产保护
    → 不是因为没有好资产
    → 而是因为流动性枯竭
```

## 三、流动性危机 vs 偿付能力危机的区别

```
                    核心区别

偿付能力危机（Solvency Crisis）：
  本质：资产价值 < 负债价值
  问题：长期根本性亏损
  解决方案：注资（政府入股）+ 重组
  典型：银行真的资不抵债

流动性危机（Liquidity Crisis）：
  本质：有足够资产，但无法变现
  问题：短期现金不足
  解决方案：注入流动性（央行借钱）+ 担保
  典型：健康机构被恐慌逼死

2008年雷曼：
  事后分析：即使资产变卖，也不足以偿债
  → 实际上也是偿付能力危机
  → 但直接触发因素是流动性枯竭

2008年AIG：
  本质上是偿付能力问题（CDS赔付超出资本）
  → 但最初表现为流动性危机
  → 政府救助后偿付能力问题才暴露
```

## 四、流动性危机的传导链条

```
流动性危机传导的三个阶段：

阶段一：特定市场流动性枯竭
  次级MBS违约增加
        ↓
  CDO/MBS市场找不到买家
        ↓
  持有CDO的机构无法变现
        ↓

阶段二：蔓延到相关市场
  CDO相关的衍生品（CDS）市场承压
        ↓
  持有类似资产的机构互相担忧
        ↓
  银行间同业市场开始紧张
        ↓
  LIBOR-OIS利差扩大
        ↓

阶段三：全局流动性冻结
  货币市场基金被挤兑
        ↓
  商业票据市场冻结
        ↓
  回购市场冻结
        ↓
  整个短期融资体系瘫痪
        ↓
  实体经济失去流动资金来源
        ↓
  企业裁员/倒闭
        ↓
  失业率上升
```

## 五、央行在流动性危机中的角色

```
                    央行"最后贷款人"
                    (Lender of Last Resort)

古典金本位时代（白芝浩原则，1873年）：
  央行应该：
  ✓ 在危机时向有偿付能力但缺流动性的机构放贷
  ✓ 要求高质量抵押品
  ✓ 惩罚性利率（防止道德风险）

现代央行（2008年危机）突破白芝浩原则：

  ✓ 接受低质量抵押品（包括MBS）
  ✓ 提供无限流动性
  ✓ 设立新型工具直接购买商业票据（CPFF）
  ✓ 与外国央行建立互换额度

这些措施有效阻止了更深的危机
但也加深了道德风险（认为央行会兜底）
```

## 六、影子银行体系的流动性脆弱性

```
传统银行 vs 影子银行的流动性结构：

传统银行（接受存款）：
  存款保险（FDIC $25万）→ 储户不恐慌 → 不挤兑
  → 即使短期流动性不足也有缓冲

影子银行（依赖批发融资）：
  依赖回购协议和商业票据
        ↓
  没有存款保险
        ↓
  一旦市场有风吹草动
  → 货币基金/短期资金提供者立刻撤资
        ↓
  影子银行被迫抛售资产换现金
        ↓
  市场进一步下跌
        ↓
  更多的撤资
        ↓
  螺旋式崩溃
```

## 七、流动性危机的自我强化机制

```
                 流动性螺旋

抵押品价值下跌
        ↓
追加保证金通知（Margin Call）
        ↓
被迫抛售资产
        ↓
资产价格进一步下跌
        ↓
抵押品价值进一步缩水
        ↓
更多的保证金追缴

↓（同时）

资不抵债担忧
        ↓
交易对手停止交易
        ↓
无法融资/展期
        ↓
被迫清算
        ↓
资不抵债
        ↓
交易对手更不敢交易

流动性螺旋 ↔ 偿付能力担忧
两个螺旋相互加强
→ 单独处理一个都不够
→ 需要同时解决流动性+偿付能力
```

## 八、流动性危机的历史对比

```
                  流动性危机对比

大萧条（1929-1933年）：
  起因：股市崩盘 → 银行挤兑
  传导：银行倒闭 → 货币供给收缩
  结果：M2减少30% → 大萧条加剧
  教训：没有央行提供流动性

2008年（全球金融危机）：
  起因：次贷违约 → CDO崩塌
  传导：影子银行 → 货币市场 → 全球
  结果：全球信贷冻结
  教训：影子银行风险被低估
  央行创新工具但响应较慢

2020年3月（新冠危机）：
  起因：疫情冲击 → 流动性枯竭
  传导：速度极快（一个月内）
  结果：但央行响应极快（无限QE）
  教训：快速响应可以阻断螺旋
```
