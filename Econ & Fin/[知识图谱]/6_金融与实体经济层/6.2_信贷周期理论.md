# 6.2 信贷周期理论

## 一、信贷周期的定义

**信贷周期**是金融体系信用供给的**扩张与收缩循环**，是经济周期的核心驱动因素之一。

```
                    信贷周期运行图

                      繁荣期
                    （过度扩张）
                         ↑
          银行竞争   ←─────────────────   低利率环境
          激烈放贷        ↑                   |
               ↓          │                   |
          贷款标准     自我强化            房价上涨
          持续放松        ↑                   |
               ↓          │                   |
          风险大量    ←────┘                   |
          积累                                    |
                   ↑ 临界点                    |
             （[[5.1_明斯基时刻]]）              |
                   ↑                              |
              违约增加    ←────────────────────┘
                   |
                   ↓
                衰退期
               （收缩去杠杆）

银行收紧 ←───────────   违约飙升
贷款标准              资产价格下跌
                   │
                   ↓
                漫长恢复期
               （资产负债表修复）
                   │
                   ↓
                新一轮周期开始
```

## 二、信贷周期与经济周期的关系

```
传统经济周期理论：
  经济波动主要由实体因素驱动
  （技术冲击/财政政策/人口变化）

信贷周期视角：
  金融体系的扩张/收缩是独立的驱动因素
  信贷扩张本身就是经济增长的来源
  （"借来的需求"）

两者的关系：
  信贷扩张 → 需求增加 → 经济增长 → 资产价格上涨
        ↑                               ↓
        ←── 自我强化（正反馈）←────── 抵押品价值↑
                                              ↓
                                    银行更愿意放贷

信贷收缩 → 需求减少 → 经济衰退 → 资产价格下跌
        ↑                               ↓
        ←── 自我强化（负反馈）←────── 抵押品价值↓
                                              ↓
                                    银行更不愿意放贷
```

## 三、信贷扩张如何制造资产泡沫

```
                            信贷创造过程

第一步：初始条件
  低利率环境（央行降息/全球储蓄过剩）
        ↓
  借款人利息负担减轻
        ↓
  第二步：抵押品效应
  房价上涨 → 抵押品价值↑
        ↓
  银行更愿意放贷（"房价永远涨"）
        ↓
  可以借更多钱（Loan-to-Value比例提高）
        ↓
  更多人买得起房 → 需求↑ → 房价↑
        ↓
  第三步：羊群效应
  看到邻居/朋友买房赚钱
        ↓
  更多资金流入房地产
        ↓
  专业炒房者利用杠杆放大收益
        ↓
  第四步：金融创新推波助澜
  次级贷款让没有资质的人也获得贷款
        ↓
  MBS/CDO让全球投资者分担风险
        ↓
  银行资本被释放 → 继续放贷

结果：信贷疯狂扩张
  次级贷款占新增房贷：5% → 20%
  CDO规模：$1万亿 → $6万亿
  杠杆率：银行30-40倍
```

## 四、信贷收缩的经济杀伤力

```
                        信贷紧缩的传导

第一步：去杠杆化
  危机爆发 → 资产负债衰退
        ↓
  银行/企业/居民开始还债
        ↓
  信贷规模收缩

第二步：资产负债表衰退
  债务人全力还债 → 不再借贷 [[5.2_资产负债表衰退]]
        ↓
  总需求大幅萎缩

第三步：信用利差扩大
  违约担忧 → 信用利差急剧上升
        ↓
  即使愿意借贷的企业融资成本也↑↑
        ↓
  第四步：银行惜贷
  违约率上升 → 银行不良贷款↑
        ↓
  银行风险偏好下降 → 惜贷
        ↓
  即使好企业也借不到钱

第五步：实体经济衰退
  企业无法融资 → 裁员/关闭
        ↓
  失业率上升 → 居民收入↓
        ↓
  消费↓ → 企业营收↓
        ↓
  回到第三步 → 螺旋恶化
```

## 五、信贷周期的量化特征

```
                    2008年危机前后的信贷扩张

  居民部门杠杆率（债务/GDP）：

  2000年：70%
  2008年：98%    ← 翻了近30个百分点
  2010年：95%    ← 开始去杠杆

  企业部门杠杆率：
  2000年：65%
  2008年：75%
  2009年后：先降后升（QE后再次上升）

  金融部门杠杆率：
  2000年：60%
  2008年：100%+  ← 最危险的部分

  国际清算银行(BIS)研究：
  信贷/GDP缺口（实际-趋势）
  > 10% → 未来3年内发生危机的概率显著上升

  美国2007年：信贷/GDP缺口超过15%
```

## 六、杠杆率与经济的关系

```
高杠杆是危机的根源

宏观杠杆率分解：
  总杠杆率 = 政府 + 居民 + 企业 + 金融
  = 全部债务 / GDP

杠杆率上升的代价：
  同一收入流可以支撑的债务是有限的
  → 杠杆率上升 → 还本付息负担↑
  → 可用于消费的现金流↓
  → 经济增长放缓

杠杆率下降（去杠杆化）的痛苦：
  违约/债务减记（债权人损失）
  或：节约还债（债务人减少消费）
  或：通胀稀释（所有人共同承担）
  或：央行买债+政府借债（债务转移）
```

## 七、信贷周期对货币政策制定者的启示

```
                        货币政策的两难

目标：维护价格稳定（通胀2%）
        ↓
但信贷周期会产生资产价格泡沫
        ↓
泡沫破裂 → 金融危机 → 经济衰退
        ↓
衰退比通胀更难治理

传统货币政策的局限：
  货币政策主要针对通胀
  → 关注CPI（消费者物价指数）
  → CPI稳定时，信贷泡沫可能已经很大

"金融周期"概念：
  央行应该同时关注：
  ① 通胀（物价稳定）
  ② 信贷/GDP比率（金融稳定）
  ③ 房价（抵押品价值）

宏观审慎政策（Macro-prudential Policy）：
  用监管工具（不是货币政策）应对信贷泡沫
  如：逆周期资本缓冲（经济好时多留资本）
      贷款价值比(LTV)限制（控制抵押贷款成数）
```
