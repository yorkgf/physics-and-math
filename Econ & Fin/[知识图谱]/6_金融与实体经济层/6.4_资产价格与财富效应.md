# 6.4 资产价格与财富效应（Wealth Effect）

## 一、财富效应的定义

**财富效应（Wealth Effect）** 指的是**居民持有的资产价格变动如何传导到消费支出**的机制。

```
                    财富效应传导链

资产价格上涨
（股票/房产/债券）
        ↓
居民部门财富增加
        ↓
居民感到更富有
        ↓
边际消费倾向（MPC）发挥作用
        ↓
消费支出增加
        ↓
企业营收增加 → 就业增加 → 收入增加
        ↓
经济产出（GDP）增加

反向（负财富效应）：
  资产价格暴跌 → 财富缩水 → 消费↓ → 经济衰退
```

## 二、财富效应的规模测算

```
                    财富效应的量化

MPC（边际消费倾向）：
  每增加$1财富，消费增加多少？

美国数据：
  股票财富：MPC ≈ 0.03-0.05（$1股票涨→$0.03-0.05消费涨）
  房产财富：MPC ≈ 0.05-0.08（$1房产涨→$0.05-0.08消费涨）
  养老金财富：MPC ≈ 0.04

为什么房产MPC更高？
  房产流动性差，不能随时变现
  但"住了更好的房子"的满足感更直接
  且房产价值影响居民的"安全感"

计算示例（2008年危机）：
  美国家庭财富蒸发：$11万亿
  其中房产：-$3万亿，股票：-$8万亿

  消费减少估算：
  = $8万亿 × 0.04 + $3万亿 × 0.06
  ≈ $320亿 + $180亿
  ≈ $5000亿

  相当于GDP的3-4%
  → 消费大幅下滑
  → 经济严重衰退
```

## 三、不同资产的财富效应

```
                    各类资产的财富效应

房产（最重要）：
  ✓ 直接影响居住体验（居住满足感）
  ✓ 影响消费信心（"感觉有钱了"）
  ✓ 杠杆效应（房价↑→可用抵押贷款↑）
  ✓ 流动性低 → 变现不频繁，但心理影响大
  缺点：流动性低（不能像股票一样随时卖）

股票：
  ✓ 流动性高 → 可以随时变现
  ✓ 变现后直接增加消费
  缺点：持有集中于富裕人群（但覆盖面仍广）

债券：
  债券价格上涨通常反映利率下降
  → 对消费的刺激较间接
  → 持有者多为机构投资者

养老金/401(k)：
  美国退休储蓄主要在股市
  股市涨跌直接影响退休计划
  → 对老年人的消费影响大
```

## 四、资产价格传导到消费的完整路径

```
                    资产→消费的完整路径

路径一：直接变现效应
  股价↑ → 投资者卖掉部分股票 → 现金增加 → 消费↑
  房产↑ → 申请房屋净值贷款(HELOC) → 现金增加 → 消费↑
  （后金融危机时代：这条路基本关闭）

路径二：消费信心效应
  看到账户资产↑ → 感到财务状况改善
  → 对未来更有信心 → 现在多消费
  （心理上的"财富感受"）

路径三：跨期替代效应
  资产上涨 → 预期未来财富↑
  → "反正我以后会有钱，现在可以多花"
  → 当前消费增加

路径四：资产负债效应
  资产↑ + 负债固定 → 净资产↑
  → 财务状况改善 → 借贷成本↓
  → 更愿意借贷消费
  （低利率环境下尤其重要）
```

## 五、负财富效应（Reverse Wealth Effect）

```
2008年危机的负财富效应：

房价下跌30%：
  美国400万个家庭丧失房产
        ↓
  负资产家庭（房贷>房价）：超过1000万个
        ↓
  居民不敢消费（"我是负翁"）
        ↓
  消费骤降

具体传导：
  2007年高点 → 2010年低点
  消费支出：-2.5%（2008-2009年）
  消费信心指数：-50%（历史最大跌幅）

额外效应（流动性约束）：
  危机前：房价↑ → 抵押贷款再融资 → 套现消费
  危机后：这条套现路径关闭
  → 实际消费下降超出财富效应估算
```

## 六、财富效应与金融体系的双向关系

```
                    财富效应 ↔ 金融体系

金融体系 → 资产价格 → 财富效应 → 消费 → GDP
    ↑                                    ↓
    │              正反馈循环               │
    │                                    │
  财富效应通过消费影响企业盈利
  → 盈利↑ → 股价↑ → 财富效应↑

但危机时反向：
  资产价格↓ → 财富效应↓ → 消费↓ → GDP↓
    ↑                                    ↓
    │              负反馈循环               │
    │                                    │
  消费↓ → 企业营收↓ → 裁员 → 失业↑
  → 居民收入↓ → 消费进一步↓
  → 房价进一步承压 → 抵押贷款违约↑
  → 银行不良贷款↑ → 银行惜贷
  → 信贷紧缩 → 投资↓
```

## 七、财富效应的局限性

```
财富效应的三个"但书"：

1. 流动性约束
  老年人持有房产但流动性差
  → 即使房产价值上涨，也变现不了
  → 财富效应无法兑现

2. 财富集中度
  资产（股票/房产）集中于富人
  富人的MPC较低（$1对他们影响不大）
  → 整个社会的MPC比估算的低

3. 预期效应
  如果资产上涨让人担心未来会跌
  → 不会增加消费
  → 反而会更加节约

2008年后美国的实际情况：
  虽然美联储通过QE推高了股价
  但普通居民的财富效应很弱
  原因：房产没有涨回来 + 收入停滞
  → 财富效应主要惠及持有股票的富有群体
```
