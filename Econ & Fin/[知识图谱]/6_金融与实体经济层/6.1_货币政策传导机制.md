# 6.1 货币政策传导机制

## 一、货币政策传导的基本框架

货币政策传导描述的是**央行如何通过调整利率和流动性，影响实体经济**的过程。

```
                    货币政策传导路径
                          │
            央行调整政策利率 / 流动性供给
                          │
            ┌──────────────┼──────────────┐
            ↓              ↓              ↓
      货币市场利率     债券市场利率     资产价格
      (短期利率)      (长期利率)      (股票/房产)
            │              │              │
            └──────────────┴──────────────┘
                          │
                          ↓
                   信贷条件改善
                   (企业/居民愿意借钱)
                          │
                          ↓
              ┌────────────────────┐
              │      实体经济       │
              │  投资↑ / 消费↑ / 就业↑  │
              └────────────────────┘
```

## 二、货币政策传导的六大渠道

### 渠道1：利率渠道（最直接）

```
央行降息 → 政策利率↓ → 银行拆借利率↓ → 企业/居民贷款利率↓

对企业：
  融资成本↓ → 投资意愿↑ → 资本开支↑ → 产能扩张

对居民：
  按揭利率↓ → 月供↓ → 可支配收入↑ → 消费↑
  房贷利率↓ → 购房需求↑ → 房地产↑ → 财富效应↑

传导的局限：
  当利率降到接近零（流动性陷阱）时
  → 进一步降息无法再刺激借贷
  → 利率渠道失效
```

### 渠道2：信贷渠道（银行借贷）

```
货币宽松
        ↓
银行存款准备金↑
        ↓
银行可贷资金↑
        ↓
银行竞争激烈 → 放贷标准放松
        ↓
企业/居民获得更多贷款
        ↓
投资/消费↑

但当银行惜贷时（危机时）：
  即使银行有充足储备
  → 银行担心违约 → 不愿意放贷
  → 信贷渠道断裂
  → 货币政策失效
  → [[5.2_资产负债表衰退]]
```

### 渠道3：资产价格渠道（财富效应）

```
降息/宽松 → 债券价格↑（利率降则债券价格升）
        ↓
股价↑（折现率↓ → PE倍数↑）
        ↓
房价↑（抵押贷款利率↓ → 购房需求↑）
        ↓
居民财富↑（股票+房产价值）
        ↓
消费↑（财富效应）

财富效应的规模：
  美联储研究：家庭财富每增加$1，消费增加$0.03-0.05
  美国股市+房市：2009-2020年财富增长$50万亿+
  → 消费成为经济复苏的主要驱动力
```

### 渠道4：汇率渠道

```
降息 → 美元利率相对下降 → 美元贬值预期
        ↓
美元贬值 → 美国出口商品更便宜 → 出口↑
        ↓
美国进口更贵 → 进口替代 → 国内产出↑

美元的"政策信号"效应：
  美联储降息/宽松 → 市场预期美元走弱
  → 即使没有实际降息
  → 美元可能已经开始贬值
  → 传导就已经开始了
```

### 渠道5：通胀预期渠道

```
宽松货币政策 → 货币供给↑
        ↓
通胀预期上升（人们预期未来物价上涨）
        ↓
企业预期未来需求↑ → 现在就开始投资
        ↓
居民预期未来物价涨 → 现在就消费（避免以后更贵）
        ↓
现在的投资/消费↑

关键：通胀预期必须"锚定"
  如果通胀预期失控 → 企业/居民反而不敢消费
  → 等待价格回落 → 经济收缩
```

### 渠道6：资产负债表渠道

```
宽松政策 → 企业融资条件改善
        ↓
企业资产负债表修复
        ↓
外部融资溢价↓（银行更愿意借钱）
        ↓
企业投资↑

[[6.3_金融加速器理论]] 描述了
这一渠道在衰退时的逆向放大效应
```

## 三、2008年危机中的货币政策应对

```
央行政策响应时间轴：

2007年9月：开始降息
  联邦基金利率：5.25% → 开始下调
        ↓

2008年：快速降息至零
  2008年12月：联邦基金利率 0-0.25%
  → 传统利率政策触及下限
  → 常规工具用尽
        ↓

2008年11月：QE1（第一轮量化宽松）
  购买机构MBS + 国债
  规模：$6,000亿
        ↓

2009年3月：QE1扩大
  追加购买MBS/国债
  总规模：$1.7万亿
        ↓

2010年11月：QE2
  购买6000亿美元国债
  规模：$6,000亿
        ↓

2012年9月：QE3（无限版）
  每月购买$850亿资产
  直到就业市场"实质性改善"
  规模：不设上限
```

## 四、货币政策传导为何在危机中部分失效

```
利率降到零 → 传导到实体经济效果有限

原因一：银行惜贷 [[5.2_资产负债表衰退]]
  即使利率为零
  → 银行担心违约 → 不放贷
  → 企业/居民不愿借贷（资产负债表受损）
  → 货币乘数崩溃

原因二：货币流通速度下降 [[5.3_债务-通缩循环]]
  央行基础货币↑↑
  但居民/企业囤积现金（避险）
  → 货币流通速度V急剧下降
  → MV = PY → 即使M↑也不一定Y↑

原因三：流动性陷阱
  当所有人都预期未来会更差
  → 利率再低也不想借钱
  → 货币政策刺激需求的能力达到极限

唯一有效：财政政策 [[5.2_资产负债表衰退]]
  政府代替私人部门借钱/消费
  → 弥补总需求缺口
```

## 五、量化宽松（QE）的特殊传导机制

```
传统货币政策：
  央行：调整政策利率（联邦基金利率）
  传导：利率下降 → 借贷增加

量化宽松（非常规政策）：
  央行：直接购买长期资产
  传导：多条路径同时作用

路径A：资产组合再平衡
  央行买国债/ MBS
  → 投资者持有的国债↓（流动性↑）
  → 投资者被"挤出"去购买风险更高的资产
  → 股票/企业债需求↑
  → 资产价格上涨

路径B：信号效应
  央行大规模购买 → 市场认为央行在"兜底"
  → 投资者信心恢复
  → 信用利差收窄
  → 金融市场功能恢复

路径C：通胀预期管理
  大量购买 = 未来不会快速收紧
  → 锚定长期通胀预期
  → 实际利率不上升

路径D：汇率效应
  QE = 货币供给大幅增加
  → 本币贬值预期
  → 出口受益
```
