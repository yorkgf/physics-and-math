## 二、第一层：危机的起因 — 金融创新产品

### 2.1 证券化的过长资金链

```
[[3.4_分层结构与风险转移]] 详解的资金链：

次级借款人（无力还款）
        ↓
贷款经纪商（赚佣金，不管还款能力）[[3.4_道德风险]]
        ↓
发起银行（打包卖给两房）[[3.1_MBS]]
        ↓
两房（政府隐性背书，疯狂收购）[[3.1_MBS]]
        ↓
投资银行（再打包成CDO）[[3.2_CDO]]
        ↓
评级机构（给CDO AAA评级）[[3.2_CDO]]（评级假设失效）
        ↓
CDO²（极度复杂，不透明）[[3.2_CDO]]
        ↓
CDS保险（AIG承保）[[3.3_CDS]]
        ↓
全球投资者（持有各级CDO）

每个环节：
  ✓ 赚取手续费/佣金
  ✗ 没有人真正关心贷款质量
  ✗ 没有人真正持有最终风险
```

### 2.2 分层机制如何制造了虚假的安全感

```
[[3.4_分层结构]] 的核心漏洞：

CDO优先层级（评级AAA）：
  理论：前10%损失由优先层级吸收
  假设：资产池违约率<10%
  现实：次级贷款违约率 = 20%+

为什么假设失效？[[3.2_CDO]]：
  ① 违约相关性在危机时接近1（"跳伞效应"）
     → 分散化失效，所有MBS同时违约
  ② 房价开始下跌（反向强化违约）
  ③ 次级贷借款人的现金流无以为继（加息周期）
```

## 三、第二层：危机触发 — 明斯基时刻

### 3.1 从信贷扩张到明斯基时刻

```
[[6.2_信贷周期]] + [[5.1_明斯基时刻]]：

2001-2003年：[[6.1_货币政策传导机制]]
  美联储降息至1%
        ↓
  廉价资金涌入房地产市场
        ↓
  房价上涨 → 抵押品价值↑
        ↓
[[6.2_信贷周期]]扩张：
  贷款标准持续放松
  次级贷款占比：5% → 20%
  CDO规模：$1万亿 → $6万亿
        ↓
[[3.4_道德风险]] 积累：
  所有环节都在"击鼓传花"
  → 没有人真正负责
        ↓

[[5.1_明斯基时刻]] 触发（2006-2007年）：
  美联储加息周期启动（利率↑）
  可调利率次级贷月供暴涨
  房价停止上涨（再融资困难）
  → 违约率从5%飙升至15%+
        ↓
  CDO优先层级也开始亏损
        ↓
  泡沫破裂 → 危机开始
```

### 4.2 各市场的具体传导机制

#### [[4.1_汇率传导]] — 美元荒
```
[[2.4_外汇市场]]：

危机初期：美元升值（2008年9-10月）
  → 全球"美元荒"
  → 所有资产被抛售换取美元
  → 即使美联储降息也无法让美元走弱

机制：[[4.4_流动性与信心传导]]
  持有美元债务的机构需要美元还债
  → 抛售一切资产买美元
  → 美元升值 + 所有资产下跌

这就是为什么2008年9-10月
全球所有资产（股票/债券/黄金）同步下跌
```

#### [[4.2_商品价格传导]] — 需求崩溃
```
[[2.5_大宗商品市场]]：

Phase 1（2007-2008.7）：
  美元贬值 + 金融投机
  原油：$60 → $147（虚假繁荣）

Phase 2（2008.7-12月）：
  金融危机 → 需求预期崩溃
  原油：$147 → $32（跌幅78%）

Phase 3（2009-2011）：
  QE → 美元泛滥
  原油反弹 + 黄金创历史新高

[[4.2_大宗商品价格传导]]：
  油价暴跌 → 产油国财政崩溃
  油价暴跌 → 能源公司违约率上升
  油价暴跌 → 通缩压力上升 [[5.3_债务-通缩循环]]
```

#### [[4.3_证券市场价格传导]] — 信用利差崩塌
```
[[2.2_债券市场]]：

信用利差的变化：
  2007年初：~150bp（正常）
  2007年中：~300bp（开始担忧）
  2008年3月：~500bp（贝尔斯登）
  2008年10月：>1000bp（全面恐慌）

[[4.3_证券市场价格传导]]：
  信用利差↑ → 企业融资成本↑↑
  → 债券市场发行停滞
  → 企业被迫削减投资/裁员
  → 实体经济进一步恶化
```

### 5.2 关键转折点：雷曼倒闭

```
雷曼兄弟倒闭（2008年9月15日）为何如此关键？

[[5.4_系统性风险]]：
  雷曼是第四大投行，系统重要性极高
  持有大量CDS合约（交易对手遍布全球）
  → 雷曼倒闭 → 所有交易对手同时面临损失

[[5.5_流动性危机]]：
  雷曼倒闭前，贝尔斯登被摩根大通收购（AIG提供流动性支持）
  → 市场认为政府不会让大投行倒闭
  → 雷曼倒闭打破了这一预期
  → 所有人都意识到：政府不会救所有人
  → 恐慌达到顶点

[[4.4_流动性与信心传导]]：
  雷曼倒闭
      ↓
  恐慌：下一个是谁？高盛？摩根士丹利？
      ↓
  交易对手拒绝与任何投行交易
      ↓
  投行无法融资 → 被迫清盘
      ↓
  流动性螺旋式崩塌
```

### 5.3 AIG危机：CDS的连锁反应

```
[[3.3_CDS]] 的崩溃路径：

AIG作为CDS卖方：
  向全球投资者出售CDS → 收取保费
  持有CDO → 自以为风险可控

危机发生时：
  CDO违约增加 → CDS需要赔付
  AIG资本不足以赔付 → 资本缺口$2000亿+

[[5.4_系统性风险]]：
  如果AIG倒闭
      ↓
  $3000亿CDS敞口触发赔付
      ↓
  持有CDS的银行获得意外赔付（收益）
      ↓
  但AIG的交易对手拿不到钱 → 连锁违约
      ↓
  系统性崩溃

[[6.1_货币政策传导机制]]：
  美国政府救助AIG：$850亿 → 79.9%股权
  → 阻止了更深的系统性崩溃
  → 但道德风险更加严重
```

## 七、第六层：政策应对 — 知识点综合

### 7.1 货币政策的应对

```
[[6.1_货币政策传导机制]] 在危机中的运用：

降息：
  联邦基金利率：5.25% → 0-0.25%
  → 利率渠道：很快触及零下限 [[5.5_流动性危机]]

量化宽松（QE）：
  QE1：$1.7万亿（MBS+国债）
  QE2：$6,000亿（国债）
  QE3：无限版

QE的作用机制：
  [[6.1_货币政策传导机制]] 详解：
  ① 资产组合再平衡（投资者被挤出去买风险资产）
  ② 信号效应（央行兜底 → 信心恢复）
  ③ 压低长期利率（支持债券市场）
  ④ 注入流动性（[[4.4_流动性与信心传导]]）

[[5.5_流动性危机]] 的应对工具：
  CPFF（商业票据融资工具）→ 购买商业票据
  PDCF（交易商信贷工具）→ 向投行提供流动性
  TSLF（证券借贷工具）→ 用国债换风险资产
  → 直接解决了影子银行流动性枯竭问题
```

### 7.2 财政政策的应对

```
[[5.2_资产负债表衰退]] 的财政应对：

财政刺激的作用：
  政府代替私人部门举债/消费
      ↓
  弥补总需求缺口
      ↓
  企业营收改善 → 慢慢修复资产负债表

规模：
  TARP（问题资产救助计划）：$7,000亿
  奥巴马7870亿刺激计划
  救助两房：$2,000亿+
  救助AIG：$1,850亿

问题：
  救助银行 → 道德风险 [[5.4_系统性风险]]
  （银行高管薪酬不变，纳税人买单）
  → 为后来的占领华尔街运动埋下伏笔
```

## 九、对中国金融市场的启示

```
本知识图谱对中国金融市场的现实意义：

1. [[3.4_分层结构]] 的教训：
   → 金融创新必须与激励机制配套
   → 发起机构必须保留部分风险
   → 信息透明度至关重要

2. [[5.1_明斯基时刻]] 的预警：
   → 关注信贷/GDP缺口的快速上升
   → 关注居民杠杆率的急剧攀升
   → 关注影子银行规模的扩张

3. [[5.4_系统性风险]] 的启示：
   → 建立系统重要性金融机构的监管框架
   → 防止"大而不能倒"的道德风险
   → 推动衍生品中央清算

4. [[5.2_资产负债表衰退]] 的借鉴：
   → 货币政策+财政政策必须协调配合
   → 央行流动性注入不能替代财政刺激
   → 去杠杆化需要耐心

5. [[4.x_市场间传导]] 的认识：
   → 股市/汇市/房市/商品的联动性
   → 需要建立宏观审慎监管框架
   → 关注跨市场风险传染
```
