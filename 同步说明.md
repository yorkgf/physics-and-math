# Obsidian Vault WPS 云盘同步指南

## 同步脚本已创建

脚本位置: `/home/yorkgf/.local/bin/obsidian-wps-sync`

## 使用方法

### 手动同步
在终端运行：
```bash
obsidian-wps-sync
```

然后选择：
- `1` - 同步到 WPS 云盘 (Vault → Cloud)
- `2` - 从 WPS 云盘恢复 (Cloud → Vault)
- `3` - 查看同步状态
- `4` - 退出

### 设置自动同步（可选）

添加定时任务，每小时自动同步：

```bash
# 编辑 crontab
crontab -e

# 添加以下行（每小时同步一次）
0 * * * * /home/yorkgf/.local/bin/obsidian-wps-sync <<< $'1\n4\n' >> /tmp/obsidian-sync.log 2>&1
```

或者每 30 分钟同步一次：
```bash
*/30 * * * * /home/yorkgf/.local/bin/obsidian-wps-sync <<< $'1\n4\n' >> /tmp/obsidian-sync.log 2>&1
```
