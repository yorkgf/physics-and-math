# 矢量恒等式总结

本笔记汇总物理中最常用的矢量恒等式，分为**代数恒等式**和**微分恒等式**两类。所有恒等式均与坐标系无关（用指标/爱因斯坦求和约定可统一证明）。

---

## 一、代数恒等式

### 1.1 点积与叉积的基本关系

$$\mathbf{A}\cdot\mathbf{B} = \mathbf{B}\cdot\mathbf{A} \qquad \text{（点积交换律）}$$

$$\mathbf{A}\times\mathbf{B} = -\mathbf{B}\times\mathbf{A} \qquad \text{（叉积反交换）}$$

$$\mathbf{A}\cdot(\mathbf{B}\times\mathbf{C}) = \mathbf{B}\cdot(\mathbf{C}\times\mathbf{A}) = \mathbf{C}\cdot(\mathbf{A}\times\mathbf{B}) \qquad \text{（混合积轮换不变）}$$

### 1.2 BAC–CAB 规则 ⭐

$$\boxed{\mathbf{A}\times(\mathbf{B}\times\mathbf{C}) = \mathbf{B}(\mathbf{A}\cdot\mathbf{C}) - \mathbf{C}(\mathbf{A}\cdot\mathbf{B})}$$

> **记法：** $\mathbf{A}\times(\mathbf{B}\times\mathbf{C}) = \mathbf{B}(\mathbf{A}\cdot\mathbf{C}) - \mathbf{C}(\mathbf{A}\cdot\mathbf{B})$，即 **B**AC–C**A**B（中间那个减后面那个，系数是与剩余两个的点积）。

**推论：**
$$(\mathbf{A}\times\mathbf{B})\times\mathbf{C} = \mathbf{B}(\mathbf{A}\cdot\mathbf{C}) - \mathbf{A}(\mathbf{B}\cdot\mathbf{C})$$

$$|\mathbf{A}\times\mathbf{B}|^2 = |\mathbf{A}|^2|\mathbf{B}|^2 - (\mathbf{A}\cdot\mathbf{B})^2 \qquad \text{（[[矢量分析/Lagrange恒等式|Lagrange 恒等式]]）}$$

## 二、微分恒等式（∇ 算子）

### 2.1 两个基本零恒等式 ⭐⭐

$$\boxed{\nabla\times(\nabla f) \equiv 0} \qquad \text{（梯度无旋）}$$

$$\boxed{\nabla\cdot(\nabla\times\mathbf{F}) \equiv 0} \qquad \text{（旋度无散）}$$

> **物理含义：**
> - 梯度无旋 → 保守力场可以引入势函数（$\mathbf{E}=-\nabla\varphi$）
> - 旋度无散 → 磁场可以引入矢量势（$\mathbf{B}=\nabla\times\mathbf{A}$）
>
> 这两个恒等式是[[矢量分析/亥姆霍兹定理]]亥姆霍兹分解自洽的数学保证。

### 2.2 乘积法则

设 $f,g$ 为标量场，$\mathbf{F},\mathbf{G}$ 为矢量场：

**梯度的乘积法则：**
$$\nabla(fg) = f\nabla g + g\nabla f$$

$$\nabla(\mathbf{F}\cdot\mathbf{G}) = \mathbf{F}\times(\nabla\times\mathbf{G}) + \mathbf{G}\times(\nabla\times\mathbf{F}) + (\mathbf{F}\cdot\nabla)\mathbf{G} + (\mathbf{G}\cdot\nabla)\mathbf{F}$$

**散度的乘积法则：**
$$\nabla\cdot(f\mathbf{F}) = f(\nabla\cdot\mathbf{F}) + \mathbf{F}\cdot(\nabla f)$$

$$\nabla\cdot(\mathbf{F}\times\mathbf{G}) = \mathbf{G}\cdot(\nabla\times\mathbf{F}) - \mathbf{F}\cdot(\nabla\times\mathbf{G})$$

**旋度的乘积法则：**
$$\nabla\times(f\mathbf{F}) = f(\nabla\times\mathbf{F}) + (\nabla f)\times\mathbf{F}$$

$$\nabla\times(\mathbf{F}\times\mathbf{G}) = \mathbf{F}(\nabla\cdot\mathbf{G}) - \mathbf{G}(\nabla\cdot\mathbf{F}) + (\mathbf{G}\cdot\nabla)\mathbf{F} - (\mathbf{F}\cdot\nabla)\mathbf{G}$$

### 2.3 二阶微分恒等式

$$\nabla^2 f = \nabla\cdot(\nabla f) \qquad \text{（拉普拉斯算子定义）}$$

$$\nabla\times(\nabla\times\mathbf{F}) = \nabla(\nabla\cdot\mathbf{F}) - \nabla^2\mathbf{F} \qquad \text{（旋度的旋度）⭐}$$

> **"旋度的旋度"**是电动力学中推导电磁波方程的关键步骤：
> $$\nabla\times(\nabla\times\mathbf{E}) = \nabla(\underbrace{\nabla\cdot\mathbf{E}}_{=\,\rho/\varepsilon_0}) - \nabla^2\mathbf{E}$$

$$\nabla^2(fg) = f\nabla^2 g + g\nabla^2 f + 2(\nabla f)\cdot(\nabla g)$$

## 三、常用场的梯度/散度/旋度

设 $\mathbf{r} = x\hat{x}+y\hat{y}+z\hat{z}$，$r = |\mathbf{r}|$：

| 表达式 | 结果 | 备注 |
|--------|------|------|
| $\nabla r$ | $\hat{r}$ | $r$ 的梯度是单位径向矢量 |
| $\nabla(r^n)$ | $n\,r^{n-2}\mathbf{r}$ | 幂函数梯度 |
| $\nabla\left(\dfrac{1}{r}\right)$ | $-\dfrac{\hat{r}}{r^2}$ | 库仑势的梯度 ⭐ |
| $\nabla\cdot\mathbf{r}$ | $3$ | 位置矢量的散度 |
| $\nabla\cdot\hat{r}$ | $\dfrac{2}{r}$ | 单位径向矢量的散度 |
| $\nabla\cdot\left(\dfrac{\hat{r}}{r^2}\right)$ | $4\pi\,\delta^3(\mathbf{r})$ | 库仑场的散度（含 delta 函数）⭐ |
| $\nabla\times\mathbf{r}$ | $\mathbf{0}$ | 位置矢量无旋 |
| $\nabla\times\hat{r}$ | $\mathbf{0}$ | 径向场无旋 |
| $\nabla^2\left(\dfrac{1}{r}\right)$ | $-4\pi\,\delta^3(\mathbf{r})$ | 格林函数 ⭐ |

> **最重要的一行：** $\nabla\cdot\left(\dfrac{\hat{r}}{r^2}\right) = 4\pi\delta^3(\mathbf{r})$
> 这是库仑定律和高斯定律等价的数学基础。

## 四、快速记忆策略

```
代数恒等式：
  混合积 → 轮换不变（循环右移符号不变）
  BAC-CAB → 记住字母顺序，前减后

微分恒等式：
  乘积散度 → f散F + F·梯f      （"f散F" 先散，后梯点积）
  乘积旋度 → f旋F + 梯f×F      （"f旋F" 先旋，后梯叉积）
  旋度的旋度 → 梯(散) - 拉普拉斯  （"梯散减拉"）

零恒等式：
  梯度 → 无旋（∇×∇f = 0）
  旋度 → 无散（∇·∇×F = 0）
```
