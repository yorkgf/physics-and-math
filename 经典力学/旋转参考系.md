# 旋转参考系（索引）

> 旋转参考系是经典力学中最重要的非惯性参考系之一。本页为知识索引，串联所有相关笔记。

---

## 核心方程

$$m\boldsymbol{a}' = \boldsymbol{F} \underbrace{- m\boldsymbol{\omega}\times(\boldsymbol{\omega}\times\boldsymbol{r})}_{\text{离心力}} \underbrace{- 2m\boldsymbol{\omega}\times\boldsymbol{v}'}_{\text{科里奥利力}} \underbrace{- m\dot{\boldsymbol{\omega}}\times\boldsymbol{r}}_{\text{欧拉力}}$$

→ 完整推导见 [[旋转系中的牛顿第二定律]]

---

## 三大伪力

```
旋转参考系中的伪力
├── 离心力          ← 始终存在（ω ≠ 0）
│   径向向外，m ω² r⊥
│
├── 科里奥利力      ← 质点运动时存在（v' ≠ 0）
│   [[科里奥利力]]
│   -2m ω × v'，垂直于速度
│
└── 欧拉力          ← 角速度变化时存在（ω̇ ≠ 0）
    [[欧拉力]]
    -m ω̇ × r，切向
```
