# 欧拉力 (Euler Force)

## 概述

欧拉力是**旋转参考系**中，当角速度 $\boldsymbol{\omega}$ **随时间变化**（旋转加速或减速）时出现的惯性力（伪力）。它与[[科里奥利力]]、离心力共同构成旋转系中完整的惯性力体系。

$$\boxed{\boldsymbol{F}_{\text{Euler}} = -m\dot{\boldsymbol{\omega}} \times \boldsymbol{r}}$$

---

## 来源推导

在旋转参考系的完整运动方程中（见[[科里奥利力#旋转参考系中的运动方程]]）：

$$m\boldsymbol{a}' = \boldsymbol{F} \underbrace{- m\boldsymbol{\omega} \times (\boldsymbol{\omega} \times \boldsymbol{r})}_{\text{离心力}} \underbrace{- 2m\boldsymbol{\omega} \times \boldsymbol{v}'}_{\text{科里奥利力}} \underbrace{- m\dot{\boldsymbol{\omega}} \times \boldsymbol{r}}_{\textbf{欧拉力}}$$

欧拉力来自对旋转系速度求导时，$\boldsymbol{\omega}$ 本身对时间的导数项。

---

## 物理性质

| 性质 | 内容 |
|------|------|
| 方向 | **切向**（垂直于 $\boldsymbol{r}$），与离心力（径向）正交 |
| 大小 | $F_{\text{Euler}} = m|\dot{\boldsymbol{\omega}}|r\sin\theta$ |
| 本质 | 惯性力，在惯性系中不存在 |
| 做功 | 可以做功（力的方向与切向速度可以平行） |

---

## 与其他伪力的对比

| 伪力 | 公式 | 何时存在 | 方向 |
|------|------|----------|------|
| 离心力 | $-m\boldsymbol{\omega}\times(\boldsymbol{\omega}\times\boldsymbol{r})$ | $\omega \neq 0$ 即存在 | 径向向外 |
| [[科里奥利力]] | $-2m\boldsymbol{\omega}\times\boldsymbol{v}'$ | 质点有速度时 | 垂直于 $\boldsymbol{v}'$ |
| **欧拉力** | $-m\dot{\boldsymbol{\omega}}\times\boldsymbol{r}$ | $\omega$ **变化**时才出现 | 切向 |

---

## 物理直觉

### 为什么方向是切向？

旋转系突然加速（$\dot\omega > 0$）时，圆盘上每个点的切向速度都需要从 $\omega r$ 增大到 $(\omega+d\omega)r$：

```
原切向速度：ωr
新切向速度：(ω + dω)r  ← 圆盘要求
速度差：   r·dω        ← 你的惯性阻碍你跟上这个增量
```

你的惯性让你"跟不上"切向加速——等效于一个**切向反推力**，这就是欧拉力。

### 为什么大小正比于 $r$？

离轴越远，圆盘旋转加速所需的切向速度增量越大（$\Delta v_{\text{切}} = r\cdot\dot\omega$）。距轴两倍远的点，需要两倍的切向力才能跟上加速——欧拉力因此正比于 $r$。

```
轴心（r=0）：无需加速   → 欧拉力 = 0
中间（r=R）：需要 Rω̇   → 欧拉力 = mRω̇
边缘（r=2R）：需要 2Rω̇ → 欧拉力 = 2mRω̇
```

### 直觉场景：旋转木马

坐在**正在加速旋转**的旋转木马上：

```
旋转木马加速 → ω 增大 → ω̇ ≠ 0
→ 即使你静止不动（v' = 0，无科里奥利力）
→ 仍感到被切线方向推动
→ 这就是欧拉力
```

- **加速旋转**（$\dot\omega > 0$）：欧拉力与旋转方向**相反**（惯性阻碍你跟上加速，感觉被甩向切线后方）
- **减速旋转**（$\dot\omega < 0$）：欧拉力与旋转方向**相同**（圆盘减速但你"冲过头"，感觉被推向切线前方）

### 与离心力的对比

| | 离心力 | 欧拉力 |
|---|---|---|
| 触发条件 | $\omega \neq 0$ | $\dot\omega \neq 0$ |
| 方向 | 径向向外 | 切向 |
| 直觉 | "甩飞"感 | 启动/刹车感 |
| 正比于 | $\omega^2 r$ | $\dot\omega \cdot r$ |

---

## 匀速旋转时欧拉力消失

$$\dot{\boldsymbol{\omega}} = 0 \implies \boldsymbol{F}_{\text{Euler}} = 0$$

> 大多数基础物理问题假设 $\omega$ 恒定，因此欧拉力通常被忽略。[[科里奥利力]]的典型例题（旋转空间站抛球）正是在匀速旋转假设下，欧拉力为零。

## 典型场景

| 场景 | 说明 |
|------|------|
| 旋转木马加速/刹车 | 乘客感受到切向推力 |
| 飞机转弯时改变横滚角速度 | 飞行员受到欧拉力影响 |
| 工程陀螺仪启动/关闭 | 欧拉力导致进动误差 |
| 行星自转速率变化 | 对地表物体产生微小切向效应 |

## 数学小结

$$\boldsymbol{F}_{\text{Euler}} = -m\dot{\boldsymbol{\omega}} \times \boldsymbol{r}$$

- 仅在 $\boldsymbol{\omega}$ 随时间变化时存在
- 方向切向，大小正比于角加速度和到转轴的距离
- 与[[科里奥利力]]、离心力共同描述旋转系中的完整动力学
