# 科里奥利力 (Coriolis Force)

## 概述

科里奥利力是在**旋转参考系**中运动的物体所受到的一种**惯性力（伪力）**，由法国数学家 Gaspard-Gustave de Coriolis 于 1835 年提出。它不是真实存在的力，而是在非惯性参考系中描述运动时引入的修正项。

---

## 物理背景：旋转参考系

在一个以角速度 $\boldsymbol{\omega}$ 旋转的参考系 $S'$ 中，相对于惯性系 $S$，任意矢量 $\boldsymbol{A}$ 满足：

$$\left(\frac{d\boldsymbol{A}}{dt}\right)_S = \left(\frac{d\boldsymbol{A}}{dt}\right)_{S'} + \boldsymbol{\omega} \times \boldsymbol{A}$$

对位移矢量 $\boldsymbol{r}$ 应用两次，可推导出旋转系中的牛顿第二定律。

---

## 旋转参考系中的运动方程

设质点质量为 $m$，在旋转系中位置为 $\boldsymbol{r}$，速度为 $\boldsymbol{v}' = \dot{\boldsymbol{r}}'$，则运动方程为：

$$m\boldsymbol{a}' = \boldsymbol{F} - m\boldsymbol{\omega} \times (\boldsymbol{\omega} \times \boldsymbol{r}) - 2m\boldsymbol{\omega} \times \boldsymbol{v}' - m\dot{\boldsymbol{\omega}} \times \boldsymbol{r}$$

各项含义：

| 项 | 名称 | 说明 |
|----|------|------|
| $\boldsymbol{F}$ | 真实合力 | 惯性系中实际受到的力 |
| $-m\boldsymbol{\omega} \times (\boldsymbol{\omega} \times \boldsymbol{r})$ | **离心力** | 指向远离转轴方向 |
| $-2m\boldsymbol{\omega} \times \boldsymbol{v}'$ | **科里奥利力** | 与速度垂直，偏转运动方向 |
| $-m\dot{\boldsymbol{\omega}} \times \boldsymbol{r}$ | [[欧拉力]] | 仅在角速度变化时出现 |

---

## 科里奥利力的表达式

$$\boxed{\boldsymbol{F}_{\text{Cor}} = -2m\,\boldsymbol{\omega} \times \boldsymbol{v}'}$$

- $\boldsymbol{\omega}$：旋转参考系的角速度（矢量，方向由右手定则确定）
- $\boldsymbol{v}'$：质点在旋转系中的速度
- 大小：$F_{\text{Cor}} = 2m\omega v' \sin\theta$，其中 $\theta$ 为 $\boldsymbol{\omega}$ 与 $\boldsymbol{v}'$ 的夹角

### 方向判断

科里奥利力**始终垂直于速度方向**，用右手定则确定 $\boldsymbol{\omega} \times \boldsymbol{v}'$ 后取负号：

- 北半球：水平运动向**右**偏转
- 南半球：水平运动向**左**偏转
- 在转轴方向运动时：科里奥利力为零

---

## 物理直觉：为什么运动会导致偏转？

### 情形一：沿径向向外运动

设你站在旋转圆盘上，向外（远离轴）走动：

```
你在半径 r 处，切向速度 = ωr（与圆盘同步）
      ↓ 向外走到 r + dr
圆盘在此处的切向速度 = ω(r+dr) > ωr
你的切向速度没有跟上 → 相对圆盘，你"向后落"
感受：被推向旋转反方向（北半球 → 向右）
```

- 本质：到了"更快"的地方，但自己没那么快——**角动量守恒带来的切向落后**

### 情形二：沿切向顺旋转方向运动

设你在圆盘上沿切向加速运动（速度 > $\omega r$）：

```
总切向速度超过圆盘 → 相当于在"更大半径"处旋转
需要更大向心力才能跟上曲线
实际向心力不足 → 被甩向外侧（径向向外）
感受：被推向远离轴的方向（北半球 → 向右）
```

- 本质：切向速度过快，向心力"拉不住"——**多余的离心效应**

### 两种情形的统一

无论径向还是切向运动，结果都向**右**偏（北半球）。这由 $-2m\boldsymbol{\omega}\times\boldsymbol{v}'$ 自动给出：

| 运动方向 | $\boldsymbol{\omega}\times\boldsymbol{v}'$ | 科里奥利力方向 |
|---------|------------------------------------------|--------------|
| 径向向外 | 切向（顺旋转） | 切向（逆旋转）← **向右** |
| 切向顺转 | 径向向外 | 径向向内... 等效向右偏 |
| 任意方向 | ⊥ $\boldsymbol{v}'$ | 始终垂直速度，不改变速率 |

## 典型例题：旋转空间站中的抛球

> 一个圆形旋转空间站，孩子在底部抛出一个球，使得空间站转过半圈后孩子接住球。从孩子的视角，球的轨迹是什么形状？（答案：C）

### 惯性系分析

- 球在惯性系中做**直线运动**（无外力）
- 设空间站半径 $R$，角速度 $\omega$，孩子初始在底部 $(0, -R)$
- 半圈后孩子到顶部 $(0, +R)$，历时 $T = \pi/\omega$
- 球的惯性系速度：$(v_x, v_y) = (0,\; 2R\omega/\pi)$

### 旋转系中的轨迹

令 $\tau = \omega t$，通过坐标变换得到旋转系中球的轨迹：

$$x_r = R\!\left(-1 + \frac{2\tau}{\pi}\right)\sin\tau, \qquad y_r = R\!\left(-1 + \frac{2\tau}{\pi}\right)\cos\tau$$

| $\tau$ | $(x_r,\, y_r)/R$ | 描述 |
|--------|-----------------|------|
| $0$ | $(0,\ -1)$ | 出发（底部） |
| $\pi/4$ | $(-0.35,\ -0.35)$ | 向左上偏移 |
| $\pi/2$ | $(0,\ 0)$ | 穿过圆心 |
| $3\pi/4$ | $(+0.35,\ -0.35)$ | 向右下回落 |
| $\pi$ | $(0,\ -1)$ | 回到孩子手中 |

**结论**：球画出一个**封闭的泪滴形环路**，从底部出发受科里奥利力偏转，穿过站中心后回到底部。

## 科里奥利力的实际效应

### 地球上的效应

地球自转角速度 $\omega = 7.27 \times 10^{-5}\ \text{rad/s}$

| 现象 | 说明 |
|------|------|
| **大气环流** | 北半球气旋逆时针旋转，南半球顺时针 |
| **洋流偏转** | 海洋环流的旋转方向由科里奥利力决定 |
| **傅科摆** | 摆的振动平面在地球上缓慢旋转 |
| **炮弹偏转** | 长程炮弹（北半球）向右偏移 |
| **河流侵蚀** | 北半球河流右岸侵蚀更严重（白贝罗定律） |

### 工程应用

- **惯性导航系统（INS）**：需要补偿科里奥利力
- **科里奥利流量计**：利用科里奥利力测量流体质量流量
- **陀螺仪**：与科里奥利效应密切相关

---

## 常见误解

> ❌ **误**：浴缸/马桶的排水旋转方向由科里奥利力决定
>
> ✅ **实**：生活尺度下科里奥利力极小，排水旋转主要由初始水流扰动决定。科里奥利力只在大尺度（气象、海洋）运动中显著。

---

## 数学小结

$$\boldsymbol{F}_{\text{Cor}} = -2m\,\boldsymbol{\omega} \times \boldsymbol{v}'$$

| 性质 | 内容 |
|------|------|
| 方向 | 垂直于 $\boldsymbol{v}'$，**不做功** |
| 大小 | 正比于速率和角速度 |
| 本质 | 惯性力，在惯性系中不存在 |
| 效果 | 使运动方向偏转，不改变速率 |
